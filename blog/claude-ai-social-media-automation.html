Step-by-Step Setup Process

Setting up your AI social media manager takes about 30 minutes following these steps:

Step 1: Install VS Code & Claude Extension

Download VS Code and install the Claude Code extension from the marketplace. Connect it to your Anthropic account (Claude Pro required). Enable bypass permissions for autonomous operation.

Step 2: Clone the GitHub Repository

The system is available as a free GitHub repo containing all skills and setup files. Simply provide the repo link to Claude and it will handle the installation automatically.

Step 3: Configure Late API

Create a Late account (free tier available), connect your social media profiles, and generate an API key. Provide this to Claude to establish the connection.

Step 4: Test the System

Claude will verify all connections and list your connected social accounts. At this point you can start posting through natural language commands.

Pro Tip: The entire setup can be completed by simply giving Claude the repo link and API key - it will handle all technical configuration automatically.

Two Ways to Invoke Your LLM

The tutorial demonstrates two approaches to model invocation, each with distinct advantages:

Method 1: Provider-Specific

 from langchain_groq import ChatGroq llm = ChatGroq(model="llama-3-18b")

Best for: Projects committed to a specific LLM provider

Method 2: Unified Interface

 from langchain.chat_models import init_chat_model llm = init_chat_model("grock:llama-3-18b")

Best for: Projects that may switch providers or support multiple LLMs

Both methods use the same llm.invoke() pattern to get responses, but Method 2 offers more flexibility for future changes. The tutorial includes working examples of both at 12:30.

Building the Chatbot Workflow

The tutorial's chatbot.py implements three key components:

1. Model Initialization

 def create_model(model_name):     return init_chat_model(f"grock:{model_name}")

2. Question Handling

 def ask_question(model, question):     response = model.invoke(question)     return response.content

3. Main Loop

 while question.lower() != "exit":     print(ask_question(model, question))     question = input("Next question: ")

This structure demonstrates LangChain's core value: abstracting away API complexity so you focus on conversation flow. The full implementation is shown at 18:45 in the video.

Building Your First Visual Agent

Creating a custom agent involves three simple steps: configuration, data connection, and prompt engineering. The live preview updates instantly as you work, letting you see exactly how your agent will behave.

Step 1: Agent Configuration

Start by naming your agent and defining its purpose. For a data analysis agent, you might specify that it "accepts CSV/Excel files, analyzes the data automatically, and generates charts, graphs and presentation-style outputs."

Step 2: Data Connection

Thesys supports direct file uploads (CSV, Excel) or database connections. Upload your dataset - like a sales spreadsheet - and describe how the agent should use it. Multiple data sources can be connected for cross-analysis.

Step 3: Prompt Engineering

Define the system prompt that guides your agent's behavior. A good prompt specifies the agent's role (e.g., "professional data analysis assistant"), output style ("visual, not text-heavy"), and response rules ("include interactive elements").

Pro Tip: Enable tools like Web Search and Artifact Builder to expand your agent's capabilities. These allow it to pull in external data and create more complex visual structures.

What This Workflow Does

This automation solves the tedious manual process of creating Softr user accounts for new WooCommerce customers. When someone makes a purchase on your WooCommerce store, the workflow automatically creates their Softr account and sends a personalized invitation email - all without any manual intervention.

Businesses using Softr for member portals, client dashboards, or gated content often struggle with onboarding new users efficiently. This workflow eliminates the back-and-forth of manual account creation while ensuring customers get immediate access to your Softr resources after purchase.

How It Works

1. New customer detection

The workflow monitors your WooCommerce store for new customer registrations or purchases. It triggers whenever a new customer profile is created in your WooCommerce database.

2. Data mapping

Customer information (name, email, and any custom fields) is extracted from WooCommerce and formatted for Softr's API requirements. You can customize which fields transfer between systems.

3. Softr user creation

The workflow makes an API call to Softr to create the new user account with the mapped customer data. You can specify user roles, permissions, and any default settings.

4. Email invitation

Once the Softr account is created, the system sends a professionally formatted email invitation with login instructions. The template is fully customizable with your branding.

Pro tip: Add a delay before sending the invitation email if customers need time to receive their WooCommerce order confirmation first.

What This Workflow Does

This automation solves the common productivity gap between document creation in Google Drive and task management in Trello. Teams often waste time manually creating Trello cards for new documents, leading to missed files and inconsistent tracking.

The workflow automatically monitors specified Google Drive folders and creates corresponding Trello cards whenever new files are added. Each card includes the document name, direct link, and optional metadata like the creator's name or folder location. This ensures every new document becomes an actionable item in your project management system without manual intervention.

How It Works

1. Google Drive Trigger

The workflow starts when Make.com detects a new file in your specified Google Drive folder. It captures the file details including name, URL, and metadata.

2. Data Processing

Make formats the file information into Trello-compatible fields. You can customize which details transfer to the card description, such as adding folder paths as labels or including the document creator's name.

3. Trello Card Creation

The workflow creates a new card in your designated Trello board and list. The card title matches the file name, while the description contains the Google Drive link and any additional details you've configured.

Pro tip: Use Trello custom fields to add document status tracking (e.g., "Needs Review", "Approved") directly on the generated cards.

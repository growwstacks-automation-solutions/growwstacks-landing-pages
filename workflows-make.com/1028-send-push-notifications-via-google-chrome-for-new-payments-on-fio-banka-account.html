What This Workflow Does

This automation solves the problem of constantly checking your Fio banka account for incoming payments. Many businesses waste hours each week manually logging into banking portals to verify if clients or customers have paid. With this Make.com template, you'll receive instant Chrome browser push notifications the moment any payment hits your account.

The workflow connects to Fio banka's API to monitor transaction activity, then triggers desktop notifications through Google Chrome's push notification system. Unlike email alerts that can get buried in your inbox, these notifications appear prominently on your screen with optional sound alerts, ensuring you never miss an important payment.

How It Works

1. Payment Detection

The scenario periodically checks your Fio banka account via API for new transactions at intervals you configure (default: every 5 minutes). It compares incoming transactions against previously recorded ones to identify new payments.

2. Data Processing

For each new payment, the workflow extracts key details like amount, sender information, payment reference, and timestamp. This data is formatted for clear notification display.

3. Notification Delivery

Using Chrome's push notification API, the system sends a desktop alert with payment details directly to your browser. Notifications appear even when Chrome is minimized or in the background.

Pro tip: Customize notification content to show only the most relevant payment details for your workflow, like client names or invoice references.

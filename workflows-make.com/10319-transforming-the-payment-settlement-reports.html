What This Workflow Does

Hoteliers and property managers using Apaleo Pay often struggle with reconciling daily settlement reports. The manual process involves downloading multiple files, merging data, converting formats, and verifying amounts - a time-consuming and error-prone process.

This Make.com automation transforms Apaleo Pay settlement reports into accounting-friendly formats automatically. It fetches settlement files, merges transactions from multiple properties, converts data to your preferred accounting format (CSV, XLSX), and prepares them for import into your financial system.

How It Works

1. Fetch Settlement Reports

The workflow connects to your Apaleo account via API and retrieves the latest settlement reports based on your configured date range.

2. Merge Multiple Files

If you manage multiple properties, the automation combines all settlement files into a single consolidated report with proper property identifiers.

3. Transform Data Format

The raw settlement data gets restructured into columns matching your accounting software's import requirements, including proper date formats, amount calculations, and transaction categories.

4. Validate Totals

The system cross-checks settlement amounts against Apaleo's reported totals to flag any discrepancies before accounting import.

5. Deliver Processed File

The final formatted file gets saved to your preferred location - whether that's cloud storage, email attachment, or direct integration with your accounting software.

Pro tip: Schedule this workflow to run automatically after each settlement period (daily/weekly) to maintain real-time financial visibility.

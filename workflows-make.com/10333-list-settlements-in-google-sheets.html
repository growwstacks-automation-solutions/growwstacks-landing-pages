What This Workflow Does

This automation solves the tedious manual process of tracking Mollie payment settlements by automatically syncing all settlement data to a Google Sheets spreadsheet. It retrieves complete settlement details including period lines (individual payment batches), amounts, fees, and settlement dates on a scheduled basis (default weekly).

E-commerce businesses using Mollie payments often struggle with financial reconciliation across multiple payment methods and settlement periods. This workflow creates a single source of truth for all settlement activity, eliminating spreadsheet errors and saving hours of manual data entry each month.

How It Works

1. Scheduled Trigger

The workflow runs automatically at your chosen interval (default is Monday mornings) to fetch the latest settlement data from Mollie's API.

2. Mollie API Connection

Using secure OAuth authentication, the system retrieves all settlements within your specified date range, including detailed line items for each payment batch.

3. Data Transformation

The raw API data gets formatted into a clean structure with consistent date formats, calculated net amounts, and properly labeled fee categories.

4. Google Sheets Update

Transformed data gets appended to your designated spreadsheet with timestamps, creating an audit trail of all settlements over time.

Pro tip: Add a second tab in your spreadsheet with pivot tables to automatically analyze settlement trends by payment method, week, or month.

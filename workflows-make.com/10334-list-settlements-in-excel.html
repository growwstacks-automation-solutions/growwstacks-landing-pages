What This Workflow Does

This automation solves the tedious manual process of exporting and formatting Mollie payment settlement data for financial reporting. Instead of logging into Mollie's dashboard weekly to download CSV files and reformat them for accounting, the workflow automatically retrieves all settlements with their period line details and organizes them in a standardized Excel format.

The template is pre-configured to run every Monday morning, capturing the previous week's transaction data before new business activity begins. It handles all the API calls to Mollie, transforms the JSON responses into spreadsheet-ready data, and saves the results to your designated Excel file with proper column headers and formatting.

How It Works

1. Scheduled Trigger

The workflow activates automatically at your specified interval (default is weekly on Monday at 9 AM). No manual initiation required.

2. Mollie API Connection

Using your authenticated Mollie API credentials, the system retrieves all settlements within the defined date range, including detailed line items for each transaction.

3. Data Transformation

The raw API data is parsed and restructured into a clean table format with consistent column naming and proper data types for amounts, dates, and fees.

4. Excel Export

The processed data is appended to your designated Excel worksheet with timestamped entries, creating an audit trail of all settlements over time.

Pro tip: Add a second Excel sheet with pivot tables to automatically analyze payment method distribution, fee percentages, and weekly revenue trends.

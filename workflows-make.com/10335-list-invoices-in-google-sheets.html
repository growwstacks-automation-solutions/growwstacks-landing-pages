What This Workflow Does

This automation solves the tedious manual process of exporting and compiling Mollie payment invoices for financial reporting. Finance teams typically spend hours each month downloading CSV files from Mollie, cleaning up the data, and organizing it into usable formats for accounting and reconciliation.

The workflow automatically retrieves all invoices from your Mollie account at scheduled intervals (default monthly), extracts both header information and detailed line items, and structures this data in a Google Sheets spreadsheet with proper formatting. This creates a centralized, always-updated repository of your payment data ready for reporting, auditing, or integration with other financial systems.

How It Works

1. Scheduled Trigger

The scenario activates automatically at your chosen interval (default is first day of each month) using Make.com's scheduler module.

2. Mollie API Connection

It securely connects to your Mollie account via API to retrieve all invoices within the specified date range.

3. Data Processing

The workflow parses each invoice, extracting both summary information (invoice number, date, amount) and detailed line items.

4. Google Sheets Update

Data is written to designated tabs in your Google Sheet, with new records appended below existing data.

Pro tip: Add a "Last Updated" timestamp column to track when each record was refreshed, helping identify new vs existing entries.

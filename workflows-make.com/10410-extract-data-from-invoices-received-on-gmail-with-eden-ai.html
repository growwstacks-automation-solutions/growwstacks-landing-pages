What This Workflow Does

This automation solves the tedious manual process of extracting key data from invoices received via email. When vendors email PDF or image invoices to your Gmail account, the workflow automatically:

  • Identifies new invoice emails based on sender, subject, or labels
  • Downloads attached invoice files
  • Processes documents through Eden AI's extraction engine
  • Captures structured data like vendor details, invoice numbers, dates, line items, and totals
  • Outputs the data to your preferred destination (spreadsheet, accounting software, etc.)

Pro tip: The template extracts common invoice fields by default but can be customized to capture additional data points specific to your vendors or accounting requirements.

How It Works

1. Email Trigger Setup

The workflow monitors your Gmail inbox for new messages matching your criteria (specific senders, subjects containing "invoice", or labeled messages). You can configure multiple trigger conditions to catch invoices from different vendors.

2. Attachment Processing

When a matching email arrives, the system downloads any PDF or image attachments. It checks file formats and prepares documents for data extraction, handling multi-page invoices and batch processing when multiple files are attached.

3. AI-Powered Data Extraction

Eden AI's document processing analyzes each invoice, identifying and extracting key fields with machine learning trained on thousands of invoice formats. The system captures both structured data (like tables) and unstructured information.

4. Data Validation & Routing

Extracted data undergoes basic validation checks before being formatted for your destination system. You can configure rules to flag potential errors for review or automatically route verified data to accounting software.

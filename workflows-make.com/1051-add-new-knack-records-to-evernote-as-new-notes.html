What This Workflow Does

This automation solves the common challenge of maintaining parallel records between your Knack database and Evernote knowledge base. Every time a new record is created in your Knack application, this Make.com workflow automatically generates a corresponding note in Evernote with all the relevant details.

The workflow eliminates manual copy-pasting between systems, ensuring your Evernote archive stays perfectly synchronized with your operational database. It's particularly valuable for businesses that use Knack for CRM, project management, or support ticketing while relying on Evernote for documentation and knowledge retention.

How It Works

1. New record detection

The workflow monitors your Knack database for new record creation events through Make.com's Knack connector. You can specify which tables or object types to watch.

2. Data extraction

When a new record appears, the workflow extracts all specified fields - including text, numbers, dates, and even file attachments from Knack.

3. Note formatting

The extracted data is formatted into a clean Evernote note structure. You can customize the template to include field labels, section headers, and static text.

4. Evernote creation

Make.com's Evernote module creates a new note in your specified notebook, tagged appropriately for easy retrieval. The note includes all mapped Knack fields in your preferred layout.

Pro tip: Add conditional logic to only create Evernote notes for records meeting specific criteria (like priority tickets or key customers) to avoid clutter.

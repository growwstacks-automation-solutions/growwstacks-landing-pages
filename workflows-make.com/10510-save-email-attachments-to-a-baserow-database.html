What This Workflow Does

This automation solves the common problem of email attachments getting lost in crowded inboxes or buried in download folders. Every time your business receives an email with attachments, this Make.com workflow automatically:

1. Detects new emails with attachments
2. Extracts each file along with relevant email metadata
3. Creates a structured record in your Baserow database
4. Stores the attachment with contextual information like sender, date, and subject

How It Works

1. Email Trigger Setup

The workflow monitors your connected email account (Gmail, Outlook, etc.) for new messages containing attachments. You can filter by sender, subject keywords, or other criteria.

2. Attachment Processing

For each qualifying email, the system extracts all attachments while preserving the original file names and formats. The workflow handles multiple attachments per email efficiently.

3. Baserow Record Creation

Each attachment becomes a new database row with fields for the file itself plus metadata columns like sender email, received timestamp, email subject, and any custom tags you configure.

Pro tip: Add a "processed by" field to track which team member handles each attachment, creating an audit trail for document workflows.

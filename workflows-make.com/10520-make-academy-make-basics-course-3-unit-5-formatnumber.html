What This Workflow Does

This workflow template demonstrates how to use Make.com's formatNumber function to transform numerical data into properly formatted outputs. Number formatting is essential for creating professional invoices, reports, financial statements, and any business documents where numerical precision and presentation matter.

The template teaches you how to convert raw numbers into standardized formats with proper decimal places, thousand separators, and rounding rules. You'll learn to maintain data integrity while ensuring your outputs meet business presentation standards across different systems and regions.

How It Works

1. Input raw numerical data

The workflow accepts numerical input from any source, whether it's from a spreadsheet, database, or another application in your automation.

2. Apply formatNumber function

The core transformation happens using Make.com's formatNumber function, which lets you specify decimal places, thousand separators, and rounding behavior.

3. Configure formatting options

You can customize the output format to match your business requirements, including:

  • Number of decimal places
  • Use of commas or periods as thousand separators
  • Rounding rules (up, down, or to nearest)
  • Optional currency symbols

4. Output formatted results

The properly formatted numbers can then be sent to any destination system, whether it's an accounting package, CRM, email template, or report generator.

Pro tip: Always format numbers at the last possible step in your workflow to maintain maximum precision during calculations.

In automation workflows, it's best practice to keep numbers in calculation format during processing steps, then apply display formatting just before output. This preserves accuracy while delivering polished results.

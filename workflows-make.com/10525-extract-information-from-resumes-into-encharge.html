What This Workflow Does

This automation solves the tedious manual work of extracting candidate information from resumes and entering it into your CRM. When recruiters or hiring managers upload resumes to a designated Google Drive folder, the workflow automatically:

  • Processes PDF, Word, and other common resume formats
  • Extracts key candidate details using Eden AI's parsing technology
  • Creates or updates contact records in Encharge with structured data
  • Organizes candidates by skills, experience level, and other criteria

For a mid-sized company processing 200 resumes weekly, this automation can save 15-20 hours of manual data entry while improving data accuracy and candidate response times.

How It Works

1. Resume Upload Trigger

The workflow monitors a specific Google Drive folder for new resume uploads. When a new file appears, it initiates the parsing process.

2. AI-Powered Data Extraction

Eden AI analyzes the resume document, identifying and categorizing information including:

  • Candidate name and contact details
  • Work history with company names and dates
  • Education background and degrees
  • Skills and certifications

3. Encharge CRM Integration

The parsed data maps to corresponding fields in Encharge, creating comprehensive candidate profiles that include:

  • Standard contact information fields
  • Custom fields for skills and experience levels
  • Resume file attachments for reference
  • Tags for easy filtering and segmentation

Pro tip: Configure Encharge automation rules to trigger personalized email sequences based on the extracted candidate qualifications, creating immediate engagement with top talent.

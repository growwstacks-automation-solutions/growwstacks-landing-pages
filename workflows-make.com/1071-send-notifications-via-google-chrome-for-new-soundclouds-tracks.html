What This Workflow Does

This automation solves the problem of constantly checking SoundCloud for new music releases. For music enthusiasts, A&R professionals, and content creators, missing a new track from a favorite artist can mean lost opportunities. The workflow monitors specified SoundCloud accounts and delivers instant Chrome notifications whenever new content appears.

Unlike SoundCloud's native notifications that are easy to miss, these Chrome alerts appear prominently on your desktop or mobile device. You'll know immediately when artists you care about release new material, without having to manually browse SoundCloud or rely on sporadic email updates.

How It Works

1. Track Artist Activity

The workflow continuously monitors the SoundCloud profiles you specify, checking for new uploads at regular intervals.

2. Detect New Tracks

When a followed artist publishes new content, the automation instantly recognizes the new track and captures key details like title, artist, and URL.

3. Send Chrome Notification

The system generates a browser notification containing the new track information, appearing even when you're not actively using SoundCloud.

4. Direct Listening Access

Each notification includes a clickable link that takes you directly to the new track, eliminating search time.

Pro tip: Combine this with other music discovery automations to create a comprehensive monitoring system across multiple platforms.

What This Workflow Does

This automation transforms raw Gong call recordings into actionable competitive intelligence. It automatically detects when competitors are mentioned during sales conversations, analyzes what customers say about them, and structures these insights in Airtable for easy reference.

The system identifies not just competitor names but also the context: which products/services were discussed, perceived strengths and weaknesses, pricing comparisons, and common objections where competitors come up. This gives sales teams a strategic advantage by revealing patterns across customer conversations.

How It Works

1. Gong call monitoring

The workflow continuously monitors new Gong call recordings through API connections. When a new call is logged, it automatically pulls the transcript and metadata (participants, date, deal stage).

2. AI-powered competitor detection

Using natural language processing, the system scans call transcripts for competitor mentions. It goes beyond simple name recognition to understand context - identifying products discussed, customer sentiments, and comparative statements.

3. Insight categorization

Each competitor mention is analyzed and tagged with relevant categories: product comparisons, pricing discussions, feature differentiators, customer pain points, or strengths/weaknesses. This structured data makes analysis more actionable.

4. Airtable database population

The processed insights are automatically added to an Airtable base with fields for competitor name, mentioned products, sentiment analysis, call context, and timestamps. This creates a searchable competitive intelligence repository.

What You'll Need

  1. Active Gong account with API access
  2. Make.com account (formerly Integromat)
  3. Airtable account with a base prepared for competitor data
  4. List of competitor names/products to monitor
  5. Optional: AI service API key for enhanced analysis

Quick Setup Guide

  1. Connect your Gong account in Make.com using API credentials
  2. Set up the Airtable connection and map fields for competitor data
  3. Configure the competitor detection parameters (names, products, synonyms)
  4. Test with sample call transcripts to refine analysis rules
  5. Schedule the workflow to run automatically after each new call

Pro tip: Create an Airtable dashboard view filtered by competitor to quickly see all mentions and sentiment trends for each rival.

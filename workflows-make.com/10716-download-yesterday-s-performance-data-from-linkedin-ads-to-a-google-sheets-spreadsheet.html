What This Workflow Does

This automation solves the tedious manual process of downloading LinkedIn Ads performance data and copying it into spreadsheets. Marketing teams waste hours each week on repetitive reporting tasks that could be automated.

The workflow connects directly to LinkedIn's API to extract yesterday's campaign metrics including impressions, clicks, conversions, and spend. It formats this data and appends it to your designated Google Sheet, creating a running historical record of all campaign performance.

How It Works

1. Daily Trigger

The workflow automatically runs each morning to collect data from the previous day. You can adjust the schedule if you prefer weekly or monthly updates.

2. LinkedIn API Connection

Using OAuth authentication, the template securely accesses your LinkedIn Ads account through their official API. It retrieves performance data for all active campaigns.

3. Data Transformation

The raw API response gets formatted into a clean table structure with consistent column headers. Metrics are standardized for easy analysis.

4. Google Sheets Update

The processed data gets appended to your specified Google Sheet. Each day's results add a new row while preserving all historical data.

Pro tip: Add a pivot table to your Google Sheet to automatically summarize performance by campaign, date range, or objective.

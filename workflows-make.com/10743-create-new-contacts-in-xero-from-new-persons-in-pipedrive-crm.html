What This Workflow Does

This automation bridges the gap between your sales pipeline and accounting system by automatically creating Xero contacts whenever new persons are added in Pipedrive CRM. It solves the common problem of client information existing in your CRM but not being available when creating invoices or tracking payments in Xero.

By running every 15 minutes (configurable), the workflow ensures your accounting team always has access to current client details without manual data entry. This eliminates duplicate work between sales and accounting teams while reducing errors from manual transcription.

How It Works

1. Checks for New Pipedrive Contacts

The workflow starts by querying Pipedrive's API for any new person records created since the last run. It uses Pipedrive's timestamp filtering to efficiently identify only new entries.

2. Maps Contact Information

Key fields from Pipedrive (name, email, phone, company details) are mapped to corresponding Xero contact fields. The template includes standard mappings but can be customized to your specific field structure.

3. Creates Xero Contacts

Using Xero's API, the workflow creates new contact records with all mapped information. It handles API authentication and error checking to ensure reliable operation.

Pro tip: Add a custom field in Pipedrive to flag contacts that should sync to Xero (like "Billable Client"). Then modify the workflow to only sync contacts with this flag.

Quick Setup Guide

  1. Clone the template in your Make.com account
  2. Connect your Pipedrive and Xero accounts via OAuth
  3. Review the default field mappings between systems
  4. Test with a sample Pipedrive contact to verify Xero creation
  5. Adjust sync frequency if needed (default is every 15 minutes)

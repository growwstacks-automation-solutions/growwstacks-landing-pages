What This Workflow Does

This automation solves the tedious process of manually checking for new documents in OneDrive and sending them to print. Many businesses still rely on employees to download files and manually print them, which wastes time and creates bottlenecks in document workflows.

The template automatically monitors your specified OneDrive folder for new files every 15 minutes (configurable). When it detects a new document, it securely sends the file to your ezeep Blue cloud printing service, which routes it to the appropriate printer based on your organization's rules. This eliminates manual steps while maintaining security and control over document printing.

How It Works

1. Scheduled OneDrive Monitoring

The workflow begins by checking your connected OneDrive account at regular intervals. It looks for files added since the last check, focusing on the folder(s) you specify during setup.

2. New File Detection

When the system identifies new files, it captures key details like filename, file type, size, and folder location. This information helps with routing decisions and provides audit trails.

3. Secure File Transfer

Each new document is securely transferred from OneDrive to ezeep Blue's cloud printing platform. The connection uses Microsoft's authentication protocols to maintain security.

4. Automated Printing

ezeep Blue receives the file and processes it according to your organization's printing rules. This can include specific printer selection, duplex settings, color preferences, and other print parameters.

Pro tip: Configure different OneDrive folders to route to different printers. For example, send contracts to your legal department printer and invoices to accounting automatically.

What This Workflow Does

This automation solves the challenge of manually welcoming every new member in your Discourse community - a time-consuming task that often gets neglected as communities grow. The template automatically detects new member signups and sends them a personalized welcome message within minutes of joining.

By automating this first touchpoint, you ensure every member receives consistent, timely onboarding while freeing up community managers to focus on higher-value interactions. The system pulls the new member's name and profile details to craft messages that feel personal rather than generic.

How It Works

1. New member detection

The workflow monitors your Discourse community for new user registrations through webhooks or API polling. When a new account is created, it triggers the automation sequence.

2. Profile data collection

The system retrieves the member's username, join date, and any available profile information from Discourse to personalize the welcome message.

3. Message composition

Using your predefined message template with placeholder variables, the system generates a personalized welcome that includes the member's name and relevant community information.

4. Automated delivery

The completed message is sent directly to the new member through Discourse's private messaging system, appearing to come from your designated community manager account.

Pro tip: Include 2-3 recommended discussion threads or resources in your welcome message to immediately guide new members toward valuable content.

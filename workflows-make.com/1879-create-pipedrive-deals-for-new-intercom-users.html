Create Pipedrive Deals for New Intercom Users

Automatically convert Intercom signups into Pipedrive deals to streamline your sales pipeline

Get This Workflow → Make.com · CRM · Free Template

What This Workflow Does

This automation bridges the gap between your customer support and sales teams by automatically creating Pipedrive deals whenever new users sign up through Intercom. It eliminates the manual work of copying user information between systems and ensures your sales team never misses a potential lead.

The workflow triggers the moment a new user is created in Intercom, capturing all relevant contact information and creating a corresponding deal in Pipedrive with the appropriate pipeline, stage, and custom fields. This creates a seamless handoff between teams while maintaining data accuracy and consistency across platforms.

How It Works

1. New user detection in Intercom

The workflow monitors Intercom for new user creation events. When triggered, it captures the user's name, email, company, and any custom attributes you've configured in Intercom.

2. Data processing and enrichment

Make processes the incoming user data, formatting it appropriately for Pipedrive and optionally enriching it with additional information from other connected apps or databases.

3. Deal creation in Pipedrive

The workflow creates a new deal in your specified Pipedrive pipeline, mapping all relevant fields from Intercom to the corresponding Pipedrive deal and contact fields.

4. Error handling and logging

The automation includes built-in error handling to manage cases where required fields are missing or API connections fail, with detailed logging for troubleshooting.

Who This Is For

This automation is ideal for businesses that use Intercom for customer communication and Pipedrive as their CRM. It's particularly valuable for:

  • SaaS companies with free trial signups through Intercom
  • Sales teams that want faster lead follow-up
  • Companies struggling with manual data entry between support and sales
  • Businesses looking to improve lead conversion rates
  • Teams wanting better visibility into their sales pipeline

What You'll Need

  1. An active Intercom account with admin access
  2. A Pipedrive account with API access enabled
  3. A Make.com account (free plan sufficient for basic setup)
  4. Clear mapping of which Intercom fields should go to which Pipedrive fields
  5. Decide on your default Pipedrive pipeline and stage for new deals

Quick Setup Guide

  1. Clone the template from Make.com to your account
  2. Connect your Intercom account in the first module
  3. Connect your Pipedrive account in the deal creation module
  4. Map the field values from Intercom to Pipedrive
  5. Set your default pipeline and stage for new deals
  6. Test with a sample user to verify the flow
  7. Activate the scenario and monitor initial executions

Pro tip: Add conditional logic to route different types of users to different pipelines. For example, free trial users might go to a "Nurture" pipeline while paid signups go directly to "Qualified Leads."

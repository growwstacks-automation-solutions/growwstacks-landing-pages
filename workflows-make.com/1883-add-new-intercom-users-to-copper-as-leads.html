What This Workflow Does

This automation solves the common problem of sales teams missing valuable leads captured in customer support conversations. When users engage with your Intercom chat or messaging system, they often reveal buying intent that should be followed up by sales.

The workflow automatically creates Copper CRM lead records for new Intercom users, ensuring no potential customer falls through the cracks between teams. It eliminates manual data entry while maintaining data accuracy between systems.

How It Works

1. New user detection in Intercom

The workflow monitors Intercom's API for new user creation events. This could be when someone starts a chat, signs up for your product, or submits a contact form.

2. Data extraction and formatting

Key user information like name, email, company, and conversation context is extracted from Intercom. The data is formatted to match Copper's lead record structure.

3. Lead creation in Copper

The formatted data is sent to Copper CRM via API to create a new lead record. Custom field mappings ensure all relevant information transfers accurately.

4. Error handling and logging

The workflow includes robust error handling to manage API rate limits, field mismatches, or duplicate records. All actions are logged for troubleshooting.

What This Workflow Does

This automation solves a common problem for businesses using Jotform for lead capture and Intercom for customer communication. Manually transferring form submissions to Intercom is time-consuming and prone to errors. This workflow eliminates that manual process by automatically creating Intercom user profiles whenever someone submits your Jotform.

The integration ensures your sales and support teams have immediate access to new leads in Intercom, where they can track conversations, send targeted messages, and monitor engagement - all without any manual data entry. This is particularly valuable for businesses with high-volume forms like contact requests, demo signups, or support inquiries.

How It Works

1. New Jotform Submission Trigger

The workflow starts whenever someone submits your Jotform. Make.com monitors your form for new responses in real-time.

2. Data Extraction

The system extracts all submitted form fields including name, email, and any custom questions you've included in your Jotform.

3. Intercom User Creation

Make.com then creates a new user profile in Intercom using the submitted information. You can customize which fields map to which Intercom attributes during setup.

4. Optional Data Enrichment

The workflow can optionally add tags, custom attributes, or notes to the Intercom user based on their form responses for better segmentation.

Pro tip: Add a hidden field in your Jotform to track the submission source (e.g. "Website Contact Form") that will automatically tag users in Intercom.

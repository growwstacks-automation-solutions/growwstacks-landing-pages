Save Optimy Projects to Google Sheets

Automatically export project data from Optimy to Google Sheets with custom filters

Get This Workflow → Make.com · Free Template

What This Workflow Does

This automation solves the challenge of manually exporting project data from Optimy to spreadsheets for reporting and analysis. Teams often waste hours each week copying project details into Google Sheets to create status reports, budget trackers, or approval workflows.

The workflow automatically transfers Optimy project records to your specified Google Sheet based on customizable filters. You can set criteria like project status (active/completed), specific forms, owners, or custom fields. The template maintains data consistency while eliminating manual data entry errors.

How It Works

Step 1: Configure Optimy Connection

The workflow starts by authenticating with your Optimy account using API credentials. You'll specify which projects to include based on your filter criteria.

Step 2: Retrieve Project Data

Make.com queries Optimy's API to fetch all projects matching your filters, including custom form responses, status changes, and assigned team members.

Step 3: Format for Google Sheets

The workflow transforms the Optimy API response into spreadsheet-friendly columns, handling nested data structures and converting timestamps to readable formats.

Step 4: Update Google Sheet

Finally, the processed data gets appended to your specified Google Sheet, with options to overwrite existing data or add new rows while preserving historical records.

Pro tip: Create separate Sheets tabs for different project categories (e.g., "Active Projects" vs "Completed") by duplicating the workflow with different filters.

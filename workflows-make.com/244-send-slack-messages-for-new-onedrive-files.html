What This Workflow Does

This automation solves the common problem of team members missing important file updates in OneDrive. Without automation, teams waste time manually checking folders or sending "I've uploaded the file" emails. The workflow monitors your OneDrive for new files and instantly posts structured notifications to your chosen Slack channel.

Each notification includes the filename, upload timestamp, and a direct link to access the file in OneDrive. For businesses handling client documents, contracts, or creative assets, this ensures everyone stays synchronized without constant manual follow-ups.

How It Works

1. OneDrive File Detection

The workflow continuously monitors your specified OneDrive folders for new file additions. Using Make.com's reliable trigger system, it detects uploads within seconds while maintaining security through Microsoft's API.

2. File Information Processing

For each new file, the system extracts key details like filename, file type, size, and the sharable OneDrive link. This metadata is formatted into a clean message template for Slack.

3. Smart Slack Notification

The formatted message posts to your designated Slack channel with @mentions for relevant team members. The notification appears as a rich message with file type icons and quick-access buttons.

Pro tip: Add custom filters to only notify about specific file types (like .pdf or .xlsx) or folders to reduce notification noise.

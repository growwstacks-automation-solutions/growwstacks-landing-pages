What This Workflow Does

This automation solves the common problem of disjointed scheduling and task management systems. When clients book appointments through Acuity Scheduling, service providers often need to manually create corresponding tasks in ClickUp for follow-up, preparation, or service delivery. This workflow eliminates that manual process.

Every time a new appointment is scheduled in Acuity, Make.com automatically creates a corresponding task in ClickUp with all relevant details. This ensures nothing falls through the cracks and your team has immediate visibility of new appointments in your task management system.

How It Works

1. New appointment trigger

The workflow starts when Acuity Scheduling detects a new appointment booking. Make.com receives this event in real-time through Acuity's API.

2. Data extraction

Key appointment details are extracted including client name, appointment time, service type, duration, and any custom fields configured in Acuity.

3. ClickUp task creation

The workflow formats the appointment data into a ClickUp task structure and creates it in your specified ClickUp list. You can configure default assignees, priorities, and statuses.

4. Optional notifications

The workflow can optionally send Slack or email notifications to team members about new tasks created from appointments.

Pro tip: Configure different ClickUp task templates based on Acuity appointment types. For example, consultations can create tasks with one set of fields, while service appointments create tasks with different requirements.

  1. Copy this template to your Make.com account

  6. Test with a new Acuity appointment
  7. Turn on the workflow

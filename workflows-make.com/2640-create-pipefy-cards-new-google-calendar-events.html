What This Workflow Does

This automation solves the common problem of calendar events getting lost in the shuffle between scheduling and execution. Many teams schedule meetings in Google Calendar but then need to manually create corresponding tasks in Pipefy, leading to missed follow-ups and duplicated work.

The workflow automatically creates a Pipefy card every time a new event is added to your Google Calendar. This ensures every scheduled item becomes a tracked task with full context from the calendar event, including title, description, attendees, and timing.

How It Works

1. Trigger: New Google Calendar Event

The workflow starts whenever a new event is created in your connected Google Calendar. Make.com monitors your calendar in real-time and detects new events within minutes.

2. Extract Event Details

The automation pulls all relevant information from the calendar event including title, description, start/end times, location, and attendees. This data will populate the Pipefy card fields.

3. Create Pipefy Card

Using Pipefy's API, the workflow creates a new card in your specified board and phase. The card inherits properties from the calendar event, creating a seamless transition from scheduling to task management.

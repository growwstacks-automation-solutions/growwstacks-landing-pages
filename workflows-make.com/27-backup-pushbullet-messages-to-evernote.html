What This Workflow Does

This automation solves the problem of losing important Pushbullet messages in your message history. Pushbullet is great for quick sharing between devices, but its interface isn't optimized for long-term message storage or organization. Important links, notes, and files can easily get buried in your message history.

The workflow automatically copies every new Pushbullet message to Evernote as a separate note. This creates a permanent, searchable archive of your messages in Evernote's powerful organizational system. You'll never lose track of important messages again, and can easily find them later using Evernote's robust search capabilities.

How It Works

1. Trigger on new Pushbullet messages

The workflow monitors your Pushbullet account for new incoming messages. Each time a message arrives, the automation is triggered to begin processing it.

2. Extract message content

The workflow parses the Pushbullet message to extract all relevant content - including text, links, file attachments, and metadata like sender information and timestamps.

3. Format for Evernote

The message content is formatted into Evernote's note structure, preserving all important information in a clean, readable format. Attachments are downloaded and prepared for Evernote upload.

4. Create new Evernote note

The formatted content is then sent to Evernote where it's created as a new note in your specified notebook. The note includes all original message content plus useful metadata.

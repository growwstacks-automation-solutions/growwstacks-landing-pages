What This Workflow Does

This automation solves a common pain point for businesses using Xero - the lack of automatic notifications when credit notes are issued. Credit notes represent financial adjustments, refunds, or corrections to invoices, and customers often need to be informed about them promptly.

The workflow monitors your Xero account for new credit notes and automatically sends an SMS notification via ClickSend as soon as one is created. This eliminates manual follow-ups, reduces customer confusion, and ensures your clients receive immediate confirmation about financial adjustments affecting their accounts.

How It Works

1. Xero credit note detection

The workflow continuously monitors your Xero account for newly created credit notes using Make.com's Xero integration. It checks at regular intervals you specify (typically every 15-60 minutes).

2. Data extraction

When a new credit note is detected, the workflow extracts key details like customer contact information, credit note amount, reference number, and reason for issuance from Xero's API.

3. SMS message composition

The system composes an SMS using your predefined template, inserting the relevant credit note details dynamically. You can customize the message format to match your brand voice.

4. ClickSend delivery

The composed message is sent via ClickSend's API to the customer's mobile number. The workflow includes error handling to retry failed deliveries and logs all transactions.

  1. Install the template from Make.com's template library

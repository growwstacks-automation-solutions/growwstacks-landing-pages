What This Workflow Does

This automation bridges Magento 2 and ClickSend to instantly notify customers via SMS when they place orders. Instead of relying solely on email confirmations that often go unread, this workflow sends text messages with key order details directly to customers' phones.

The system triggers automatically whenever a new order is created in Magento 2. It extracts the customer's phone number and order information, formats an SMS message, and sends it through ClickSend's reliable global SMS gateway. This happens within seconds of order placement, improving customer experience and reducing support inquiries about order status.

How It Works

1. New order detection

The workflow monitors Magento 2's order API for new purchases. When an order appears, it captures all relevant details including customer contact information, products ordered, and payment method.

2. Data processing

The system validates the phone number format and checks for SMS opt-in preferences. It then composes the message using customizable templates that can include order number, item list, total amount, and estimated delivery date.

3. SMS delivery

Using ClickSend's API, the formatted message is sent to the customer's mobile number. The workflow logs delivery status and any errors for troubleshooting.

4. Confirmation and logging

After successful delivery, the system can optionally update Magento order notes or trigger follow-up actions like sending tracking information later.

Pro tip: Add a short link to the order status page in your SMS to reduce customer service inquiries by up to 40%.

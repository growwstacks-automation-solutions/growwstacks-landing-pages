What This Workflow Does

This automation solves the problem of losing important business SMS messages in crowded phone inboxes. Every time your ClickSend number receives a new text message, the workflow automatically creates a corresponding note in your Evernote account with the message content, sender information, and timestamp.

Unlike manual forwarding or screenshots, this solution preserves the original message data in a searchable format. Evernote's powerful organization features let you tag, categorize, and retrieve these messages months later when you need to reference customer requests, delivery confirmations, or other critical communications.

How It Works

1. Message Trigger

The workflow monitors your ClickSend account for new incoming SMS messages in real-time. Each new message serves as the trigger to start the automation process.

2. Data Extraction

Make.com extracts key details from each message including the sender's number, message content, and timestamp. This data is formatted for clean integration with Evernote.

3. Note Creation

The workflow creates a new Evernote note containing all message details. You can customize the note format to include specific fields or additional context that helps with organization.

Pro tip: Configure the workflow to add automatic tags based on keywords in the message or specific sender numbers for better organization.

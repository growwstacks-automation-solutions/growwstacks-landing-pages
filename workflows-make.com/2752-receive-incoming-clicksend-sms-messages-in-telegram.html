What This Workflow Does

This automation bridges the gap between ClickSend's SMS capabilities and Telegram's superior messaging interface. Whenever you receive an SMS message through your ClickSend account, the workflow instantly forwards it to your preferred Telegram chat, group, or channel.

Businesses using ClickSend for customer communications often struggle with managing SMS alongside other messaging platforms. This solution centralizes all conversations in Telegram, where you benefit from better search functionality, multi-device access, and team collaboration features.

How It Works

1. ClickSend receives an incoming SMS

The workflow triggers whenever your ClickSend number gets a new SMS message from a customer or other contact.

2. Make.com processes the message

The automation extracts key details like the sender's number, message content, and timestamp. You can add filters here to only forward messages meeting specific criteria.

3. Message forwards to Telegram

The formatted message delivers instantly to your chosen Telegram destination, with clear labeling of the original sender and time received.

Pro tip: Set up a dedicated Telegram channel for customer SMS messages to maintain organized communication history separate from personal chats.

Who This Is For

This workflow benefits any business using ClickSend for customer communications that wants to:

  • Enable team access to customer SMS messages without sharing phone access

  • Respond to messages from desktop computers or multiple devices

What You'll Need

  1. An active ClickSend account with SMS capabilities
  2. A Telegram account (personal or business)
  3. A Make.com account (free plan sufficient for basic setup)
  4. The phone number or chat ID where messages should be forwarded

Quick Setup Guide

  1. Connect your ClickSend account in Make.com using API credentials
  2. Set up the Telegram bot connection with your chat ID
  3. Configure any message filtering rules (optional)
  4. Test with a sample SMS message to verify delivery
  5. Activate the scenario for continuous operation

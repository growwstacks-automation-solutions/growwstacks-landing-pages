What This Workflow Does

This automation solves the communication gap between Telegram users and recipients who rely on traditional SMS. Many businesses and organizations use Telegram for internal communications but need to reach customers, field staff, or older demographics who don't use messaging apps. The workflow automatically bridges this gap by forwarding Telegram messages as SMS texts via ClickSend's reliable delivery network.

Key functionality includes real-time forwarding of Telegram messages (including group messages and channels), customizable message formatting for SMS, delivery confirmation tracking, and the ability to filter which messages get forwarded based on content or sender. This ensures critical communications reach all intended recipients regardless of their preferred communication method.

How It Works

1. Telegram message trigger

The workflow starts whenever a new message arrives in your configured Telegram chat, group, or channel. Make.com's Telegram bot integration captures the message content and metadata in real-time.

2. Message processing

The system can optionally filter messages based on your criteria (keywords, sender, etc.) and format the content for SMS transmission, including truncating long messages to fit SMS character limits.

3. SMS transmission via ClickSend

Processed messages are sent through ClickSend's API to deliver as SMS to your specified recipients. The workflow handles all API communication and error recovery automatically.

4. Delivery confirmation

ClickSend's delivery reports are captured and can be logged to your database or trigger notifications if messages fail to deliver after retries.

What You'll Need

  1. A Make.com account with available connections
  2. Telegram bot API credentials (can be created through @BotFather)
  3. ClickSend API credentials (username and API key from your account)
  4. A list of destination phone numbers or a system to dynamically determine recipients

Quick Setup Guide

  1. Clone the template from Make.com and open it in your account
  2. Connect your Telegram account by authenticating with your bot token
  3. Configure your ClickSend connection with API credentials
  4. Set your destination phone number(s) - can be static or pulled from another system
  5. Customize message formatting if needed (default sends Telegram message text as-is)
  6. Test with sample messages and verify SMS delivery

Pro tip: For high-volume forwarding, consider adding a delay between messages to avoid SMS carrier rate limits and potential blocking.

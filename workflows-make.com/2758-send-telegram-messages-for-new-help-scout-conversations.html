What This Workflow Does

This automation solves the critical challenge of delayed response times in customer support by instantly notifying your team via Telegram whenever a new conversation starts in Help Scout. Traditional email notifications often get missed or buried in inboxes, leading to slower response times that frustrate customers.

The workflow monitors your Help Scout account for new conversations and automatically pushes formatted alerts to designated Telegram channels or private chats. Each notification includes key details like the customer's name, subject line, and a snippet of their first message - giving your team immediate context without needing to log into Help Scout.

How It Works

1. Trigger on new Help Scout conversations

The workflow starts when Help Scout creates a new conversation in your connected mailbox. Make.com detects this event through Help Scout's API within seconds.

2. Extract conversation details

Key information like customer name, email, subject line, and message preview are pulled from the conversation. The workflow can also identify priority levels or tags if configured.

3. Format Telegram message

The system structures the notification with emoji indicators for priority, clickable links back to Help Scout, and clean formatting for mobile readability.

4. Deliver to Telegram

Make.com sends the formatted message to your pre-configured Telegram channel or private chat using Telegram's bot API. Team members receive push notifications instantly on their mobile devices.

Pro tip: Create separate Telegram channels for different priority levels or mailboxes to help your team triage notifications effectively.

What You'll Need

  1. A Make.com account (free plan sufficient for basic usage)
  2. Admin access to your Help Scout account to set up API integration
  3. A Telegram account and permissions to create bots/channels
  4. The Telegram app installed on team members' mobile devices

Quick Setup Guide

  1. Clone the template from Make.com and connect your Help Scout account
  2. Create a Telegram bot via @BotFather and note the API token
  3. Configure which Help Scout mailboxes should trigger notifications
  4. Set up your Telegram channel and add the bot as an admin
  5. Customize the message format with your preferred fields and emoji
  6. Test with a sample conversation and adjust as needed

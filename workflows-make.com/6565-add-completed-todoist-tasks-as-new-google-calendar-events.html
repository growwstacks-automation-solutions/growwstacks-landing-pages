What This Workflow Does

This automation solves the common productivity challenge of losing visibility into completed work. While Todoist excels at task management, it doesn't provide a historical timeline of when work was actually done. By automatically creating Google Calendar events for completed tasks, you gain:

A visual productivity log that shows your actual work patterns alongside scheduled events. This helps with time audits, client billing, and personal improvement tracking. The system captures task completion times (not just due dates) to reveal your real working rhythms and scheduling accuracy.

How It Works

1. Monitoring completed Todoist tasks

The workflow continuously checks your Todoist account for newly completed tasks, either on a schedule or in real-time depending on your plan. It captures the task name, completion timestamp, project, and any relevant labels.

2. Processing task details

Each completed task is processed to extract relevant information. You can configure filters to only log certain types of tasks (like those from specific projects or meeting duration thresholds) to prevent calendar clutter.

3. Creating calendar events

The system creates a new Google Calendar event for each completed task, using the completion time as the event start time. Event duration can be set as fixed (like 30 minutes) or calculated based on task estimates.

Pro tip: Add task labels as Google Calendar event tags to enable powerful filtering and reporting across both platforms.

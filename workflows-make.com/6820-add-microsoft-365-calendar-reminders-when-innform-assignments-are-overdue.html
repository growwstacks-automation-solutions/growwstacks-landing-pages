What This Workflow Does

For HR teams, L&D managers, and team leaders, tracking overdue training assignments is a constant, manual chore. You might rely on Innform's dashboard reports or hope that notification emails don't get lost in inboxes. This reactive approach leads to compliance risks, skill gaps, and last-minute scrambles to complete mandatory training.

This automation solves that by creating a proactive, fail-safe system. Every time an assignment is marked as overdue in your Innform Learning Management System (LMS), this Make.com workflow instantly creates a corresponding event in your Microsoft 365 Calendar. This transforms an invisible administrative task into a visible, time-blocked reminder that demands attention alongside your other daily priorities.

The business value is clear: it enforces accountability and closes the loop on training compliance. Instead of logging into multiple systems to chase progress, managers see overdue items directly in the tool they use to plan their day. This seamless integration ensures critical learning deadlines are never forgotten, protecting the organization and empowering employees to stay on track.

How It Works

The workflow operates on a simple but powerful trigger-action principle, connecting two critical business tools without any manual intervention.

Step 1: Monitoring Innform for Overdue Assignments

The scenario begins by periodically checking your Innform account via the Make.com Innform module. It searches for any new assignments that have passed their due date and are now flagged as "overdue." This trigger can be scheduled to run hourly, daily, or at any interval that matches your follow-up urgency.

Step 2: Formatting the Reminder Data

For each overdue assignment found, Make.com gathers key details: the course name, the assigned user's name, the original due date, and a direct link to the assignment. This data is then structured into a clear and actionable format, ready to be sent to your calendar.

Pro tip: Customize the calendar event title to include the employee's name and the word "OVERDUE" for immediate visual priority when scanning your weekly schedule.

Step 3: Creating the Microsoft 365 Calendar Event

Finally, the workflow uses the Microsoft 365 Calendar module in Make.com to create a new calendar event. The event is typically set for the current day or the next business day, with a duration that signifies a "follow-up block." The description can include all the relevant details and links, making it a one-click portal to resolve the issue.

What You'll Need

To implement this automation, you will need the following accounts and permissions:

  1. A Make.com account (free or paid plan).
  2. An active Innform subscription with administrator or API access to read assignment data.
  3. A Microsoft 365 account (business or personal) with Calendar access, and permission to create events.
  4. The ability to create secure OAuth connections between Make.com and both Innform and Microsoft 365. This is a standard, click-through process within Make.

Quick Setup Guide

You can have this automation running in under 15 minutes by following these steps:

  1. Clone the Template: Click the "Get This Workflow" button above to open the template in your Make.com account and create a copy.
  2. Connect Innform: In the first module, click to add your Innform connection. Authorize Make.com to access your Innform data (typically read-only for assignments).
  3. Connect Microsoft 365: In the Calendar module, add and authorize your Microsoft 365 account, granting permissions to create calendar events.
  4. Test the Trigger: Run the scenario once manually to check if it can find an overdue assignment in your Innform account. Use a test assignment if necessary.
  5. Customize the Event: Adjust the calendar event title, description, and timing to fit your preferred follow-up style. You can set the event for 15 minutes later today or block 30 minutes tomorrow.
  6. Schedule & Activate: Set the scenario to run on a schedule (e.g., every 6 hours) and turn it on. It will now run automatically, creating reminders as needed.

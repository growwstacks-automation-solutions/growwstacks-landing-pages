What This Workflow Does

For video editors, producers, and creative teams, the review and feedback cycle on Frame.io is where projects move forward—or get stuck. The traditional process involves constantly refreshing the Frame.io page or waiting for email notifications, which are easy to miss in a crowded inbox. This delay can add hours or even days to project timelines, leading to frustrated clients and stalled workflows.

This automation solves that by creating a real-time alert system. Every time a stakeholder adds a comment, annotation, or approval in Frame.io, this Make.com scenario instantly triggers a push notification that appears directly on your computer desktop via the Google Chrome browser. It turns passive waiting into active notification, ensuring the right person sees feedback the moment it's given.

How It Works

The workflow is a simple yet powerful connection between Frame.io and your browser, orchestrated by Make.com.

Step 1: Monitor Frame.io for New Comments

The scenario starts with a Make.com "Watch Comments" module connected to your Frame.io account. It continuously polls your specified projects or entire account for new comment activity.

Step 2: Format the Notification Data

When a new comment is detected, Make.com captures key details: the project name, the specific video or asset, the commenter's name, the timestamp, and the comment text itself. This data is structured into a clear, concise message.

Step 3: Send the Chrome Push Notification

Using the Make.com Chrome extension, the formatted message is sent as a push notification. This notification will appear on the desktop of any computer where Chrome is running and logged into the same Make.com account, even if the browser is minimized.

Pro tip: Use the Chrome extension's customization options to set a distinct notification sound. This auditory cue helps differentiate urgent client feedback from other system alerts.

Quick Setup Guide

  1. Clone the Template: Click "Get This Workflow" to copy the scenario into your Make.com account.
  2. Connect Frame.io: In the first module, authorize Make.com to access your Frame.io account. You can choose to watch all projects or specify individual ones.
  3. Install the Chrome Extension: Add the "Make - Customizable Webhooks" extension from the Chrome Web Store to your browser and log in with your Make.com credentials.
  4. Test the Flow: Add a test comment to a Frame.io asset. Within moments, you should see a push notification appear on your desktop.
  5. Customize (Optional): Adjust the notification message format in Make.com to include the information most relevant to your team.

What This Workflow Does

How many great ideas or urgent action items have you discussed in a Telegram chat, only to forget about them moments later? The constant switching between communication apps and your task manager kills productivity and leads to missed deadlines. This workflow solves that by creating an instant, automated bridge between conversation and execution.

Every time a new message arrives in your specified Telegram channel, group, or private chat, this Make.com automation captures it and creates a corresponding task inside Amazing Marvin. The task title, description, and even the sender's name can be automatically populated, turning a casual message into a tracked, manageable to-do item in your trusted productivity system. It's like having a personal assistant who listens to your team's conversations and organizes the work for you.

How It Works

The automation runs on a simple but powerful trigger-action principle, connecting two best-in-class apps without any manual intervention.

Step 1: Monitor Telegram for New Messages

The workflow starts by watching a specific Telegram source you choose—this could be a private channel for work notes, a team project group, or even a chat with yourself. Make.com acts as the listener, instantly detecting when a new message is posted.

Step 2: Process the Message Content

Once a message is detected, the automation can parse its contents. It extracts the text, identifies the sender, and can apply basic filters (e.g., only create tasks from messages containing keywords like "TODO" or "ACTION"). This step ensures only relevant messages become tasks.

Step 3: Create a Task in Amazing Marvin

The processed data is then sent to Amazing Marvin via its API. A new task is created with the message content as the title or description. You can configure the automation to add the task to a specific project, assign a priority, set a due date, or add labels—all based on rules you define.

Pro tip: Use a dedicated Telegram channel as your "inbox." Send quick voice notes or typed thoughts there throughout the day. This workflow will process them all into your Amazing Marvin inbox for later review and scheduling, creating a flawless capture habit.

Quick Setup Guide

You can have this automation running in under 10 minutes.

  1. Get the Template: Click the "Get This Workflow" button to copy the ready-made scenario to your Make.com account.
  2. Connect Telegram: In the first module, authorize Make.com to access your Telegram account and specify the exact chat or channel to watch.
  3. Connect Amazing Marvin: In the task creation module, authorize the connection to your Amazing Marvin account using your API key (found in Marvin's Settings > Integrations).
  4. Map the Data: Ensure the Telegram message text is mapped to the "Title" or "Description" field in the Amazing Marvin task creation module. You can add static project names or labels here.
  5. Test & Activate: Send a test message to your chosen Telegram source. Check if a task appears in your Amazing Marvin account. Once confirmed, turn the scenario on. It will now run automatically.

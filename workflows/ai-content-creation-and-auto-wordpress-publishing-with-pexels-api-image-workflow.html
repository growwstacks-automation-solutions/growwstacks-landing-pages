Generate articles with AI, source images automatically, and publish directly to WordPress—all from a simple Google Sheets content calendar.

What This Workflow Does

Manual blog content creation is a time-consuming bottleneck for marketing teams. Between brainstorming topics, writing drafts, finding images, formatting, and publishing, a single post can take 5-8 hours. This workflow eliminates that manual effort by automating the entire process from idea to published article.

The system connects Google Sheets (your content calendar), AI writing tools (Google Gemini), image APIs (Pexels), and your WordPress site into a seamless pipeline. You manage everything through a simple spreadsheet while the automation handles writing, image sourcing, and publishing with zero daily intervention.

Beyond just saving time, this ensures consistent publishing schedules, maintains brand voice through AI prompts, and creates SEO-optimized content that drives organic traffic—all while freeing your team to focus on strategy rather than execution.

How It Works

The workflow operates through two main pathways, both triggered from an interactive form that lets you choose between generating content or adding new ideas.

1. Content Generation Path

The system first checks your Google Sheet for topics marked "not generated." It selects the next idea, sends it to AI to write a complete article, generates search keywords for images, finds relevant photos via Pexels API, creates the WordPress post, uploads the image as a featured image, and updates the sheet with publication details.

2. Idea Management Path

When you need fresh content ideas, the workflow prompts for a general topic, uses AI to generate five SEO-friendly blog titles, adds them to your Google Sheet as "not generated" items, and can loop to add more topics or complete the session.

3. Integration & Coordination

All components communicate through structured data flows: Google Sheets provides the source material, AI models generate text, HTTP requests fetch images, and WordPress REST API handles publishing. Interactive forms provide user control points throughout the process.

What You'll Need

  1. n8n instance (cloud or self-hosted) with workflow execution capabilities
  2. Google Sheets account with a prepared content calendar spreadsheet
  3. WordPress website with REST API enabled and authentication credentials
  4. Google Gemini API key for AI content generation (available through Google AI Studio)
  5. Pexels API key (free tier available) for image sourcing
  6. Basic understanding of how to configure API connections in n8n

Pro tip: Start with a test WordPress site and a simple Google Sheet with just 3-5 topic ideas. Run the workflow once to verify all connections work before scaling to your production blog.

Quick Setup Guide

Follow these steps to implement this automation in under 30 minutes:

  1. Download the template using the button above and import it into your n8n instance.
  2. Configure credentials for Google Sheets, WordPress, and Google Gemini in their respective nodes.
  3. Update API endpoints in the Pexels and WordPress HTTP Request nodes with your actual keys and site URL.
  4. Prepare your Google Sheet with columns for Prompt, Generated (set to "no"), Date, Title, and Post ID.
  5. Test with one idea by adding a simple topic to your sheet and triggering the workflow.
  6. Monitor the execution in n8n's workflow viewer to ensure all steps complete successfully.
  7. Schedule or webhook the workflow to run daily, weekly, or via manual trigger as needed.

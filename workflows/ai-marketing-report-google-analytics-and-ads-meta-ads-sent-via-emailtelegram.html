What This Workflow Does

This automation solves the tedious, time-consuming task of manually compiling weekly marketing performance reports. It connects to your key advertising and analytics platforms—Google Analytics (for multiple domains), Google Ads, and Meta Ads—to pull the last 7 days of data along with the same period from the previous year for comparison.

The workflow then structures this data, uses an AI model (like OpenAI, Anthropic, or Google's Gemini) to analyze trends, identify top performers, and spot anomalies. It generates a clear, concise summary and a formatted data table. Finally, it delivers this intelligence directly to your inbox as a detailed email report and sends a condensed, actionable version to a designated Telegram channel for quick mobile updates.

By automating this entire process, you eliminate hours of manual spreadsheet work, reduce human error, and ensure stakeholders receive consistent, data-driven insights every single week without fail.

How It Works

The workflow is built as a sequence of logical steps in n8n, making it easy to understand, customize, and troubleshoot.

1. Scheduled Trigger & Data Collection

A Schedule Trigger initiates the workflow (e.g., every Monday at 7 AM). It then executes a series of sub-workflows or direct API calls to fetch raw performance data from your connected platforms for the defined date ranges.

2. Data Structuring & Year-Over-Year Comparison

The raw data from each platform is normalized and merged. Key metrics (like impressions, clicks, conversions, spend, ROAS) are extracted and organized. The workflow simultaneously retrieves data from the same 7-day period in the previous year, enabling immediate performance comparison and trend analysis.

3. AI-Powered Analysis & Summarization

The consolidated current and historical data is passed to an AI Agent or LLM node. The AI is prompted to analyze the numbers, highlight significant changes, pinpoint winning campaigns or underperformers, and generate a narrative summary in plain English, focusing on business implications.

4. Report Generation & Distribution

The AI-generated summary and the structured data table are compiled into a clean HTML email body. This report is sent via SMTP to your predefined list of recipients. In parallel, the AI creates a much shorter, punchy version of the key takeaways, which is sent instantly via the Telegram Bot to keep your team updated on the go.

What You'll Need

  1. n8n instance (Cloud or Self-Hosted).
  2. Google Analytics API access with credentials for each property/domain you wish to track.
  3. Google Ads API access via a developer token and OAuth credentials.
  4. Meta Ads (Facebook Graph API) access with a system user token and appropriate ad account permissions.
  5. AI API access (OpenAI, Anthropic, Google AI, or a local Ollama instance) for the analysis step.
  6. SMTP credentials (from a service like SendGrid, Gmail, or your company email server) to send the email report.
  7. Telegram Bot Token & Chat ID (optional, for mobile notifications).

Quick Setup Guide

Follow these steps to get your automated marketing report running in under an hour.

  1. Import the Template: Download the JSON file above and import it into your n8n workspace.
  2. Configure Credentials: In n8n, set up credentials for Google Analytics, Google Ads, Meta Ads, your chosen AI provider, and your SMTP service.
  3. Map Your Data Sources: Update the workflow nodes with your specific Google Analytics Property IDs, Google Ads Customer ID, and Meta Ad Account ID.
  4. Set Recipients: Enter the destination email addresses and your Telegram Chat ID in the respective Send Email and Telegram nodes.
  5. Customize the AI Prompt: Tweak the prompt in the AI node to ask for the specific insights and format you prefer in your summary.
  6. Test & Activate: Run the workflow once manually to ensure all connections work and the output looks correct. Then, set the Schedule Trigger to your desired frequency (e.g., weekly).

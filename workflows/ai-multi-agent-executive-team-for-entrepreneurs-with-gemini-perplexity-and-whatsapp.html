What This Workflow Does

This workflow creates a complete virtual executive team powered by artificial intelligence, designed specifically for startup founders and small business owners who need comprehensive support but lack the resources for a full team. It addresses the critical challenge of wearing multiple hats by distributing key business functions across specialized AI agents that work together seamlessly.

The system transforms WhatsApp into your business command center, where you can interact naturally with your AI team. Instead of manually researching competitors, planning marketing campaigns, tracking tasks, and managing schedules, you delegate these functions to your virtual team. The AI President coordinates everything, ensuring you receive consolidated, actionable insights and support exactly when needed.

How It Works

The workflow operates through a sophisticated coordination system where each AI agent specializes in specific business functions while maintaining communication with the central controller.

1. President Agent Coordination

The President agent acts as the central intelligence, analyzing incoming WhatsApp messages and routing tasks to the appropriate specialist agent. It maintains context across conversations and ensures all agents work toward coherent business objectives.

2. Sales & Marketing Director Operations

This agent continuously scans market opportunities using SerpAPI, identifying potential leads, trending keywords, and competitive gaps. It suggests targeted marketing campaigns and analyzes engagement metrics to recommend strategy adjustments.

3. Business Research Director Analysis

Powered by Perplexity AI, this agent monitors competitor activities, industry developments, and market shifts. It provides concise competitive intelligence reports and early warning signals about market changes that could impact your business.

4. Accountability Director Support

This agent maintains progress tracking, sends motivational reminders, and ensures consistency in business operations. It helps establish productive routines and provides gentle nudges when tasks fall behind schedule.

5. Executive Secretary Administration

Connected to Google services, this agent handles scheduling, email management, and follow-up communications. It automates meeting coordination, sends summary notes, and manages administrative workflows.

6. WhatsApp Integration & Delivery

All agent responses are consolidated and delivered through WhatsApp Business API, providing a unified, conversational interface. The system maintains conversation history and context for personalized interactions.

What You'll Need

  1. n8n instance (cloud or self-hosted) with workflow execution capabilities
  2. Google Cloud account with OAuth credentials for Sheets, Calendar, and Gmail integration
  3. Meta Developer account with WhatsApp Business API access for messaging
  4. API keys for Gemini AI, Perplexity AI, and SerpAPI services
  5. Supabase account for vector storage and persistent memory management
  6. Basic understanding of API credential management and n8n node configuration

Pro tip: Start with one or two agents first (like Research and Accountability) before enabling the full team. This allows you to validate the system's value and refine prompts before scaling to complete automation.

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance using the import workflow feature
  2. Configure API credentials in each corresponding node: Gemini, Perplexity, SerpAPI, and WhatsApp Business
  3. Set up Google OAuth connection for Calendar, Sheets, and Gmail integration nodes
  4. Create Supabase project and configure the vector store nodes for memory persistence
  5. Customize system prompts for each agent to align with your specific business context and goals
  6. Test the workflow by sending WhatsApp messages and verifying agent responses
  7. Activate the workflow and begin interacting with your virtual executive team

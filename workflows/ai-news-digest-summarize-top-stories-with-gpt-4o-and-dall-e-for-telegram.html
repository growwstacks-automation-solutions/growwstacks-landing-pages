What This Workflow Does

In the fast-moving world of artificial intelligence, staying informed is a competitive necessity—but manually scanning dozens of news sites is a massive time drain. This automation solves that problem by acting as your personal AI news editor.

The workflow automatically collects the latest articles from top tech and AI publications, uses GPT-4o to generate concise, engaging summaries, creates a unique header image with DALL-E, and delivers a polished digest directly to your Telegram chat every morning. It transforms information overload into a streamlined, actionable briefing.

Beyond simple aggregation, it includes an interactive "Chat Mode" where you can ask follow-up questions about the news directly within n8n, turning your digest into a conversational knowledge base.

How It Works

Step 1: Scheduled News Collection

Every day at 8:00 AM (configurable), the workflow triggers and fetches RSS feeds from configured sources like Google News AI, The Verge, and TechCrunch. It pulls in the latest headlines and article snippets.

Step 2: Smart Filtering & Deduplication

A Code node processes the incoming articles, removing duplicates based on title and URL similarity, and ranks them by publication time to select the top 5 most recent and relevant stories.

Step 3: AI-Powered Summarization

The selected articles are passed to an AI Agent (GPT-4o) with a custom system prompt. The agent rewrites headlines for clarity, extracts key insights, and generates a concise 2–3 sentence summary for each story in a consistent, engaging tone.

Step 4: Visual Content Creation

Simultaneously, DALL-E 3 generates a unique, futuristic header image based on the day's overarching news theme (e.g., "AI regulation breakthroughs"). This visual makes the digest more engaging and shareable.

Step 5: Formatted Delivery to Telegram

The summaries and image are compiled into a well-formatted message using Markdown and emojis. The workflow then sends this complete digest to your specified Telegram chat or channel via the Telegram Bot API.

Step 6: Interactive Q&A (Optional Branch)

A separate workflow branch listens for questions via the n8n Chat interface. You can ask about today's news or general AI topics, and the AI Agent provides answers based on the aggregated content and its knowledge.

What You'll Need

  1. n8n Instance: A self-hosted n8n setup or n8n.cloud account that supports LangChain nodes (version 1.0+ recommended).
  2. OpenAI API Key: An account with access to GPT-4o (or GPT-4o-mini) and DALL-E 3 for summarization and image generation.
  3. Telegram Account & Bot: A Telegram account to create a bot via @BotFather and obtain your numeric Chat ID using @userinfobot.
  4. Basic Configuration Skills: Ability to paste API keys and Chat IDs into the workflow settings node.

Quick Setup Guide

Follow these steps to get your automated news digest running in under 10 minutes:

  1. Download & Import: Click the download button above to get the JSON file. In your n8n instance, go to Workflows → Import from File and select the downloaded template.
  2. Configure Credentials: In the "Credentials" section of n8n, set up new credentials for OpenAI and Telegram using the API keys and bot token you obtained.
  3. Set Your Chat ID: Open the "Workflow Configuration" node in the canvas. Find the telegramChatId field and paste your numeric Chat ID (obtained from @userinfobot).
  4. Adjust Sources (Optional): In the same configuration node, you can modify the RSS feed URLs to track different publications or topics.
  5. Activate & Test: Toggle the workflow to "Active." Manually trigger it once to test. Check your Telegram for the first digest!
  6. Schedule: The workflow is pre-set to run daily at 8 AM UTC. Adjust the "Schedule Trigger" node to your preferred time zone and frequency.

Pro tip: For team use, create a Telegram group, add your bot, and use the group's Chat ID. The entire team will receive the daily digest, fostering discussion.

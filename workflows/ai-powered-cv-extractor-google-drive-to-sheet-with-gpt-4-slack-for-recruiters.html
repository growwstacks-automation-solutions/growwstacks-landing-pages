What This Workflow Does

For recruiters and HR teams, the manual process of opening each CV, copying details into a spreadsheet, and notifying the team is a massive time sink. This intelligent automation solves that by creating a hands-free pipeline from resume submission to structured data.

The workflow monitors a designated Google Drive folder for new or updated PDF resumes. When a file appears, it downloads the document, uses AI (GPT-4) to intelligently parse and extract key candidate information—like name, contact details, skills, and experience—and then neatly logs this structured data into a Google Sheet. Finally, it posts a summary to a Slack channel, keeping the entire hiring team instantly in the loop.

This transforms a chaotic folder of PDFs into a searchable, sortable candidate database and an active notification system, saving hours per week and ensuring no candidate gets lost in the shuffle.

How It Works

The automation follows a logical, step-by-step process to ensure accuracy and reliability.

1. Trigger & File Detection

The workflow is triggered whenever a new PDF file is added to a specific Google Drive folder. It can also be set to watch for file updates, ensuring the database stays current if a candidate submits an updated CV.

2. Document Processing & AI Extraction

The system downloads the PDF and converts its text into a readable format. This raw text is then sent to an AI model (like GPT-4) with a carefully crafted prompt instructing it to identify and return specific data points: name, email, phone, job title, years of experience, key skills, education, and location.

3. Data Validation & Transformation

The AI's response is parsed from JSON into clean, individual data fields. A function node can be added to validate formats (e.g., checking for valid email structures) or to standardize data (e.g., converting "5+ years" to a numerical value).

4. Database Update (Google Sheets)

The clean data is then upserted into a Google Sheet. Using the candidate's email as a unique identifier, the workflow checks if a record already exists. If it does, it updates the row; if not, it appends a new row. This prevents duplicates and keeps the database accurate.

5. Team Notification (Slack)

Once the data is saved, a formatted message is sent to a designated Slack channel. This alert includes the candidate's name, role, and a link to their profile in the sheet, enabling immediate team discussion and action.

Pro tip: Use the candidate's email as the unique key in your Google Sheet. This prevents duplicate entries when the same person applies multiple times or updates their resume.

What You'll Need

  1. An n8n Instance: A self-hosted n8n setup or an n8n.cloud account.
  2. Google Service Account: Credentials to access a specific Google Drive folder and a Google Sheet.
  3. OpenAI API Key: Access to the GPT-4 or GPT-3.5 API for intelligent text extraction.
  4. Slack Webhook URL: An incoming webhook for a Slack channel where notifications will be posted.
  5. A Google Drive Folder: A dedicated folder where resumes (PDF format) will be uploaded.
  6. A Prepared Google Sheet: A Sheet with column headers matching the data you want to extract (e.g., Email, Full Name, Job Title, Skills).

Quick Setup Guide

Follow these steps to get this automation running in your n8n environment.

  1. Download & Import: Click the "Download Template" button above and import the JSON file into your n8n instance.
  2. Configure Google Drive: In the "Google Drive Trigger" node, authenticate with your Google account and select the specific folder to monitor for new PDF files.
  3. Set Up OpenAI: In the "OpenAI Chat Model" node, add your API key and review the system prompt. You can customize the prompt to extract different or additional fields.
  4. Connect Google Sheets: In the "Google Sheets" node, authenticate and specify the ID of your target spreadsheet and worksheet. Ensure the column headers match the data being sent.
  5. Add Slack Webhook: In the "Slack" node, paste the incoming webhook URL for your desired channel. Customize the message format if needed.
  6. Test & Activate: Drop a sample resume PDF into your monitored Google Drive folder and activate the workflow. Check the Sheet for the new entry and your Slack channel for the notification.

What This Workflow Does

This AI-powered stock analysis assistant transforms how traders and investors process market information. Instead of spending hours manually analyzing charts and reading financial news, this automation delivers professional-grade technical analysis directly to your Telegram messages. The system combines conversational AI with visual chart interpretation to provide comprehensive market insights.

The workflow addresses the common problem of information overload in trading. With countless stocks to monitor and complex indicators to track, individual traders often miss opportunities or make decisions based on incomplete data. This automation solves that by continuously analyzing market conditions, identifying patterns, and delivering clear, actionable insights exactly when you need them.

By connecting Telegram with Claude AI for reasoning and GPT-4O Vision for chart analysis, you get a virtual trading assistant that understands both numerical data and visual patterns. Whether you're monitoring specific stocks, tracking market sentiment, or looking for entry/exit signals, this system provides the analytical firepower previously available only to institutional traders with expensive software subscriptions.

How It Works

1. Telegram Message Reception

The workflow begins when you send a stock symbol or analysis request through Telegram. The system captures your message and initiates the analysis pipeline, sending typing indicators to show the AI is processing your request.

2. Data Collection & Processing

Market data is gathered from financial APIs, including price history, volume, and key technical indicators. The system prepares both numerical data for Claude's analysis and chart images for GPT-4O's visual interpretation.

3. Dual AI Analysis

Claude 3.5 Sonnet processes the numerical data, applying technical analysis principles and generating reasoned insights about market conditions. Simultaneously, GPT-4O Vision examines the chart images, identifying patterns, support/resistance levels, and visual anomalies that might not be apparent from numbers alone.

4. Insight Synthesis

The workflow merges findings from both AI systems, creating a comprehensive analysis that combines quantitative metrics with qualitative visual observations. This dual-approach ensures you get both the "what" (numerical indicators) and the "why" (pattern context).

5. Response Delivery

The synthesized analysis is formatted into clear, actionable insights and delivered back to your Telegram chat. The system maintains conversation context, allowing for follow-up questions and deeper dives into specific aspects of the analysis.

What You'll Need

  1. n8n instance (self-hosted or cloud) version 1.7+ with Langchain nodes enabled
  2. Telegram Bot Token from BotFather to enable the messaging interface
  3. OpenRouter API key for accessing Claude 3.5 Sonnet
  4. OpenAI API key with GPT-4O Vision access for chart analysis
  5. Chart-IMG API access for generating technical analysis charts
  6. Financial data source (like Alpha Vantage, Yahoo Finance API, or your preferred market data provider)

Security Note: Never hardcode API keys directly into the workflow JSON. Always use n8n's credential manager for secure storage of all sensitive authentication tokens.

Quick Setup Guide

Follow these steps to implement this AI stock analysis assistant in your environment:

  1. Import the workflow into your n8n instance using the downloaded JSON file
  2. Configure credentials in n8n's credential manager for Telegram, OpenRouter, OpenAI, and your chart generation service
  3. Set up webhooks for Telegram to receive messages from your bot
  4. Test the connection by sending a stock symbol to your Telegram bot
  5. Customize analysis parameters like timeframes, indicators, and risk tolerance levels
  6. Add financial disclaimers to ensure compliance with regulatory requirements
  7. Deploy and monitor the workflow, adjusting AI parameters based on performance

Pro tip: Start with a small set of stocks you're familiar with to validate the AI's analysis against your own knowledge. Gradually expand to more symbols as you gain confidence in the system's accuracy.

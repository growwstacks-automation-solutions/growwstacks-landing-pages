AI-Powered Tarot Readings via iMessage

Automate mystical guidance & tarot interpretations using GPT & Blooio. Deliver soulful readings instantly via iMessage.

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automation transforms your iMessage into a mystical tarot reading channel. Clients or users can send a text question or upload a photo of their tarot spread, and the system instantly returns a thoughtful, AI-generated interpretation. It blends modern messaging convenience with timeless spiritual guidance, creating an accessible, scalable way to offer personalized insights.

The workflow listens for incoming iMessage via Blooio, checks if the message contains an image, uses AI vision to analyze tarot cards if present, crafts a poetic reading using GPT, and sends a comforting, insightful reply back through iMessage. It also handles onboarding greetings and virtual card draws for users who just want a quick daily pull.

For spiritual businesses, coaches, or wellness creators, this automation eliminates the manual time spent on individual readings while maintaining a high-quality, emotionally resonant experience. It turns a simple text channel into a revenue-generating service or a community engagement tool.

How It Works

1. Trigger: iMessage Webhook

The workflow starts when a user sends a message or image to your dedicated Blooio-connected iPhone number. Blooio captures the incoming content and passes it to n8n as a webhook trigger, containing the user's text, optional image attachment, and sender details.

2. Filter & Onboarding

A logic check ensures the message isn't from your own number to prevent loops. If the message is a simple "Hi" or "hello," the workflow sends a welcoming onboarding message explaining how to request a reading or upload a card photo.

3. Image Detection & AI Analysis

If an image is attached, the workflow routes it to an AI vision model (like OpenAI's GPT-4V) which identifies tarot cards, their positions, and symbolic elements. The AI combines this visual data with the user's written question to form a contextual understanding.

4. Reading Generation

Using a custom prompt persona—"Mystic Tarot Reader"—the AI generates a warm, poetic interpretation. The prompt instructs the AI to avoid technical jargon, focus on emotional insight, and frame the reading as guidance rather than prediction.

5. Response Delivery

The final crafted reading is sent back through Blooio's iMessage API to the user's phone. The reply feels personal, immediate, and spiritually attuned, closing the loop within seconds.

What You'll Need

  1. Blooio Account with a Dedicated or Enterprise plan (required for image support).
  2. OpenAI API Key or another AI model with vision capabilities.
  3. n8n Instance (cloud or self-hosted) to run the workflow.
  4. A dedicated iPhone connected to Blooio for iMessage sending/receiving.
  5. Optional: A tarot deck or reference imagery if you want to train/customize the AI's symbolic recognition.

Pro tip: Use Blooio's multi-channel support to expand this workflow to WhatsApp, Telegram, or SMS later. The core AI interpretation remains the same; you just swap the trigger and send nodes.

Quick Setup Guide

  1. Import the template into your n8n workspace using the downloaded JSON file.
  2. Sign up for Blooio, get your API key, and paste it into the HTTP Request node's Authorization header as a Bearer token.
  3. Configure your OpenAI node with your API key and adjust the "Mystic Tarot Reader" persona prompt to match your preferred tone.
  4. Test with a photo of a tarot card or a simple text question sent to your Blooio number.
  5. Monitor the logs to ensure images are processed and replies are sent successfully.
  6. Expand functionality by adding nodes to log readings to Airtable, trigger follow-up emails, or integrate with your booking system.

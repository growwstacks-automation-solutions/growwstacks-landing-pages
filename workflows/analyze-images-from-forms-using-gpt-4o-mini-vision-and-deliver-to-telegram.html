What This Workflow Does

This automation solves a common business problem: manually reviewing images submitted by customers, team members, or through public forms. Whether it's expense receipts, product photos, inspection reports, or support tickets with screenshots, visual data requires human interpretation—until now.

The workflow automatically processes images submitted through any form (Typeform, Google Forms, JotForm, or n8n's built-in form trigger), sends them to OpenAI's GPT-4o-mini Vision model for analysis, and delivers the AI-generated description directly to a designated Telegram chat or channel. This creates an instant feedback loop where visual information becomes actionable text insights.

Instead of someone constantly monitoring form submissions or email attachments, your team receives concise, analyzed summaries in their everyday communication tool. The AI can be instructed to look for specific elements—extract text, identify objects, assess quality, or flag issues—making it adaptable for countless business scenarios.

How It Works

The automation follows a clean three-step process that transforms raw images into intelligent notifications.

1. Form Submission Trigger

The workflow starts when a user submits a form containing image attachments. This could be a customer support form, a field service report, an expense submission, or a quality inspection portal. The trigger captures all form data, including image URLs or binary files.

2. OpenAI Vision Analysis

Each submitted image is sent to OpenAI's Vision API with a custom prompt you define. For example: "Extract the total amount, date, and vendor from this receipt" or "Describe any visible damage in this insurance claim photo." The AI returns a detailed textual analysis based on your specific requirements.

3. Telegram Delivery

The analyzed text, along with original submission details (user email, timestamp, form responses), is formatted into a clear message and sent to your pre-configured Telegram chat or channel. Team members get instant notifications with all context needed for follow-up.

Pro tip: Use Telegram's bot features to create interactive responses. Add buttons to approve/reject submissions or request more information directly from the notification.

Quick Setup Guide

Get this automation running in under 15 minutes with these steps:

  1. Download the template using the button above and import it into your n8n instance.
  2. Configure your form trigger by connecting your form platform (or set up n8n's webhook form trigger).
  3. Add your OpenAI API key in the OpenAI Vision node and customize the analysis prompt for your use case.
  4. Set up Telegram credentials by creating a bot via @BotFather and adding your bot token and chat ID to the Telegram node.
  5. Test the workflow by submitting a sample image through your form and verifying the analysis appears in Telegram.
  6. Activate the workflow and start processing images automatically.

Pro tip: For production use, add an error handling step that retries failed API calls and notifies administrators if the workflow stops. Consider adding a database node to log all analyses for audit trails.

What This Workflow Does

This automation solves a critical business problem: information trapped in audio recordings. Teams waste countless hours manually transcribing meetings, reviewing podcasts, or analyzing customer calls. Important insights get lost, action items are forgotten, and decision-makers lack structured summaries.

The workflow automatically converts audio content into executive-ready briefs. It handles both direct file uploads and Google Drive links, transcribes speech to text using AssemblyAI, analyzes the content with GPT-4 for sentiment and key insights, and delivers formatted reports via email. What used to take hours of manual work now happens automatically with consistent quality.

How It Works

1. Audio Input & Processing

The workflow accepts audio through two channels: direct file upload or Google Drive link. It automatically validates the file format and prepares it for transcription. For Drive links, it extracts the file ID and fetches the content securely.

2. AI-Powered Transcription

Using AssemblyAI's speech-to-text engine, the audio is converted to accurate text with punctuation and speaker identification when available. The workflow includes intelligent polling to handle longer recordings that take more time to process.

3. Intelligent Analysis & Summarization

GPT-4 analyzes the transcript to extract executive summary, sentiment score, key discussion points, action items, notable quotes, and relevant topics. This transforms raw text into structured business intelligence.

4. Automated Reporting & Distribution

A professionally formatted email is generated with all insights and delivered to specified recipients. The report includes both high-level summaries and detailed analysis for different stakeholder needs.

What You'll Need

  1. AssemblyAI API key for speech-to-text transcription
  2. OpenAI API key for GPT-4 analysis and summarization
  3. Google Drive access (if using Drive links for audio files)
  4. Gmail or email service credentials for report delivery
  5. n8n instance (cloud or self-hosted) to run the workflow

Quick Setup Guide

  1. Download the template using the button above
  2. Import the JSON file into your n8n instance
  3. Configure credentials for AssemblyAI, OpenAI, and Gmail
  4. Test with a sample audio file (meeting recording or podcast)
  5. Customize the email template and recipient list as needed
  6. Set up triggers (form submissions or scheduled runs)

Pro tip: For best results with longer recordings (30+ minutes), consider splitting the audio into 10-15 minute chunks before processing. This improves transcription accuracy and makes the AI analysis more focused.

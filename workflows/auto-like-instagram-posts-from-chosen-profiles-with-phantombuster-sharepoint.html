This n8n workflow automates Instagram engagement by systematically liking posts from specific profiles you want to build relationships with. It combines Phantombuster's reliable Instagram automation with SharePoint's secure data management to create a scalable, trackable system that maintains your brand presence 24/7 while freeing up your team for higher-value activities.

How It Works

The automation follows a smart, controlled sequence to ensure effective engagement while minimizing detection risk.

1. Scheduled Trigger & Data Collection

The workflow activates on a schedule (default: every 2 hours). It first retrieves your list of target Instagram profiles from a SharePoint CSV file, then uses Phantombuster to extract recent posts from each profile while rotating through authenticated session cookies for security.

2. Intelligent Post Selection

From the collected posts, the system randomly selects one that hasn't been liked previously (checked against your "already liked" tracking file). This randomization creates natural-looking engagement patterns rather than predictable behavior that platforms might flag.

3. Controlled Engagement & Tracking

Phantombuster's autolike agent engages with the selected post, then the workflow updates your tracking files in SharePoint. Built-in wait nodes throttle activity to approximately 12 likes per profile daily—a sustainable rate that mimics human behavior while building consistent presence.

What You'll Need

  1. n8n instance (version 1.33 or higher) either self-hosted or cloud
  2. Phantombuster account with Growth plan for Instagram agent access
  3. Microsoft 365 subscription with SharePoint for secure file storage
  4. Instagram business or creator account with proper authentication
  5. Target profile list of Instagram accounts you want to engage with

Pro tip: Start with a small list of 5-10 target profiles and monitor engagement rates for a week before scaling. This helps you refine your targeting strategy and ensure you're engaging with accounts that actually align with your goals.

Quick Setup Guide

  1. Import the template into your n8n instance using the downloaded JSON file
  2. Configure credentials for Phantombuster API and SharePoint OAuth2 connections
  3. Set up SharePoint folder named "Phantombuster" with three files: instagram_session_cookies.txt, instagram_posts_already_liked.csv, and profiles_instagram.csv
  4. Populate your data – add target profile URLs to the CSV, session cookies to the text file
  5. Adjust schedule if needed (default every 2 hours works well for most use cases)
  6. Activate the workflow and monitor initial runs to ensure everything works correctly

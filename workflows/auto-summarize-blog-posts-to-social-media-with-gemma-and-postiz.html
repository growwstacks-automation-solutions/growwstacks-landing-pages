What This Workflow Does

This automation solves a common content marketing challenge: consistently promoting blog content across social media platforms without manual effort. Each time you publish a new blog post, this workflow automatically fetches it via RSS, creates an engaging AI-powered summary, extracts relevant images, generates hashtags, and posts to Facebook, LinkedIn, X (Twitter), and Instagram simultaneously.

The system ensures your social media content is optimized for each platform's specific requirements—character limits, image dimensions, and posting formats. It prevents duplicate posts and maintains a professional, consistent brand presence while freeing your team from repetitive posting tasks.

Pro tip: Configure the AI prompt to match your brand voice. A technical blog might use a professional tone, while a lifestyle blog could use casual, engaging language.

How It Works

1. RSS Feed Monitoring

The workflow regularly checks your blog's RSS feed for new content. When a new post is detected, it extracts the title, content, link, and featured image.

2. AI-Powered Summarization

Using a local LLM (like Gemma via Ollama) or cloud AI service, the system creates a concise summary optimized for social media. It extracts key points while maintaining your article's core message.

3. Hashtag Generation & Optimization

The AI analyzes the content to generate 3-5 relevant hashtags that increase discoverability. The system reserves character space specifically for these hashtags in each platform's limit.

4. Cross-Platform Posting

Via the Postiz API, the workflow posts the summarized content, image, and hashtags to all connected social media accounts. Each post is tailored to the platform's specific requirements.

What You'll Need

  1. A self-hosted or cloud n8n instance (version 1.107.3 or later)
  2. Access to your blog's RSS feed URL
  3. Postiz API credentials for social media posting
  4. Ollama setup with Gemma model (for local AI) or alternative cloud AI service
  5. Admin access to your social media business accounts

Quick Setup Guide

  1. Import the template: Download the JSON file and import it into your n8n instance.
  2. Configure RSS feed: Replace the placeholder URL with your actual blog RSS feed.
  3. Set up AI integration: Connect your Ollama instance or alternative AI service.
  4. Connect social accounts: Add your Postiz API credentials and link your social platforms.
  5. Test and activate: Run a test with an existing blog post, then activate the workflow schedule.

Pro tip: Start with one social platform during testing. Once confirmed working, add additional platforms to the Postiz configuration.

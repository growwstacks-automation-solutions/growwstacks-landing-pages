Auto Workflow Backup to Google Drive

Automated export of all your workflows with scheduled cloud backups

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automation solves the critical problem of workflow loss in n8n by creating scheduled backups of all your automations. Manual exports are time-consuming and easily forgotten, leaving your business vulnerable to accidental deletions or configuration errors.

The workflow automatically exports each workflow as a separate JSON file to Google Drive, organized in dated folders. This provides version history and disaster recovery capabilities without any manual effort.

How It Works

1. Scheduled Trigger

The workflow can be set to run automatically on your preferred schedule (weekly recommended).

2. Workflow Retrieval

Connects to your n8n instance via API to fetch a complete list of all workflows.

3. Folder Creation

Creates a new timestamped folder in your specified Google Drive location.

4. Individual Exports

Each workflow is converted to JSON format and saved as a separate file in the backup folder.

5. Cleanup (Optional)

Can be configured to automatically delete older backups after a set retention period.

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure Google Drive OAuth credentials
  3. Set your target folder ID in the "Create Folder" node
  4. Configure the schedule trigger to your preferred backup frequency
  5. Test with manual trigger and verify files appear in Google Drive

Pro tip: For mission-critical workflows, consider running this daily and storing backups in multiple locations.

What This Workflow Does

Creating professional video ads traditionally requires multiple tools, designers, editors, and significant time investment. This workflow automates the entire pipeline—from initial idea to published ad—using cutting-edge AI tools. It solves the problem of scaling video content production while maintaining quality and consistency across campaigns.

The automation begins with a simple Telegram input containing a product image and caption. From there, it generates multiple AI visuals using different engines, creates cinematic video ads with voiceovers, merges clips, and publishes to all major social platforms automatically. This eliminates manual bottlenecks and allows marketers to test dozens of ad variations quickly.

How It Works

1. Input & Idea Generation

The workflow starts when you send a product image and caption via Telegram. An AI agent analyzes the input and generates multiple ad concepts, targeting different audience segments and value propositions.

2. Multi-AI Visual Creation

Three different AI image engines work in parallel: Seedream 4.0 creates realistic product visuals, NanoBanana enhances and edits images, and ChatGPT Image generates creative variations. This ensures diverse, high-quality visual assets.

3. Video Production with Veo 3

AI-generated scripts are fed into Veo 3 to produce cinematic video clips. The workflow can create multiple video variations with different tones, lengths, and styles for A/B testing.

4. Clip Merging & Finalization

Multiple short clips are merged into polished final ads, with transitions, background music, and branding elements added automatically.

5. Multi-Platform Publishing

Using Blotato integration, the workflow publishes finished videos to TikTok, Instagram, LinkedIn, X, Threads, Facebook, Pinterest, Bluesky, and YouTube simultaneously.

6. Tracking & Notification

All metadata, performance data, and generated assets are stored in Google Sheets and Google Drive. You receive Telegram notifications with video links and performance metrics.

What You'll Need

  1. n8n instance (cloud or self-hosted)
  2. Telegram Bot API credentials for input and notifications
  3. Google Cloud account with Sheets and Drive API enabled
  4. Kie AI API access for Seedream and Veo 3
  5. Fal.ai API key for NanoBanana and video merging
  6. OpenAI API key for GPT-4o script generation
  7. Blotato API credentials for social media publishing
  8. Google Sheet prepared with brand information and settings

Pro tip: Start with a single social platform when testing. Once the workflow runs smoothly, enable additional platforms gradually to manage complexity and track performance differences.

Quick Setup Guide

  1. Download the template JSON file and import it into your n8n instance
  2. Create a new Google Sheet with columns for product name, category, features, offer, and website URL
  3. Set up API credentials for all required services in n8n's credentials management
  4. Configure the Telegram bot node with your bot token and chat ID
  5. Update Google Drive folder IDs in the storage nodes to match your preferred location
  6. Test the workflow with a single product image and caption
  7. Once verified, activate the workflow and connect your social media accounts via Blotato

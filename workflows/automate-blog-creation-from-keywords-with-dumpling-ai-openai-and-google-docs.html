Automate Blog Creation from Keywords with Dumpling AI & OpenAI

Transform a single keyword into a complete, polished blog draft in Google Docs—automatically. Save hours on research and writing.

What This Workflow Does

This automation solves a major pain point for content creators: turning keyword ideas into complete, publish-ready blog posts. Manually researching, summarizing articles, and drafting content can take 6-8 hours per post. This workflow compresses that process into minutes.

When you submit a keyword, the system automatically expands it into trending search suggestions using Dumpling AI, fetches the latest relevant news articles, filters for fresh content (1-2 days old), cleans and aggregates the information, then uses OpenAI to generate a professional blog draft with a compelling title. The final draft is saved directly into Google Docs, formatted and ready for your team to review and publish.

The result is a consistent, scalable content production system that maintains quality while eliminating the repetitive manual work. It's like having a research assistant and junior writer working 24/7.

How It Works

1. Keyword Input & Expansion

The workflow starts when you submit a target keyword through a form trigger. Dumpling AI's Autocomplete API expands this into multiple trending search suggestions, giving the system a broader content foundation to work from.

2. News Gathering & Filtering

Each suggestion queries Dumpling AI's Google News endpoint for recent articles. A code node filters results to only include content published within the last 1-2 days, ensuring your blog stays current and relevant.

3. Content Cleaning & Aggregation

The workflow scrapes article content, removes clutter (ads, navigation, unrelated sections), and combines the cleaned text from multiple sources. This creates a rich information base for the AI to work from.

4. AI Blog Generation

OpenAI receives the aggregated content and generates a complete blog draft in markdown format, including a compelling title, introduction, body sections, and conclusion—all structured for readability and SEO.

5. Google Docs Creation

The final step creates a new Google Doc with the generated title and inserts the full blog content. The document is automatically shared with your team for review and editing.

Quick Setup Guide

Follow these steps to implement this automation in your n8n environment:

  1. Download and import the JSON template using n8n's import feature.
  2. Configure credentials for Dumpling AI, OpenAI, and Google Docs in n8n's credentials management.
  3. Test the form trigger by submitting a sample keyword to ensure the workflow initiates correctly.
  4. Adjust filtering parameters in the code node if you prefer content older than 1-2 days.
  5. Set up a schedule (optional) using n8n's scheduler trigger to run this workflow automatically for your content calendar.

Pro tip: For best results, create a list of target keywords in a Google Sheet and use n8n's Google Sheets node to feed them into this workflow on a schedule. This creates a fully automated content pipeline.

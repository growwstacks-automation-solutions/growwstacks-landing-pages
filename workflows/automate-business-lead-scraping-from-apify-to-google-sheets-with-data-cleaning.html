What This Workflow Does

Manual lead generation is time-consuming and error-prone. Sales teams spend hours searching directories, copying contact details, and cleaning spreadsheets—only to end up with inconsistent data that slows down outreach.

This n8n workflow automates the entire process: it runs Apify scrapers to extract fresh business leads, automatically cleans and formats the data, and exports everything to Google Sheets. What used to take hours now happens automatically, giving your team a constantly updated pipeline of qualified prospects.

The system handles data normalization—stripping unwanted characters from phone numbers, standardizing email formats, and structuring company information—so every lead arrives in your spreadsheet ready for immediate outreach.

How It Works

1. Trigger & Configuration

The workflow starts manually or can be scheduled to run automatically. It first loads your Apify credentials and target scraper configuration, ensuring secure access to the scraping platform.

2. Apify Scraper Execution

n8n triggers your specified Apify task, which crawls target websites or directories to extract lead information. The scraper runs in the cloud and returns structured data including company names, contact details, and other relevant information.

3. Data Cleaning & Transformation

Raw scraped data often contains inconsistencies. This workflow automatically processes each lead: phone numbers are stripped to digits only, emails are converted to lowercase and trimmed, and company names are properly formatted.

4. Google Sheets Export

Cleaned leads are appended to your designated Google Sheet with proper column headers. The workflow maps scraped fields to your spreadsheet structure, maintaining data integrity and making the information immediately usable by your sales team.

What You'll Need

  1. Apify account with an active API token and at least one configured scraper task
  2. Google Sheets access with OAuth2 credentials configured in n8n
  3. n8n instance (cloud or self-hosted) with HTTP Request, Set, and Google Sheets nodes available
  4. Target spreadsheet prepared with appropriate column headers for your lead data
  5. Clear scraping targets - knowing which websites or directories contain your ideal prospects

Pro tip: Start with a small test run before scaling up. Configure your Apify scraper to return 10-20 leads initially to verify data quality and formatting before processing larger volumes.

Quick Setup Guide

  1. Download and import the template JSON file into your n8n instance
  2. Configure Apify credentials in the Variables node with your API token and task ID
  3. Set up Google Sheets connection in n8n if not already configured
  4. Update the Google Sheets node to point to your target spreadsheet and worksheet
  5. Test the workflow manually to verify data flows correctly from Apify to Sheets
  6. Schedule automation (optional) to run daily or weekly for fresh leads

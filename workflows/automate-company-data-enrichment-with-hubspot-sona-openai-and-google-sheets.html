What This Workflow Does

Sales and marketing teams spend hours researching companies to find ideal customers. This workflow automates that entire process. It takes a list of company domains from a Google Sheet, scrapes their websites, analyzes the content with AI to extract strategic insights, enriches the data with firmographic details via Sona, and creates complete company profiles in HubSpot.

Instead of manual research, you get automated intelligence gathering. The system combines three powerful data sources: web content (what the company says about itself), AI interpretation (what it means), and verified enrichment data (facts like employee count, revenue, tech stack). This creates a 360-degree view of each prospect, enabling hyper-targeted outreach and accurate lead scoring.

How It Works

Step 1: Data Input & Web Scraping

The workflow reads company domains from your Google Sheet. For each domain, it sends an HTTP request to scrape the website's main content. It cleans the HTML, removing navigation, footers, and noise, leaving only the core business content for analysis.

Step 2: AI Analysis & Insight Extraction

The cleaned content is sent to OpenAI's chat model. The AI extracts structured intelligence: industry, positioning, key features, target personas, pricing models, value propositions, and competitive advantages. All results are aggregated into a standardized format ready for CRM import.

Step 3: HubSpot Field Preparation

The workflow creates custom fields in your HubSpot CRM to hold the enriched data. It splits the aggregated AI insights into individual company records and prepares them for batch processing.

Step 4: Enrichment & Final Sync

Each company is processed individually. The Sona Enrich API adds firmographic data: company size, employee count, revenue estimates, headquarters location, funding status, and technographic stack. A company record is created in HubSpot, and all custom fields are populated with the combined AI + Sona data.

Pro tip: Modify the OpenAI prompt to extract specific fields relevant to your ICP. Add custom scoring logic based on enrichment results to automatically tag high-priority accounts.

What You'll Need

  1. A Google Sheet with a "Website Domain" column containing company URLs.
  2. An OpenAI API key for AI analysis (sign up at platform.openai.com).
  3. Sona API credentials for firmographic enrichment (get access at platform.sonalabs.com).
  4. A HubSpot account with API access (create a legacy app for token generation).
  5. An n8n instance (cloud or self-hosted) to run the workflow.

Quick Setup Guide

  1. Download the template and import it into your n8n workspace.
  2. Connect the Google Sheets node to your spreadsheet containing company domains.
  3. Add your OpenAI API key in the AI Agent node and customize the extraction prompt if needed.
  4. Authenticate the Sona Enrich node with your API credentials.
  5. Connect both HubSpot nodes using your HubSpot API token (from a legacy app).
  6. Run the workflow with 2–3 test companies to verify records appear correctly in HubSpot.
  7. Scale up by adding your full company list and optionally adding a Schedule Trigger for weekly updates.

What This Workflow Does

This automation solves a common problem in modern workplaces: keeping teams informed about daily schedules without manual effort. Instead of someone spending 15-30 minutes each morning compiling and sharing meeting lists, this workflow automatically fetches all Outlook calendar events for the current day, formats them into a clear summary, and posts it directly to a designated Microsoft Teams channel.

The result is consistent, error-free communication that ensures everyone starts their day with the same information. Remote teams stay aligned, project managers have instant visibility into team availability, and no important meetings get missed because someone forgot to check their calendar. It transforms calendar data from a personal tool into a team resource.

How It Works

1. Scheduled Trigger

The workflow activates automatically every morning (configurable to your timezone). It runs without manual intervention, ensuring the daily digest is always delivered on time.

2. Fetch Today's Outlook Events

Using the Microsoft Graph API, the workflow queries your Outlook calendar for all events scheduled for the current day. It can filter by specific calendars, exclude personal appointments, and handle recurring meetings intelligently.

3. Format the Information

Raw calendar data is transformed into a readable format. Meeting times are converted to local time zones, subjects are highlighted, and descriptions are included. The formatting creates a clean, professional message ready for team consumption.

4. Post to Microsoft Teams

The formatted summary is posted as a rich message to your chosen Teams channel. You can configure which team and channel receives the message, and even tag specific people or groups if needed.

Pro tip: Set this to run 30 minutes before your team's typical start time. This gives everyone time to review the day's schedule before their first meeting or work block begins.

Quick Setup Guide

Follow these steps to get your automated calendar digest running:

  1. Download the template using the button above and save the JSON file to your computer.
  2. Import into n8n by going to your n8n dashboard, clicking "Workflows," then "Import from file," and selecting the downloaded JSON.
  3. Configure credentials for both Microsoft Outlook and Microsoft Teams in n8n's credential manager. You'll need to authenticate with your Microsoft 365 account.
  4. Set the schedule by editing the Schedule Trigger node to match your preferred delivery time (e.g., 8:00 AM local time).
  5. Select your Teams channel in the Microsoft Teams node configuration.
  6. Test the workflow by executing it once manually to verify events are fetched and the message posts correctly.
  7. Activate the workflow and let it run automatically every day.

Pro tip: Start with a test Teams channel or a small pilot group before rolling out to your entire organization. This lets you refine the format and content based on real feedback.

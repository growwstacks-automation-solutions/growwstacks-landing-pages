What This Workflow Does

If you're drowning in emails, struggling to separate signal from noise, this automation is your lifeline. It tackles the manual, time-consuming task of email triage by using AI to instantly label incoming Gmail messages based on their content and intent.

But it goes beyond simple filtering. The workflow saves emails tagged with specific labels (like "Business" or "Research") into an InfraNodus knowledge graph. This graph visually maps the connections between topics within your emails, transforming a cluttered inbox into a structured knowledge base. The system then analyzes this graph to identify "structural gaps"—missing connections between concepts—and uses AI to generate novel questions or ideas that bridge these gaps, turning your email stream into a source of strategic insight.

Finally, you receive instant Telegram notifications for emails with critical labels and for the new ideas generated, ensuring you never miss what matters and can act on insights immediately.

How It Works

The workflow is triggered automatically for each new email, creating a hands-off, continuous intelligence system.

Step 1: Trigger on New Email

The n8n workflow is activated whenever a new message arrives in your connected Gmail account. This sets the entire process in motion without any manual intervention.

Step 2: AI-Powered Classification

The email content is sent to Google's Gemini AI (via the n8n AI Classifier node). You define categories (e.g., "Urgent", "Newsletter", "Sales Inquiry", "Networking") and the AI intelligently assigns the most relevant label based on the email's context, not just keywords.

Step 3: Apply Gmail Label & Store

The workflow applies the AI-assigned label directly within Gmail, organizing your inbox. Simultaneously, emails of a chosen category (e.g., "Business") are formatted and sent to InfraNodus via its API.

Step 4: Build & Analyze the Knowledge Graph

InfraNodus constructs a visual knowledge graph from the email content. It identifies main topics and how they relate. The system's unique algorithm then finds the most significant "blind spots"—concepts that are weakly connected or missing—within this network.

Step 5: Generate Insightful Questions

Leveraging the identified structural gap, the workflow prompts the AI to generate a pertinent, insightful question or idea. This turns your passive email consumption into an active ideation engine.

Step 6: Deliver Notifications via Telegram

A Telegram bot sends you a concise message containing the newly generated insight and a direct link to the updated knowledge graph. You can also configure alerts for emails tagged with high-priority labels.

Pro tip: Use the "contextSettings" parameter in the InfraNodus node to switch between a text-based graph (topics only) and a social graph (which includes sender/recipient relationships) for different analysis layers.

What You'll Need

  1. A Gmail account and a Google Cloud Project with the Gmail API enabled to generate OAuth credentials.
  2. A Google Gemini AI API key (available via Google AI Studio).
  3. An InfraNodus account and API key from their website.
  4. A Telegram Bot token (created via @BotFather in seconds) and your Chat ID.
  5. An n8n instance (version 1.94+ for the AI Classifier node), either self-hosted or cloud.

Quick Setup Guide

Follow these steps to deploy this intelligence system in under 30 minutes.

  1. Download & Import: Click the "Download Template" button and import the JSON file into your n8n instance.
  2. Configure Credentials: In the workflow, set up the credentials for Gmail OAuth, Gemini AI, InfraNodus (Bearer token), and your Telegram Bot.
  3. Define Your Categories: In the AI Classifier node, specify the labels you want (e.g., "Urgent", "Lead", "News", "Networking").
  4. Name Your Graph: Update the "graphName" field in the InfraNodus HTTP Request nodes to your desired project name.
  5. Set Telegram Chat ID: Enter your personal or group Chat ID in the Telegram node to receive notifications.
  6. Test & Activate: Run the workflow once manually to test all connections, then activate the Gmail trigger to run automatically.

Pro tip: Start with a single, broad category like "Business" to populate your knowledge graph quickly. You can always add more specific labels later as you refine the system.

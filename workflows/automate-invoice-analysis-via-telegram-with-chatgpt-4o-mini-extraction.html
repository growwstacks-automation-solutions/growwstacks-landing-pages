What This Workflow Does

Manual invoice processing is a time-consuming, error-prone task for finance teams. Employees receive invoices via email, chat, or uploads, then manually extract vendor details, amounts, dates, and tax information into accounting systems. This process eats up hours each week and introduces data entry mistakes.

This automation solves that by letting users upload invoice files directly to a Telegram bot. The workflow instantly downloads the file, uses ChatGPT-4o-mini AI to extract structured data, and returns a clean, formatted analysis back to the user. It turns a multi-step manual task into a single upload action.

The result is faster payment cycles, reduced clerical workload, and higher data accuracy. Finance teams can focus on analysis rather than data entry.

How It Works

Step 1: Telegram Trigger

A Telegram bot listens for incoming messages containing document attachments. When a user sends an invoice file (PDF, image, etc.), the workflow activates.

Step 2: Document Validation

The workflow checks if the uploaded file is a valid document type and routes errors if the file is unsupported or corrupted.

Step 3: File Retrieval

The system retrieves the file metadata from Telegram and prepares it for download.

Step 4: HTTP Download

The invoice file is downloaded from Telegram's servers to a temporary location for processing.

Step 5: AI Extraction

ChatGPT-4o-mini reads the invoice content—whether it's a scanned image or PDF—and extracts key fields: vendor name, invoice number, date, total amount, tax details, and line items.

Step 6: Data Formatting

The raw extracted data is structured into a consistent JSON or spreadsheet format, ready for integration with accounting software.

Step 7: Response Delivery

The structured analysis is sent back to the user via Telegram, providing instant feedback and enabling quick verification.

What You'll Need

  1. A running n8n instance (cloud or self-hosted).
  2. A Telegram Bot Token created via BotFather.
  3. An OpenAI API key with access to ChatGPT-4o-mini.
  4. Basic understanding of n8n node configuration.
  5. Optional: Integration credentials for your accounting software (QuickBooks, Xero, etc.) if you want to push data directly.

Quick Setup Guide

Follow these steps to import and configure the workflow in your n8n environment.

  1. Download the template JSON file using the button above.
  2. In n8n, go to Workflows → Import and upload the JSON file.
  3. Open the Telegram Trigger node and add your Bot Token from BotFather.
  4. Configure the OpenAI Agent node with your API key and adjust the extraction prompt if needed.
  5. Test by sending an invoice file to your Telegram bot—the workflow should respond with extracted data.
  6. Optionally, add nodes to send the structured data to your accounting software or database.

Pro tip: Start with a few test invoices to verify extraction accuracy. Adjust the AI prompt to capture specific fields your business needs, like project codes or department tags.

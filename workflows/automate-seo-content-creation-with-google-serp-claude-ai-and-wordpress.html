What This Workflow Does

Creating SEO-optimized content is time-consuming. Between keyword research, analyzing competitors, drafting, and publishing, a single article can take 4–8 hours. This automation streamlines the entire process into a one-click pipeline.

The workflow starts with a prioritized keyword in Google Sheets. It fetches the top 10 Google search results, uses AI to analyze competitors' tone and content gaps, generates a detailed content brief, creates a full blog draft with Claude AI, and automatically publishes it to WordPress as a draft. What used to take hours now happens in minutes, with consistent quality and SEO structure.

This isn't just AI content generation—it's intelligent, research-backed content creation that mimics the best practices of top-ranking pages, giving your content a competitive edge from the start.

How It Works

The automation follows a logical, multi-step process that ensures quality and relevance at each stage.

Step 1: Keyword Trigger

You maintain a Google Sheet with your target keywords. When you mark a keyword as "prioritized" and click "Prepare Content," the sheet sends the keyword via webhook to n8n, triggering the workflow.

Step 2: SERP Analysis

n8n uses the Google Custom Search API to fetch the top 10 search results for your keyword. It extracts titles, meta descriptions, URLs, and snippets—everything a human researcher would manually collect.

Step 3: AI Competitor Analysis

Claude AI reviews the SERP data, identifying common themes, content structure, tone, and—critically—gaps where your content can provide unique value. It produces a structured content brief with target word count, headings, and key points to cover.

Step 4: AI Content Generation

Using the brief, another AI call generates a complete blog draft with proper HTML formatting, SEO meta tags, and internal linking suggestions. The draft is written to match the analyzed tone while filling identified gaps.

Step 5: WordPress Publishing

The draft is automatically uploaded to your WordPress site as a "draft" post, with categories, tags, and featured image placeholders set. Your Google Sheet is updated with the status and link, closing the loop.

What You'll Need

  1. n8n instance (cloud or self-hosted) with workflow import capabilities.
  2. Google Cloud account with Custom Search JSON API enabled and an API key.
  3. Anthropic API key for Claude AI access (or OpenAI key with minor adjustments).
  4. WordPress site with Application Passwords enabled for API access.
  5. Google Sheets with edit permissions (the template will be provided).
  6. Basic understanding of webhook URLs and API key management.

Pro tip: Start with a test WordPress site and a non-competitive keyword to see the full workflow in action before using it for production content. This helps you understand the output quality and adjust AI prompts if needed.

Quick Setup Guide

  1. Download the template using the button above and import it into your n8n instance.
  2. Duplicate the Google Sheets template (link in workflow notes) to your Drive.
  3. Configure credentials in n8n: Google Custom Search API key, Anthropic API key, and WordPress Application Password.
  4. Update webhook URLs in the Google Sheets script to point to your n8n webhook node.
  5. Test with one keyword marked as "prioritized" and click "Prepare Content" in your sheet.
  6. Review the WordPress draft, make any edits for brand voice, and publish manually the first few times.
  7. Scale up by adding multiple keywords and training your team on the editing process.

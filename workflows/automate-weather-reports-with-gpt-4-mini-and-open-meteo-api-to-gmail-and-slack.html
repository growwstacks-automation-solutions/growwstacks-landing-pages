Automate Weather Reports with GPT-4 Mini & Open-Meteo API

What This Workflow Does

This automation solves a common business problem: keeping teams informed about weather conditions without manual effort. Instead of someone checking forecasts daily and sending emails, this workflow automatically fetches weather data, generates intelligent summaries, and distributes them via email and team chat.

The system operates in two powerful modes. First, it provides scheduled daily briefings with maximum and minimum temperatures for your specified location. Second, it includes an interactive AI agent that can answer natural language questions about current or future weather conditions. This dual approach ensures both routine updates and on-demand intelligence.

For businesses with field operations, remote teams, or outdoor events, this automation reduces weather-related disruptions by providing timely information. It transforms raw meteorological data into actionable business intelligence that helps with planning, safety, and operational efficiency.

How It Works

1. Scheduled Weather Summaries

A schedule trigger runs daily at 9:00 AM (configurable) to fetch weather data from the Open-Meteo API. The workflow extracts key metrics like temperature highs and lows, formats them into a readable summary, and sends it to both Gmail and designated Slack channels.

2. Interactive AI Weather Agent

When someone sends a message to the chat trigger webhook, the GPT-4 Mini AI agent interprets the question, fetches relevant weather data via HTTP request, and formulates a natural language response. This allows team members to ask questions like "What's the humidity?" or "Will it rain tomorrow?" and get instant answers.

3. Multi-Channel Delivery

Both scheduled summaries and AI responses are delivered through Gmail for formal records and Slack for immediate team visibility. This ensures information reaches people where they already work, increasing adoption and utility.

What You'll Need

  1. n8n instance (cloud or self-hosted)
  2. OpenAI API key for GPT-4 Mini access
  3. Gmail account with app password or OAuth setup
  4. Slack workspace with webhook URL for your channel
  5. Location coordinates (latitude/longitude) for your area of interest

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance.
  2. Configure credentials for OpenAI, Gmail, and Slack in n8n's credential management.
  3. Update location parameters in both the "Fetch Weather Data" and "HTTP Request Tool for AI" nodes with your desired latitude and longitude.
  4. Set recipient details in the email nodes to your team's address and select appropriate Slack channels.
  5. Adjust schedule timing in the Schedule Trigger node if you prefer different delivery times.
  6. Activate the workflow and test both the daily summary and chat query functionality.

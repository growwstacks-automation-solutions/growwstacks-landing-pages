Find qualified businesses, extract contact details, and build targeted prospect lists automatically

What This Workflow Does

This automated system solves the time-consuming challenge of B2B lead generation by combining three powerful technologies:

First, it taps into the Google Places API to discover businesses matching your ideal customer profile based on category and location. Each lead receives a quality score based on ratings, website presence, and operational status. The workflow then uses Scrape.do to reliably access each business website (avoiding IP blocks) and finally employs AI to extract valuable contact details like emails and social media profiles that manual searches often miss.

How It Works

1. Business Discovery

The workflow starts by querying Google Places API with your defined search parameters (business category and location). It retrieves verified listings with key data points like addresses, phone numbers, and customer ratings.

2. Lead Scoring

A custom scoring algorithm evaluates each business based on multiple factors: Google rating (higher = better), having a website (website = higher score), and being marked as operational (open businesses score higher).

3. Website Scraping

For high-scoring leads, the workflow uses Scrape.do's professional scraping service to reliably fetch the business website HTML while avoiding CAPTCHAs and IP blocks that would occur with direct requests.

4. AI-Powered Contact Extraction

An AI agent analyzes the scraped website content to intelligently identify and extract contact details from page footers, about pages, and contact forms - finding information that simple pattern matching would miss.

What You'll Need

  1. Google Cloud account with Places API enabled
  2. Scrape.do account for reliable website access
  3. OpenAI API key for contact extraction
  4. Google Sheet to store collected leads
  5. n8n instance (cloud or self-hosted)

Pro tip: Start with a small geographic area to test your search parameters before scaling up to larger regions.

Quick Setup Guide

  1. Import the workflow JSON into your n8n instance
  2. Configure API credentials for Google Places, Scrape.do, and OpenAI
  3. Set your target business category and location parameters
  4. Connect your Google Sheet and map the data fields
  5. Schedule the workflow to run daily or weekly

What This Workflow Does

Every day, businesses lose potential customers and waste valuable employee time on manual email triage. This automated workflow solves that by turning your Gmail inbox into an intelligent, self-service response system.

When a new inquiry email arrives, the workflow uses Google's Gemini AI to instantly analyze the content. It determines whether the customer is asking about availability, requesting a booking, seeking pricing, or needs general support. Based on this intent, it automatically routes the email, checks external systems for availability, and sends a personalized, context-aware reply—all within seconds.

This isn't just a simple auto-responder. It's a smart assistant that understands nuance, extracts key details like dates and service types, and ensures customers get accurate, helpful information immediately, 24/7.

How It Works

The automation follows a logical sequence to process each email intelligently.

1. Trigger on New Email

The workflow is activated whenever a new email arrives in your specified Gmail inbox or label. It captures the sender, subject, and full body of the message for processing.

2. AI-Powered Intent Analysis

The email content is sent to the Google Gemini Chat Model. A custom prompt instructs the AI to classify the inquiry type (e.g., "booking," "availability check," "general question") and extract relevant parameters like requested dates, services, or contact information.

3. Data Parsing & Logic Routing

A Code node parses the structured JSON output from the AI. A "Wait for Data" node ensures synchronization before the workflow uses an "If" node to route the email down the correct path based on the detected intent.

4. Automated Response & Action

Depending on the route, the workflow can: send an immediate acknowledgment, query an external booking API for availability, forward details to a team member, or send a tailored response with next steps—all through connected Gmail nodes.

What You'll Need

  1. A Gmail account (or Google Workspace) with API access enabled.
  2. Access to the Google Gemini API (via Google AI Studio or Vertex AI).
  3. An n8n instance (cloud or self-hosted) to run the workflow.
  4. (Optional) Access to an external booking or calendar API if you want to check real-time availability.

Pro tip: Before going live, test the AI's intent classification with a batch of past customer emails. Tweak the prompt to better recognize your specific industry terminology and common request formats.

Quick Setup Guide

Get this intelligent email processor running in your n8n environment in under 30 minutes.

  1. Download & Import: Click the "Download Template" button above and import the JSON file into your n8n instance.
  2. Connect Gmail: Create a new Gmail node credential in n8n with the necessary "read" and "send" permissions.
  3. Configure Gemini AI: Add your Google Gemini API key to the "Google Gemini Chat Model" node and review the default prompt. Customize it for your business if needed.
  4. Set Up Triggers & Responses: Point the Gmail trigger to the correct inbox/label. Edit the response templates in the Gmail reply nodes to match your brand voice.
  5. Test Thoroughly: Activate the workflow and send test emails. Monitor the AI's classification accuracy and adjust the prompt or logic nodes as necessary.
  6. Go Live: Once testing is successful, let the automation handle incoming emails. Plan to review its decisions periodically for the first week to ensure quality.

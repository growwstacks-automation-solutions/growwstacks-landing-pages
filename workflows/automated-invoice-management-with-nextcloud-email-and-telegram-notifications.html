What This Workflow Does

Manual invoice processing is a time-consuming bottleneck for freelancers and small businesses. You must regularly check folders for new invoices, remember to email them to clients, and keep track of what's been sent. This workflow eliminates that entire process.

The automation monitors your Nextcloud folder for new PDF invoices, automatically sends them to designated recipients via email, notifies you instantly via Telegram, and archives processed files—all without any manual intervention. It transforms a 15-minute daily task into a fully automated system that runs reliably in the background.

How It Works

The workflow follows a logical sequence to ensure every invoice is processed correctly and notifications are delivered promptly.

1. Scheduled Trigger & File Detection

The workflow activates daily at 8 AM (configurable) and scans your specified Nextcloud folder (/Invoice/Incoming). It identifies new PDF files that haven't been processed previously, filtering out duplicates to prevent resending.

2. File Download & Preparation

Each new invoice file is downloaded from Nextcloud. The system extracts metadata like filename, size, and modification date, preparing it for email attachment and notification messages.

3. Email Delivery

The PDF invoice is attached to a professionally formatted email and sent to your predefined recipient (e.g., accounting@client.com). The email includes the invoice filename in the subject line for easy reference.

4. Instant Notification

Simultaneously, a Telegram message is sent to your designated chat or channel, confirming the invoice has been processed. The notification includes the filename and timestamp, providing immediate visibility without checking email.

5. Archiving & Cleanup

Finally, the processed invoice is moved from the incoming folder to an organized archive directory (/Invoice/2025/archive), maintaining a clean workspace and creating an audit trail of all processed documents.

Pro tip: Configure the schedule to run during off-peak hours (like early morning) to ensure system resources are available and invoices are processed before the business day begins.

What You'll Need

  1. n8n instance – Self-hosted or n8n.cloud account
  2. Nextcloud access – With API credentials for the folder containing invoices
  3. SMTP email service – Gmail, Outlook, or your business email server credentials
  4. Telegram Bot Token – Created via @BotFather with permission to send messages
  5. Folder structure – Organized Nextcloud directories for incoming and archived invoices

Quick Setup Guide

Follow these steps to implement this automation in your environment:

  1. Download the template using the button above and import it into your n8n instance
  2. Configure Nextcloud node with your server URL, username, and app password
  3. Set up Email node with your SMTP server details and recipient email address
  4. Add Telegram credentials by creating a bot via @BotFather and entering the token
  5. Adjust folder paths to match your Nextcloud directory structure
  6. Test with a sample invoice to verify email delivery and Telegram notifications
  7. Activate the workflow and let it run on its scheduled trigger

Pro tip: Start with a test folder and dummy invoices before pointing the workflow to your production invoice directory. This prevents accidental emails to real clients during setup.

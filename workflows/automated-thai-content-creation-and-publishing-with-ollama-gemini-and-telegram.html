Automated Thai Content Creation & Publishing

Generate unique Thai articles, create AI images, and publish automatically with Ollama, Gemini, and Telegram integration.

What This Workflow Does

This automation solves the time-intensive challenge of creating and publishing multilingual content for Thai audiences. Manual content creation requires writers, translators, designers, and publishing coordination—costing hours per article and limiting output frequency.

The workflow automatically transforms raw news data from Google Sheets into complete, SEO-optimized Thai articles with AI-generated images. It intelligently selects image styles, adapts writing formats, and publishes directly to your website while sending Telegram notifications. This eliminates manual steps while maintaining cultural relevance and quality.

Businesses using this automation can publish daily Thai content without hiring local teams, improving regional SEO, audience engagement, and brand presence in the Thai market at minimal operational cost.

How It Works

Step 1: Data Trigger & Analysis

The workflow activates when new Thai news data arrives in your Google Sheet. An AI model analyzes the content to determine optimal image style—photorealistic for general news or artistic risoprint for creative topics.

Step 2: Article Generation

Using Ollama AI, the system drafts a long-form Thai article adapting its structure (listicle or analytical) to match the topic's nature. The AI ensures language accuracy, cultural appropriateness, and engaging readability.

Step 3: SEO & Marketing Optimization

A specialized AI generates concise Thai and English titles, engaging excerpts, relevant tags, and precise image prompts. This metadata boosts search visibility and social shareability across platforms.

Step 4: AI Image Creation

Google Gemini produces stunning visuals based on the AI's prompts, creating either photorealistic or artistic risoprint-style images that visually complement the article content.

Step 5: Automated Publishing

The complete article with image and metadata posts automatically to your website via API, including optional fields like background color, affiliate links, and embedded content.

Step 6: Instant Notification

Telegram sends you a summary of the new post, keeping you informed without manual checking.

Pro tip: Configure your Google Sheet with clear Thai news descriptions and categories. This improves AI analysis accuracy and ensures more relevant image style selections.

What You'll Need

  1. Google Sheet with Thai news data and configured sheet name/column
  2. Ollama API credentials for local or remote Ollama instance access
  3. Google Gemini API key for AI image generation capabilities
  4. Website API credentials including posting URL, API key, user_id, and username
  5. Telegram bot token and chatId for notification delivery

Quick Setup Guide

Follow these steps to implement this automation:

  1. Import the template into your n8n instance using the downloaded JSON file
  2. Configure Google Sheets by connecting your account and specifying the data sheet
  3. Set up Ollama by adding API credentials and testing model connectivity
  4. Add Gemini API with your key to enable image generation functionality
  5. Connect your website by entering API details and optional publishing parameters
  6. Setup Telegram with bot token and target chatId for notifications
  7. Test the workflow with sample Thai news data to verify full pipeline operation

Pro tip: Start with a small test dataset before full deployment. Verify image style selections match your content preferences and adjust AI parameters accordingly.

What This Workflow Does

This automation solves a common problem for digital marketers and course creators: manually copying email addresses from your Systeme.io sales funnels to your Beehiiv newsletter list. Every time someone opts into your funnel, you need to ensure they're added to your email marketing system—a repetitive task that wastes valuable time and risks errors.

The workflow automatically creates a subscriber in your specified Beehiiv publication whenever a new opt-in is registered in your Systeme.io sales funnel. It acts as a bridge between your lead capture system and your email marketing platform, ensuring your audience building happens seamlessly in the background.

Beyond simple email transfer, the workflow preserves valuable marketing data like UTM parameters (utm_source, utm_medium, utm_campaign) and can map custom fields from Systeme.io to Beehiiv. This means you maintain complete tracking of where your subscribers come from, enabling better segmentation and campaign analysis.

How It Works

1. Webhook Trigger from Systeme.io

When someone opts into your Systeme.io sales funnel, Systeme.io sends a webhook notification to this n8n workflow. The workflow validates that the request actually comes from Systeme.io by checking IP addresses, ensuring security and preventing unauthorized access.

2. Data Processing and Mapping

The workflow extracts the subscriber's email address, name (if available), and any UTM parameters from the Systeme.io payload. It then formats this data according to Beehiiv's API requirements, preparing it for subscriber creation.

3. Beehiiv Subscriber Creation

Using Beehiiv's API, the workflow creates a new subscriber in your specified publication. The subscriber is added with proper attribution data, allowing you to track exactly which funnel generated this lead for future marketing analysis.

4. Error Handling and Notifications

If the Beehiiv API call fails for any reason (API limits, invalid data, network issues), the workflow sends an email alert to specified addresses. This ensures you're immediately aware of any problems and can manually add the subscriber if needed.

What You'll Need

  1. A Systeme.io account with at least one sales funnel configured to capture opt-ins
  2. A Beehiiv account with an active publication where you want to add subscribers
  3. A Gmail account (or other email service) to receive error notifications
  4. n8n instance (cloud or self-hosted) to run the workflow
  5. API credentials for both Systeme.io (webhook configuration) and Beehiiv

Pro tip: Before automating, test your Systeme.io funnel and Beehiiv integration manually with a few test emails to ensure both systems work as expected. This helps identify any configuration issues before they affect real subscribers.

Quick Setup Guide

Follow these steps to implement this automation in under 30 minutes:

  1. Download and import the template into your n8n instance
  2. Configure the webhook node to get your unique webhook URL
  3. Set up Systeme.io to send webhooks when new opt-ins occur (found in funnel automation settings)
  4. Add your Beehiiv publication ID in the configuration node
  5. Connect your accounts by adding Beehiiv and Gmail credentials to their respective nodes
  6. Test with a real opt-in to verify the complete flow works end-to-end
  7. Activate the workflow and let it run automatically

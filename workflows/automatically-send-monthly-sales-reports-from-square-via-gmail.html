What This Workflow Does

Manually compiling and sending monthly sales reports is a tedious, error-prone task that consumes valuable time. This n8n workflow automates the entire process by connecting directly to your Square account, pulling sales data for the previous month, generating a comprehensive CSV report, and emailing it to your management or finance team automatically.

The template specifically matches the sales summary figures shown in your Square Dashboard, ensuring consistency and accuracy. It runs on the 1st of every month, eliminating the risk of forgetting to send reports and providing stakeholders with reliable, timely data for decision-making.

Beyond just saving time, this automation ensures data integrity by removing manual transcription errors and creates an audit trail of monthly performance that's easily accessible for historical analysis.

How It Works

The workflow follows a logical sequence to transform raw Square data into actionable business intelligence delivered directly to inboxes.

1. Scheduled Trigger

A Schedule Trigger node initiates the workflow on the 1st of each month at 8:00 AM. This timing ensures the previous month's data is complete and ready for processing.

2. Fetch Square Locations

The workflow first retrieves all business locations associated with your Square account using the Square API. This ensures the report covers your entire operation, whether you have one storefront or multiple locations.

3. Retrieve Monthly Order Data

For each active location, the workflow makes API calls to Square to fetch all completed orders from the previous calendar month. It handles pagination and filters for relevant transaction data.

4. Process and Aggregate Sales

A Code node processes the raw order data, calculating total sales, tax collected, net revenue, and other key metrics that match Square's own reporting format. Locations with no sales activity are automatically filtered out.

5. Generate CSV Report

The aggregated data is formatted into a clean CSV file with clear column headers, making it easy to import into Excel, Google Sheets, or accounting software for further analysis.

6. Email Delivery via Gmail

The final node attaches the CSV report to a professional email and sends it to your designated recipients using your Gmail account, with customizable subject lines and message body.

  3. n8n Instance (cloud or self-hosted) with the HTTP Request, Gmail, and Schedule Trigger nodes available.
  4. Recipient Email Addresses for stakeholders who should receive the monthly reports.

Pro tip: Before going live, test the workflow with a small date range to verify the data output matches your Square Dashboard reports. This ensures your automation is accurate from day one.

Quick Setup Guide

Get this automation running in your n8n instance in under 30 minutes with these steps:

  1. Import the template into your n8n workspace using the downloaded JSON file.
  2. Configure Square credentials in n8n as Header Auth with Name "Authorization" and Value "Bearer YOUR_SQUARE_ACCESS_TOKEN".
  3. Set up Gmail credentials using OAuth2 authentication in n8n's credential management.
  4. Update the email node with your recipient addresses and customize the message body as needed.
  5. Test the workflow manually first with a short date range to verify data accuracy.
  6. Activate the workflow and let it run automatically on the 1st of each month.

What This Workflow Does

This workflow solves a common pain point for YouTube creators and businesses: manually updating video descriptions across dozens or hundreds of videos. When you need to add a new affiliate link, update contact information, or insert a promotional section between existing rows, doing it manually takes hours and risks inconsistencies.

The automation connects to your YouTube channel via API, fetches all videos, scans each description for two predefined rows (like "Resources:" and "Affiliate Links:"), and inserts your new content exactly between them. It then updates the description automatically, ensuring every video gets the same update instantly.

This transforms a tedious, error-prone task into a one-click operation that maintains professional consistency across your entire channel.

How It Works

Step 1: Fetch All Videos

The workflow starts by connecting to YouTube's API with your authenticated credentials. It retrieves a list of all videos from your channel, including their IDs, titles, and current descriptions.

Step 2: Process Each Description

For each video, the workflow reads the existing description text and searches for two specific rows you've predefined (rowBefore and rowAfter). These could be section headers like "Sponsored Links" or "Additional Resources."

Step 3: Insert New Content

Using a code node, the workflow inserts your new text (rowToInsert) between the identified rows. It preserves the original formatting and ensures the new content appears in the correct position.

Step 4: Update YouTube Descriptions

The modified description is sent back to YouTube via API, updating the video metadata. The workflow processes videos in batches to handle large libraries efficiently.

What You'll Need

  1. YouTube Channel Access: Owner or editor permissions to modify video descriptions.
  2. Google Cloud Project: With YouTube API enabled and OAuth credentials configured.
  3. n8n or Zapier Account: To run the automation workflow (self-hosted n8n or cloud version).
  4. Predefined Description Sections: Consistent rows in your descriptions that the workflow can target (e.g., "Links:" followed by "Contact:").
  5. New Content to Insert: The text, link, or HTML you want to add between existing rows.

Quick Setup Guide

  1. Download the Template: Click the download button above to get the JSON workflow file.
  2. Import into n8n/Zapier: Import the JSON file into your automation platform.
  3. Configure YouTube Connection: Add your YouTube API credentials in the authentication node.
  4. Set Your Variables: Define rowBefore, rowToInsert, and rowAfter in the "Set String to Insert" node.
  5. Test with One Video: Run the workflow on a single video first to verify the insertion works correctly.
  6. Run Full Automation: Execute the workflow to update all videos in your channel.
  7. Review Results: Check a few updated descriptions to ensure accuracy.

Pro tip: Before running on all videos, filter by publish date to update only recent videos. This reduces risk and lets you verify the logic works perfectly before scaling.

What This Workflow Does

This automation transforms Slack into a powerful business intelligence hub. Instead of manually searching databases, spreadsheets, or external tools, your team can ask natural language questions directly in Slack channels. The assistant interprets queries, fetches relevant data from Explorium's comprehensive business intelligence API, processes it with Claude AI for clarity and context, and returns concise, actionable answers.

Common use cases include finding technology companies in specific locations with certain employee ranges, analyzing competitors' software stacks, discovering executive contacts for targeted outreach, and gathering market insights for strategic planning. This eliminates hours of manual research per week and enables faster, data-driven decisions.

Pro tip: Configure the assistant to respond only in specific Slack channels dedicated to research, sales intelligence, or competitive analysis to maintain focus and avoid distraction in general conversations.

How It Works

Step 1: Slack Trigger & Query Capture

The workflow starts when someone mentions your bot or sends a direct message in Slack. The trigger node captures the full query text, user information, and channel context. This ensures the assistant knows who asked the question and where to respond.

Step 2: Query Analysis with Claude AI

Claude AI analyzes the natural language query to understand intent, extract key parameters (like location, company size, industry), and structure it into a formal API request. This translation layer is crucial—it turns "find tech companies in SF with 50-200 employees" into precise search criteria.

Step 3: Explorium API Data Retrieval

The structured query is sent to Explorium's API via the MCP Client node. Explorium returns comprehensive business data including company profiles, technology installations, executive contacts, financial indicators, and market positioning information.

Step 4: Response Formatting & Delivery

Claude AI again processes the raw data, summarizing key findings, highlighting relevant insights, and formatting the response for clear Slack presentation. The final answer is posted back to the originating channel or thread, completing the intelligence loop.

What You'll Need

  1. Slack workspace with admin access to create apps and configure bot permissions.
  2. Explorium API key for accessing their business intelligence database.
  3. Anthropic API key (or alternative LLM provider) for Claude AI query processing and response formatting.
  4. n8n instance (cloud or self-hosted) to run the automation workflow.
  5. Basic understanding of Slack app configuration for setting up event subscriptions and bot tokens.

Quick Setup Guide

  1. Import the template: Download the JSON file and import it into your n8n instance.
  2. Configure Slack app: Create a new Slack app with bot token scopes (app_mentions:read, channels:history, chat:write, etc.), enable event subscriptions, and install to your workspace.
  3. Connect credentials: Add your Slack bot token, Anthropic API key, and Explorium API key to the respective nodes in n8n.
  4. Test the trigger: Use the webhook URL from the Slack Trigger node as your Slack app's Request URL for event subscriptions.
  5. Activate the workflow: Turn on the workflow in n8n and test with sample queries like "@YourBot find healthcare companies in Boston."

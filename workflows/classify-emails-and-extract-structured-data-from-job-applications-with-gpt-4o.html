What This Workflow Does

This automation solution transforms the chaotic process of handling job applications into a streamlined, AI-powered workflow. Designed for HR teams and recruiters, it automatically:

  • Classifies incoming emails as job applications or other categories
  • Extracts text from email bodies and attachments (PDFs, Word docs)
  • Identifies and structures candidate information using GPT-4o
  • Prepares standardized data for your ATS or recruitment pipeline

By eliminating manual data entry, this workflow reduces processing time by 70% while improving data consistency across all applications.

Email classification node configured to identify job applications

How It Works

1. Email Classification

The workflow monitors your recruitment inbox, using AI to distinguish job applications from other emails. Custom categories can be added for different application types or departments.

2. Content Extraction

For identified applications, the system extracts text from both the email body and any attachments. It handles PDFs, Word documents, and even image-based resumes using OCR.

3. Structured Data Creation

GPT-4o analyzes the extracted text to identify and organize candidate information into predefined fields like name, contact details, education history, and work experience.

Data extraction node mapping resume fields to structured output

4. Output Preparation

The structured data is formatted for your specific ATS or recruitment software, ready for review or automatic import.

What You'll Need

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure your IMAP credentials in the email trigger node
  3. Add your OpenAI API key to the classification and extraction nodes
  4. Map your desired output fields in the data transformation nodes
  5. Connect to your ATS or database using one of our integration templates
  6. Test with sample applications and refine your field mappings

What This Workflow Does

This automation acts as an expert shopping assistant that eliminates the tedious, manual process of product research. For businesses, teams, or individual consumers, comparing products involves hours of reading reviews, checking specifications across different websites, and trying to weigh factors like price, performance, and reliability objectively.

The workflow solves this by turning a simple request in Telegram (e.g., "iPhone 15 vs Samsung S24") into a comprehensive, data-driven analysis. It scrapes real-time data, uses AI to evaluate the information, and generates a professional visual scorecard with a clear verdict, delivered directly back to the chat. This transforms subjective guesswork into an automated, objective decision-support tool.

How It Works

Step 1: Intent Classification via Telegram

The workflow is triggered by a message sent to a Telegram bot. An AI classifier first determines if the user wants a casual chat or a specific product comparison. If a comparison is requested, the system extracts the product names and proceeds.

Step 2: Data Scraping with BrowserAct

BrowserAct executes a background task to scrape current specifications, pricing, and user reviews from relevant sources like Amazon, manufacturer websites, or review aggregators. This gathers raw, up-to-date data for both products.

Step 3: AI Analysis & Evaluation

An AI agent aggregates the scraped data and evaluates it based on defined criteria such as composition (features), performance (benchmarks), and economics (price, value). It performs an objective analysis to determine a winner and key differentiating factors.

Step 4: Visual Scorecard Blueprint Creation

A second AI agent then creates a detailed "blueprint" prompt for a comparison infographic. This blueprint specifies how to visually represent the key differences, scores, and the final verdict in an engaging, easy-to-understand format.

Step 5: Image Generation & Delivery

Google Gemini generates a high-quality 4K image based on the blueprint. The final visual scorecard, along with a formatted summary caption, is sent back to the user's Telegram chat, completing the automated research loop.

What You'll Need

  1. A Telegram account and a Bot Token (created via BotFather).
  2. A BrowserAct account with the "Product Comparison & Visualize Bot" template saved.
  3. A Google Sheets account to serve as a temporary data store.
  4. A Google Gemini API key for image generation.
  5. An n8n instance (cloud or self-hosted) to run the workflow.

Quick Setup Guide

  1. Download the template using the button above and import it into your n8n workspace.
  2. Configure credentials in n8n for Telegram, Google Sheets, BrowserAct, and Google Gemini.
  3. Prepare BrowserAct: Ensure your account has the required comparison template and note your API Key and Workflow ID.
  4. Create a Google Sheet and paste its ID into the "Get row(s) in sheet" and "Append row in sheet" nodes in the workflow.
  5. Set up your Telegram bot via BotFather, add the API token to n8n, and activate the workflow.
  6. Test the automation by sending a comparison request (e.g., "laptop A vs laptop B") to your bot in Telegram.

Pro tip: Customize the AI analysis prompts within the workflow to prioritize factors most important to your business, such as "total cost of ownership" over "initial price" for B2B procurement.

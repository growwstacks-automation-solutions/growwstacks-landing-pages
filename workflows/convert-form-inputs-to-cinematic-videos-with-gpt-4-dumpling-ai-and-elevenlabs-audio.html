How It Works

The workflow follows a sophisticated pipeline that leverages the strengths of different AI platforms at each stage of video production.

1. Form Submission & Data Preparation

Users submit a form with four country/animal names and a preferred style (cinematic, epic, fantasy, etc.). The workflow formats these inputs into structured data that can be processed by subsequent AI services.

2. GPT-4 Prompt Generation

OpenAI's GPT-4 creates rich, cinematic descriptions for each input. These aren't simple labels but detailed visual narratives that serve as perfect prompts for image generation. The AI considers the requested style and generates descriptions that will translate well to visual media.

3. Dumpling AI Visual Creation

Each GPT-4 description is sent to Dumpling AI, which generates high-quality images matching the cinematic vision. The system handles API communication, error checking, and result processing to ensure consistent visual output.

4. Motion Prompt Conversion & Animation

GPT-4 rewrites the image prompts into motion-based video prompts, which are then sent to Replicate.com along with the generated images. Replicate's animation models add movement and cinematic effects to transform static images into dynamic video clips.

5. ElevenLabs Audio Generation

Based on the overall style, GPT-4 creates an ambient sound concept that ElevenLabs transforms into custom audio. The generated audio is uploaded to Google Drive for integration into the final video.

6. Creatomate Video Assembly

All four animated clips and the ambient audio track are sent to Creatomate, which stitches them together into a cohesive cinematic video with proper timing, transitions, and audio synchronization.

7. Final Delivery & Logging

The completed video is uploaded to Google Drive, and all details are logged in Google Sheets for tracking and analysis. The entire process happens automatically once the initial form is submitted.

Quick Setup Guide

Follow these steps to implement this cinematic video automation in your n8n environment.

  1. Import the Template: Download the JSON file and import it into your n8n instance using the "Import from File" function in the workflow editor.
  2. Configure Credentials: Set up credentials in n8n for OpenAI, Dumpling AI, Replicate, ElevenLabs, Google Drive, Google Sheets, and Creatomate. Test each connection individually before proceeding.
  3. Prepare Google Resources: Create a Google Sheet with "Title" and "Generated videos" columns. Create two Google Drive folders—one for ambient audio and one for final videos—and note their folder IDs.
  4. Update Resource References: In the workflow, update the Google Sheet ID, tab name, and Google Drive folder IDs in the appropriate nodes.
  5. Test with Sample Data: Run the workflow manually with test data (four countries and a style) to verify each step executes correctly and troubleshoot any API limits or connection issues.
  6. Connect Your Form Trigger: Replace the sample trigger with your actual form integration, ensuring field names match what the workflow expects.
  7. Implement Quality Checks: Add validation nodes if needed to check video length, quality, or content appropriateness before final delivery.

What This Workflow Does

This n8n workflow solves a common frustration for fitness enthusiasts and personal trainers: the tedious manual entry of workout plans into tracking apps like Hevy. Whether you receive programs as PDFs from a coach, screenshots from social media, or text documents, this automation extracts the exercise details using AI and creates a perfectly formatted routine in your Hevy account automatically.

Instead of spending 15-30 minutes typing exercises, sets, reps, and rest periods, this workflow completes the entire process in seconds. It handles the entire pipeline from file upload to structured data extraction to API-based routine creation, eliminating human error and saving valuable time that's better spent actually training.

The system is particularly valuable for coaches managing multiple clients, athletes following periodized programs, or anyone who frequently switches between different workout templates. It turns what was once a administrative chore into a seamless, automated process.

How It Works

The automation follows a intelligent five-step process that mimics how a human would interpret and transfer workout data, but with AI precision and API speed.

1. File Submission & Processing

The workflow starts when a PDF or image file is submitted through an n8n form trigger. The system converts the file to base64 format and prepares it for AI analysis. This could be triggered manually, via email attachment, or through a shared upload link for clients.

2. AI-Powered Text Extraction

Using Google Gemini via OpenRouter, the workflow extracts all textual content from the workout document. The AI is specifically prompted to identify exercise names, set counts, repetition ranges, weight information, rest intervals, and any special notes or superset indicators.

3. Hevy Exercise Database Matching

Before creating the routine, the workflow fetches Hevy's complete exercise database via their API. This ensures that extracted exercise names are matched against official Hevy exercises, handling variations in naming conventions (e.g., "Barbell Bench Press" vs "Bench Press - Barbell").

4. Structured Data Transformation

The raw extracted text is processed through a structured output parser that converts it into clean JSON format matching Hevy's API requirements. This includes organizing exercises into proper workout days, arranging sets in sequence, and formatting all parameters correctly.

5. Routine Creation in Hevy

The final structured data is sent to Hevy's API to create the new workout routine in your account. The workflow handles authentication, error checking, and provides confirmation that the routine has been successfully added to your Hevy library.

Quick Setup Guide

Getting this workflow running takes about 15 minutes if you have your API credentials ready. Follow these steps to implement it in your n8n instance.

  1. Import the template using the JSON file downloaded from this page into your n8n instance.
  2. Configure Hevy API credentials in the HTTP Request nodes. You'll need your Hevy API key from your account settings.
  3. Set up OpenRouter/Gemini API credentials in the AI node. Sign up for OpenRouter if you don't have access to Gemini directly.
  4. Test with a sample workout PDF to ensure the extraction and creation work correctly. Start with a simple plan to verify formatting.
  5. Customize the AI prompt if your workout plans use unusual formatting or terminology to improve extraction accuracy.
  6. Activate the workflow and connect it to your preferred trigger (form, webhook, scheduled, or manual execution).

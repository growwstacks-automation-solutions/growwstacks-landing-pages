How It Works

The workflow is triggered by any message sent to your Telegram bot. It intelligently routes the request based on the command or context.

1. Session Management & Routing

When a user sends a message, the workflow first checks Google Sheets to see if they have an active session. Commands like /new create a fresh session ID, while /resume lets users pick up where they left off. This session state is managed entirely within a Google Sheet, acting as a simple database.

2. Context Retrieval & AI Processing

For ongoing conversations, the workflow fetches the last several messages from that user's session log in Google Sheets. This context is then formatted and sent to the OpenAI Chat Model node (GPT-4o-mini). The AI generates a response that is coherent and relevant to the entire conversation history, not just the latest message.

3. Response Delivery & Logging

The AI's response is sent back to the user via Telegram. Simultaneously, both the user's message and the AI's response are appended to the Google Sheet. This creates a complete, auditable log of every interaction, which can be used for training, quality assurance, or analytics.

4. Specialized Commands

The workflow handles specific commands: /summary instructs the AI to condense the conversation into a brief overview, and /question allows users to ask direct questions about their chat history. The "If" and "Switch" nodes route these commands to the appropriate logic blocks.

Quick Setup Guide

Get your AI chatbot live in under 30 minutes by following these steps:

  1. Import the Template: Download the JSON file above and import it into your n8n instance.
  2. Configure Telegram: In the "Telegram Trigger" node, add the bot token from BotFather. Set the webhook for your n8n endpoint.
  3. Set Up Google Sheets: Duplicate the template sheet. In the "Google Sheets" nodes, authenticate with your Google account and point them to your new sheet's ID and tab names ('Session' and 'Database').
  4. Add Your OpenAI Key: Enter your API key in the "OpenAI Chat Model" nodes. You can adjust parameters like temperature for more creative or deterministic responses.
  5. Test & Activate: Send a /new command to your bot on Telegram. Check the Google Sheet to see the session being created. Activate the workflow in n8n.

Your bot is now live. Users can start conversations, and the AI will respond with context-aware answers, storing everything in your sheet for review.

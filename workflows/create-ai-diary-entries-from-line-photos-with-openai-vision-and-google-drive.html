Create AI Diary Entries from LINE Photos with OpenAI Vision and Google Drive

Automatically turn photos sent to LINE into AI-generated diary entries and save everything neatly in Google Drive with this free n8n workflow template.

Download Template JSON · n8n compatible · Free

What This Workflow Does

This innovative automation solves a common problem for parents, caregivers, and anyone wanting to document daily life without the manual effort. Every time someone sends a photo to a LINE bot, this workflow automatically creates a timestamped photo file and a matching AI-generated diary entry, then saves everything in an organized Google Drive folder structure (KidsDiary/YYYY/MM).

The system transforms casual photo sharing into meaningful documentation by leveraging OpenAI's Vision technology to analyze images and generate thoughtful, diary-style captions in about 50 Japanese characters. What would normally require manual downloading, organizing, and captioning now happens automatically with zero ongoing effort.

How It Works

The workflow follows a logical sequence that ensures reliable processing and organization of every photo sent through LINE.

1. LINE Webhook Trigger

When a user sends an image message to your LINE bot, the LINE Webhook receives the notification and extracts essential metadata including the messageId and replyToken for subsequent API calls.

2. Image Download & Processing

The workflow calls the LINE content API to fetch the actual image as binary data. This ensures the original photo quality is preserved for storage while being prepared for AI analysis.

3. AI Diary Text Generation

OpenAI Vision analyzes the image and generates a one-sentence, diary-style caption. The AI is prompted to create thoughtful, personal observations about what's happening in the photo, making each entry feel authentic and meaningful.

4. Google Drive Organization

A year/month folder structure is created (or reused) in Google Drive under a "KidsDiary" root folder. This automatic organization ensures photos are always findable and grouped logically by time.

5. File Storage & Confirmation

The photo saves as YYYY-MM-DD_HHmmss.jpg and the diary text as YYYY-MM-DD_HHmmss_diary.txt in the same folder. Finally, the LINE bot replies to confirm successful saving, completing the user experience.

Pro tip: Customize the AI prompt to match your preferred tone—more playful for children's photos, more sentimental for family memories, or more factual for documentation purposes.

What You'll Need

  1. LINE Developer Account: Access to LINE Messaging API credentials for webhook configuration and content API access.
  2. OpenAI API Key: Access to GPT-4 Vision or similar vision-capable models for image analysis and text generation.
  3. Google Drive Access: A Google account with Drive API enabled and appropriate folder permissions.
  4. n8n Instance: A running n8n installation (cloud or self-hosted) to deploy the workflow.
  5. Webhook URL: A publicly accessible endpoint for LINE webhook notifications.

Quick Setup Guide

Follow these steps to implement this automation in your environment:

  1. Import the Template: Download the JSON file and import it into your n8n instance using the workflow import function.
  2. Configure LINE Credentials: Update the HTTP Request nodes with your LINE Messaging API channel access token and channel secret.
  3. Set Up OpenAI: Add your OpenAI API key to the OpenAI node and adjust the vision prompt if desired.
  4. Connect Google Drive: Authenticate the Google Drive nodes with your account and specify the root folder for diary storage.
  5. Register Webhook: Copy your n8n webhook URL and register it in the LINE Developers console under webhook settings.
  6. Test & Activate: Send a test photo to your LINE bot, verify the files appear in Google Drive, and activate the workflow.

Customization idea: Add a second branch to post diary entries to Slack for family sharing, send weekly summaries via email, or create a Notion database of all entries for easy searching.

What This Workflow Does

This automation transforms raw video files into polished YouTube uploads ready for publishing. When you drop a video into a designated Google Drive folder, the workflow automatically:

  • Analyzes video content using Gemini AI to understand key themes
  • Generates 3 optimized title options designed for maximum CTR
  • Creates detailed descriptions with timestamps for better SEO
  • Produces 4 thumbnail variations featuring your face (using provided reference images)
  • Publishes to YouTube after human approval of all elements
The automation canvas showing the complete workflow from video upload to YouTube publishing

How It Works

1. Video Upload Trigger

The workflow monitors a specific Google Drive folder for new video uploads. When detected, it downloads the file for processing.

2. AI Content Analysis

Gemini AI analyzes the video to identify key topics, tone, and optimal timestamps for chapter markers in the description.

3. Metadata Generation

The system creates 3 distinct title/description/tag combinations, giving you strategic options to test different angles.

4. Thumbnail Creation

Using your provided facial reference images, fal.ai generates 4 thumbnail variations aligned with each title concept.

5. Human Approval

You receive a simple form to select your preferred metadata package and thumbnail before final publishing.

6. YouTube Publishing

The approved content gets published to your YouTube channel via Upload-Post with all optimized metadata in place.

What You'll Need

  1. Google Drive account (free tier sufficient)
  2. Google Gemini API key (free tier available)
  3. fal.ai account for thumbnail generation
  4. Upload-Post account connected to your YouTube channel
  5. 1-2 high-quality facial reference images for thumbnails

Quick Setup Guide

  1. Download the JSON template file
  2. Import into your n8n instance
  3. Configure all credential nodes (Google Drive, Gemini, fal.ai, Upload-Post)
  4. Set your facial reference image URLs in the thumbnail generation node
  5. Specify your target Google Drive folder ID
  6. Test with a sample video and approve the workflow

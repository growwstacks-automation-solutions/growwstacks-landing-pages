What This Workflow Does

This automation workflow solves a critical bottleneck for content creators, marketers, and educators: the time-consuming process of producing animated short videos. Traditionally, creating a polished animated story requires scripting, illustration, animation, and editing—a multi-day effort involving specialists.

This template connects cutting-edge AI tools into a single pipeline. GPT-4o-mini generates the narrative and scene descriptions, Midjourney creates the visual artwork, Kling animates those images, and Creatomate assembles the final video with subtitles. The result is a fully animated short video produced from a simple prompt, ready for social media, educational content, or advertising.

Businesses can use this to rapidly test creative concepts, produce daily content for platforms like Instagram and TikTok, create educational explainers, or generate product demo videos without hiring a full production team.

How It Works

Step 1: Script Generation with GPT-4o-mini

The workflow starts by sending your creative prompt (e.g., “a gentle girl and a fluffy rabbit explore a sunlit forest”) to GPT-4o-mini via PiAPI. The AI expands this into a detailed scene description, character actions, and narrative flow, providing the foundational script for the video.

Step 2: Image Creation with Midjourney

The generated scene descriptions are passed to Midjourney (also via PiAPI), which produces high-quality, stylized images matching the narrative. Parameters like style, aspect ratio, and artistic direction are configured to ensure visual consistency.

Step 3: Animation with Kling

Kling AI takes the Midjourney-generated images and animates them, adding motion, camera movements, and basic effects to bring the static artwork to life. This creates the core video sequences.

Step 4: Video Assembly with Creatomate

Creatomate API receives the animated sequences and script text, compositing them into a final video. It adds subtitles, transitions, and timing, producing a polished MP4 file ready for distribution.

Step 5: Output & Delivery

The final video is saved to your designated storage (cloud or local) and can be automatically uploaded to social platforms, CMS, or sent via notification.

Quick Setup Guide

1. Import the Template: Download the JSON file above and import it into your n8n workspace.

2. Configure API Credentials: Add your PiAPI X-API-key in the “Basic Params” node and your Creatomate API key in the final node.

3. Set Your Prompts: Define the style, character, and situational keywords in the workflow settings. Example: “a children’s book cover, ages 6-10” with specific Midjourney parameters.

4. Test the Prototype: Use Creatomate’s built-in assets first to verify the template works as expected before connecting the full AI pipeline.

5. Run the Workflow: Click “Test Workflow” and monitor the generation process (typically 10–20 minutes). Review the output video.

6. Expand & Customize: Add music nodes using PiAPI’s audio models or your own music service. Adjust timing, subtitles, and transitions in Creatomate.

Pro tip: For production use, add an error handling node and a quality review step before auto-publishing. This ensures brand-safe outputs.

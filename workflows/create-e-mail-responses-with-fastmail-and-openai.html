What This Workflow Does

This automation solves a common productivity bottleneck: spending hours each week crafting email replies. Whether you're handling customer support, sales inquiries, or general correspondence, manually responding to every email consumes valuable time that could be spent on higher-value work.

The workflow automatically monitors your Fastmail inbox for new messages, extracts their content, sends it to OpenAI's GPT-4 for intelligent analysis, and generates context-appropriate draft replies. These drafts are then placed directly into your Fastmail "Drafts" folder for your review and sending. It's like having a personal assistant who reads your emails and prepares thoughtful responses 24/7.

Beyond simple templating, this system understands email context, tone, and intent. It can differentiate between a customer complaint, a sales inquiry, or a technical support request, adjusting its response style accordingly. The result is professional, personalized drafts that maintain your brand voice while saving 80-90% of the time typically spent on email correspondence.

How It Works

1. Email Monitoring & Trigger

The workflow starts with an IMAP email trigger that continuously watches your specified Fastmail inbox. When a new, unread email arrives, the automation immediately captures it for processing. This trigger ensures no incoming message gets overlooked, providing real-time response capability.

2. Content Extraction & Analysis

Key information is extracted from each email: sender details, subject line, body content, and any attachments or metadata. This data is structured and prepared for AI processing, ensuring the system has complete context about what needs to be addressed in the response.

3. AI-Powered Draft Generation

The extracted email content is sent to OpenAI's GPT-4 model with carefully crafted prompts. The AI analyzes the email's intent, tone, and content, then generates a professional, context-aware draft reply. You can customize the AI instructions to match your specific brand voice and response style.

4. Fastmail API Integration

The workflow authenticates with Fastmail's JMAP API to retrieve session details and mailbox information. It identifies your "Drafts" folder and prepares the necessary API calls to create new draft emails with the AI-generated content.

5. Draft Creation & Delivery

The system compiles all response elements—AI-generated body, original email context, recipient information—and uploads the complete draft to your Fastmail drafts folder. You receive a notification and can review, edit, and send the response with one click.

What You'll Need

  1. Fastmail Account with JMAP API access enabled (most paid Fastmail plans include this)
  2. OpenAI API Key with GPT-4 access (available through OpenAI's developer platform)
  3. n8n Instance (cloud or self-hosted) with HTTP Request and Email Trigger nodes available
  4. IMAP Credentials for your Fastmail account (username/app password)
  5. Basic understanding of API authentication and webhook concepts

Pro tip: Start with a dedicated email folder for testing before applying automation to your main inbox. This lets you verify AI response quality without affecting real customer communications.

Quick Setup Guide

  1. Download & Import: Click the download button above to get the JSON file, then import it into your n8n instance.
  2. Configure Email Trigger: Set up the IMAP node with your Fastmail server details (imap.fastmail.com) and credentials.
  3. Add API Credentials: Create HTTP Header authentication credentials in n8n using your Fastmail API token.
  4. Connect OpenAI: Add your OpenAI API key to n8n's credential manager for GPT-4 access.
  5. Test & Activate: Send a test email to your monitored folder, verify the draft appears in Fastmail, then activate the workflow.

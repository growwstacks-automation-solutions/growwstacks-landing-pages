What This Workflow Does

This automation transforms your Telegram messenger into a personal LinkedIn content assistant. Instead of manually drafting posts, you simply send a topic idea—via text or voice note—to your Telegram bot. The workflow uses GPT-4 to generate a professionally crafted LinkedIn post, sends it back to you for review, and after your approval, publishes it directly to your LinkedIn account.

The approval loop ensures you maintain full control over the final output. You can request revisions (“make it shorter,” “more technical,” “add hashtags”) before the post goes live. This eliminates the friction between idea capture and publishing, turning spontaneous thoughts into polished content within minutes.

For business owners, marketers, and thought leaders, this workflow saves hours per week on content creation, ensures consistent posting frequency, and leverages AI to maintain a high-quality brand voice without the manual effort.

How It Works

1. Input & Initial Processing

The workflow starts when you send a message to your Telegram bot. If you send a voice note, it’s converted to text using OpenAI’s Whisper. The system prepares a single clean input for the AI agent.

2. AI Core & Iteration Loop

GPT-4 analyzes your request, drafts a LinkedIn post following specific formatting rules (hook, main content, hashtags), and sends it back to you via Telegram. You can reply with feedback like “make it more personal” or “add a call-to-action.” The AI remembers the conversation context and revises the draft accordingly.

3. Approval & Publishing

When you say “approved” or “ok,” the workflow extracts the final post text, sends it to LinkedIn via the Blotato API, and waits for confirmation. If publishing succeeds, you receive a success message with the post link; if it fails, you get an error notification.

4. Final Notification

The bot sends you a confirmation with the published LinkedIn post URL, closing the loop and letting you know your content is live.

What You'll Need

  1. Telegram Bot API Key – Create a bot via BotFather on Telegram.
  2. OpenAI API Key – For GPT-4 and Whisper speech-to-text.
  3. Blotato API Key – Connect your LinkedIn account to Blotato for publishing.
  4. n8n instance – Self-hosted or cloud (n8n.cloud).
  5. A LinkedIn account – Where the posts will be published.

Pro tip: Set up your Blotato LinkedIn connection first. This often requires OAuth approval via LinkedIn’s developer portal—do this before importing the workflow to avoid delays.

Quick Setup Guide

  1. Import the JSON template into your n8n instance.
  2. Add credentials for Telegram Bot, OpenAI, and Blotato in n8n’s credential management.
  3. Test the Telegram trigger by sending a message to your bot.
  4. Adjust the AI system prompt inside the “AI: Draft & Revise Post” node to match your brand voice.
  5. Activate the workflow and start sending content ideas via Telegram.

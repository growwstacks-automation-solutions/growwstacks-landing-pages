AI Travel Assistant: Create City Trip Itineraries with Telegram & GPT-4o

Transform your Telegram bot into a smart travel planner that generates personalized 3-day itineraries for any city instantly.

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automation solves the time-consuming problem of creating personalized travel itineraries. Instead of spending hours researching destinations, accommodations, and activities for each client, this workflow instantly generates detailed 3-day trip plans through a simple Telegram interface.

Travel businesses, bloggers, and agencies can deploy this as a 24/7 AI travel assistant that understands user preferences through hashtags like #luxury, #budget, #romantic, or #family. Users simply message a city name with their preferred style, and within seconds receive a comprehensive itinerary including sights, dining recommendations, local tips, and logistical advice.

The system bridges conversational messaging with advanced AI, creating an engaging customer experience that feels personal and responsive while operating completely autonomously.

How It Works

1. Telegram Message Trigger

The workflow activates when a user sends a message to your Telegram bot. It captures the text input, which typically includes a city name and optional style hashtag (e.g., "Paris #luxury" or "Tokyo #cultural").

2. User Experience Enhancement

Before processing, the bot sends a "typing..." indicator to provide immediate feedback, making the interaction feel responsive and human-like.

3. Style Detection & Routing

A switch node analyzes the message for predefined hashtags that determine travel style. It routes the request accordingly—whether it's luxury, budget, adventure, family-friendly, romantic, cultural, nature-focused, or nightlife-oriented travel.

4. AI Prompt Construction

The system builds a detailed JSON prompt for the AI, incorporating the detected city, travel style, duration (default 3 days), and specific instructions for generating practical, locally-informed recommendations.

5. AI Itinerary Generation

GPT-4o or GPT-5 processes the prompt to create a structured 3-day itinerary with daily schedules, must-see attractions, restaurant suggestions, transportation tips, and insider advice tailored to the selected travel style.

6. Response Delivery

The formatted itinerary is sent back to the user through Telegram, complete with emojis and clear section breaks for easy reading. The entire process takes seconds, delivering value immediately.

What You'll Need

  1. Telegram Bot Token: Created through @BotFather on Telegram with basic configuration.
  2. AI/ML API Access: An API key from OpenAI (GPT-4o) or another compatible AI service provider.
  3. n8n Instance: Either self-hosted n8n or n8n.cloud account with workflow execution capabilities.
  4. Basic Web Knowledge: Understanding of how to import JSON workflows and configure API credentials.
  5. Travel Domain Knowledge: Optional but helpful for customizing prompts and validating AI outputs for your specific audience.

Pro tip: Start with a limited set of hashtags (3-5 styles) to keep initial testing simple. You can expand to more niche travel styles like #solo, #foodie, or #photography once the core workflow is stable.

Quick Setup Guide

  1. Create your Telegram bot by messaging @BotFather, following the prompts to set a name and username, then securely save the provided API token.
  2. Obtain AI API credentials from your chosen provider (OpenAI, Anthropic, or others compatible with n8n's AI nodes).
  3. Import the template into your n8n instance using the downloaded JSON file via the workflow import function.
  4. Configure credentials in n8n for both Telegram and your AI service, connecting them to the corresponding nodes in the workflow.
  5. Activate the workflow and test by messaging your bot with sample requests like "Barcelona #budget" or "Kyoto #cultural".
  6. Customize prompts based on your business focus—add specific restaurant categories, activity types, or local tips relevant to your audience.

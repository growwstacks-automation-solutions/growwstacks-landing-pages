What This Workflow Does

In today's information-saturated world, professionals waste hours daily scanning news sites, blogs, and industry publications. This intelligent automation solves that problem by transforming passive news consumption into active intelligence gathering.

The workflow automatically monitors your selected RSS feeds, filters for the most recent and relevant content, uses advanced AI (GPT and Gemini) to extract key insights, and delivers concise, personalized summaries directly to your email and Telegram. It turns information overload into actionable intelligence, ensuring you stay informed without the time drain.

How It Works

1. RSS Feed Monitoring & Filtering

The workflow begins by polling your configured RSS feeds. It immediately applies time-based filters to include only articles published within the last 24 hours, ensuring your digest is always current and relevant.

2. AI-Powered Content Extraction

For each qualifying article, the system uses ScrapeGraphAI to intelligently scrape and clean the main content from web pages, removing ads, navigation, and other noise. Then, Google Gemini analyzes this clean text to identify and extract the most valuable information.

3. Intelligent Summarization & Aggregation

OpenAI's GPT models then synthesize the extracted information from multiple articles into coherent, well-structured summaries. The AI is instructed to highlight key concepts, main arguments, and actionable insights, creating a digest that's more valuable than the sum of its parts.

4. Multi-Channel Delivery

The final personalized digest is formatted as a professional email and sent via Gmail for detailed review. Simultaneously, a condensed version is delivered to your Telegram channel or chat for instant mobile access, ensuring you're informed wherever you are.

What You'll Need

  1. n8n instance (cloud or self-hosted) with community nodes enabled
  2. ScrapeGraphAI account for web content extraction
  3. OpenAI API key for advanced summarization capabilities
  4. Google Gemini API access for content analysis
  5. Gmail account configured in n8n for email delivery
  6. Telegram Bot token and chat/channel ID for instant notifications
  7. RSS feed URLs for the news sources you want to monitor

Pro tip: Start with 2-3 high-quality RSS feeds rather than dozens. The AI produces better summaries when it has focused, relevant content to work with. You can always add more feeds once the workflow is running smoothly.

Quick Setup Guide

  1. Import the template into your n8n instance using the downloaded JSON file.
  2. Install the ScrapeGraphAI community node from the n8n marketplace.
  3. Configure all credentials in n8n's settings: ScrapeGraphAI, OpenAI, Google Gemini, Gmail, and Telegram.
  4. Update the RSS Read node with your actual feed URLs.
  5. Set your Telegram chat ID in the Send to Telegram node.
  6. Test the workflow manually to ensure all connections work and the output matches your expectations.
  7. Schedule the workflow to run daily at your preferred time (e.g., 7 AM for a morning digest).

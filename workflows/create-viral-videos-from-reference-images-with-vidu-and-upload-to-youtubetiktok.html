What This Workflow Does

This automation solves the time-intensive challenge of creating consistent, engaging short-form videos for social media platforms like YouTube and TikTok. Marketing teams, content creators, and businesses often struggle to produce regular video content due to the manual effort required in editing, uploading, and optimizing for each platform.

The workflow transforms up to seven reference images into AI-generated videos using the VIDU "Reference to Video" model. It then automatically publishes these videos to YouTube and TikTok with SEO-optimized titles, eliminating repetitive manual tasks. This ensures a steady content pipeline while maintaining visual consistency across campaigns.

By automating the entire process from image input to multi-platform distribution, businesses can scale their video marketing efforts without increasing production costs or team workload. The system handles status polling, title generation, and format compatibility, delivering ready-to-publish content on schedule.

How It Works

1. Form Trigger & Input Processing

A user submits a web form containing a text prompt describing the desired video and a list of reference image URLs. The workflow processes these URLs, converting them from text strings into a structured array format suitable for the AI video generation API.

2. AI Video Generation Initiation

The processed prompt and image array are sent to the Fal.ai VIDU API endpoint to start the video generation job. This step returns a request ID used for tracking the creation process through subsequent status checks.

3. Status Polling & Completion Monitoring

The workflow enters a loop where it periodically checks the generation job status using the request ID. It waits for intervals (typically 60 seconds) and continues polling until the status confirms "COMPLETED," ensuring the video is fully generated before proceeding.

4. Result Retrieval & Title Optimization

Once the video is ready, the workflow fetches the generated video file URL. Simultaneously, the original user prompt is sent to an AI model (GPT-4o-mini) to create an engaging, SEO-optimized title for social media publication.

5. Multi-Platform Distribution

The video file is downloaded from the generated URL. A copy is saved to Google Drive for archival storage. The video, along with the AI-generated title, is automatically uploaded to YouTube and TikTok via the Upload-Post.com API service, completing the cross-platform publishing cycle.

What You'll Need

  1. n8n instance – Self-hosted or cloud version to run the automation.
  2. Fal.ai API key – For accessing the VIDU reference-to-video generation service.
  3. Upload-Post.com API key – For automated YouTube and TikTok publishing (10 free uploads monthly).
  4. Google Drive credentials – Optional for video storage archival (configure folder access).
  5. OpenAI/OpenRouter access – For AI-generated title optimization (GPT-4o-mini model).
  6. Reference images – Up to seven image URLs per video generation request.

Quick Setup Guide

  1. Import the template – Download the JSON file and import it into your n8n workspace.
  2. Configure Fal.ai credentials – Edit the "Create Video" HTTP node header auth with your API key (Authorization: Key YOUR_KEY).
  3. Set up Upload-Post.com – Add your API key to both YouTube and TikTok upload nodes (Authorization: Apikey YOUR_KEY). Replace "YOUR_USERNAME" with your configured social profile name.
  4. Adjust Google Drive settings – Modify the "Upload Video" node folderId parameter if you want different storage destination.
  5. Configure title generation – Ensure OpenAI/OpenRouter credentials are valid in the "Generate title" node.
  6. Test with sample images – Run the workflow with sample image URLs and prompt to verify full pipeline functionality.
  7. Schedule or trigger – Set up webhook triggers or schedule regular executions based on your content calendar.

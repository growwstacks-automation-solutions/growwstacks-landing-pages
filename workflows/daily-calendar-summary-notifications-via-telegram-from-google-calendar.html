What This Workflow Does

This automation solves a common productivity pain point: the morning scramble to review your calendar. Instead of manually opening Google Calendar each day, this workflow automatically fetches all your scheduled events and sends a clean, formatted summary directly to your Telegram app.

It acts as your personal assistant, delivering your agenda at a scheduled time (like 7 AM) so you can start your day fully informed. The workflow checks for events, formats details like meeting titles, times, locations, and video links, and sends a single, easy-to-read message. If you have no events scheduled, it sends a "clear day" notification instead.

The business value is significant: it reduces cognitive load, eliminates the risk of missing meetings, and centralizes your schedule in a tool you already use constantly—your messaging app. This is perfect for remote teams, busy executives, freelancers, or anyone who manages their day through Google Calendar.

How It Works

The workflow follows a logical sequence to transform calendar data into a friendly Telegram message.

Step 1: Scheduled Trigger

A Schedule Trigger node is set to run daily at your chosen time (e.g., 7:00 AM). This acts as the automation's starting pistol, initiating the process every morning without any manual intervention.

Step 2: Fetch Today's Events

The workflow connects to your Google Calendar using secure OAuth credentials. It queries for all events occurring "today," retrieving key details like event names, start/end times, descriptions, locations, and attendee lists.

Step 3: Process & Format the Data

A Function or Code node processes the raw calendar data. It filters out all-day events if needed, calculates durations, extracts video conference links, and structures the information into a readable summary string, often using bullet points or emojis for clarity.

Step 4: Send to Telegram

The formatted message is passed to the Telegram node. Using a Telegram Bot token, the workflow sends the summary to a specified chat or channel. The message appears as a regular notification from your bot, seamlessly integrating into your daily communication flow.

Step 5: Error Handling & Logging

Optional error-handling branches catch issues like calendar access failures or Telegram API errors. These can log the problem for review or send an alert to a different channel, ensuring the system's reliability.

Pro tip: You can extend this workflow to include weather forecasts for meeting locations, calculate travel time, or even send reminders to other attendees—all within the same automated sequence.

Quick Setup Guide

Follow these steps to get your daily calendar summaries running in under 10 minutes.

  1. Download the Template: Click the "Download Template" button on this page to get the JSON file.
  2. Import into n8n: In your n8n dashboard, go to Workflows > Import from File and select the downloaded JSON.
  3. Configure Google Calendar Credentials: Create an OAuth2 app in Google Cloud Console, enable the Calendar API, and add the Client ID and Secret to n8n's Google Calendar node.
  4. Set Up Your Telegram Bot: Message @BotFather on Telegram to create a new bot and obtain its API token. Add this token to the Telegram node in the workflow.
  5. Enter Your Chat ID: Find your personal Telegram Chat ID (you can use @userinfobot) and paste it into the Telegram node's "Chat ID" field.
  6. Adjust the Schedule: Modify the Schedule Trigger node to run at your preferred time (e.g., 7:00 AM in your timezone).
  7. Test & Activate: Click "Execute Workflow" once to test. If successful, activate the workflow to run automatically on the defined schedule.

Pro tip: Use n8n's "Test" feature on the Google Calendar node first to ensure it can fetch your events. This isolates any authentication issues before running the full workflow.

Daily Nearby Garage Sales Alerts via Telegram

Get a personalized list of garage sales happening today, based on your current location, directly in Telegram each morning. Automate discovery and never miss a bargain.

Download Template JSON · n8n compatible · Free

What This Workflow Does

For bargain hunters, collectors, and anyone who loves finding hidden treasures, manually searching for garage sales every day is a time-consuming chore. This automation solves that by delivering a curated list of nearby sales directly to your Telegram each morning.

The workflow integrates your location (via Home Assistant or a similar service) with Brocabrac.fr—a popular garage sale listing platform. It automatically scrapes new listings, filters them based on distance and quality, formats the details, and sends you a clean, actionable message. You wake up knowing exactly what sales are happening near you today, without ever opening a browser.

This turns sporadic, manual searching into a reliable, daily discovery system. It’s perfect for individuals who frequent garage sales, small resellers looking for inventory, or community groups wanting to share local events.

How It Works

Step 1: Trigger & Location Detection

A scheduled trigger runs every morning. It first retrieves your current location—either from a static setting or dynamically via Home Assistant if you’re mobile. This ensures the search radius is always relevant to where you are.

Step 2: Scrape Garage Sale Listings

The workflow sends a request to Brocabrac.fr (or similar site) with your location parameters. It extracts listing data including address, date, time, description, and sometimes photos.

Step 3: Filter & Prioritize

Using conditional logic, the workflow filters out listings that are too far, expired, or low-quality. You can set rules like “within 5 miles,” “only weekends,” or “contains keywords like ‘tools’ or ‘furniture.’”

Step 4: Format & Send Telegram Alert

The filtered results are structured into a readable message—often with bullet points, maps links, and highlights. This message is then sent via the Telegram Bot API to your private chat or a group.

What You'll Need

  1. A running n8n instance (cloud or self‑hosted).
  2. A Telegram Bot token (created via @BotFather).
  3. Your Telegram chat ID (personal or group).
  4. Access to Brocabrac.fr or a similar garage sale listing site.
  5. Optional: Home Assistant or another location service if you want dynamic location detection.

Quick Setup Guide

1. Download the template JSON file using the button above.

2. Import it into your n8n workspace via the import function.

3. Configure the Telegram node with your Bot token and chat ID.

4. Set up the HTTP Request node to point to Brocabrac.fr (or your preferred site).

5. Adjust the filter logic to match your desired radius and keywords.

6. Set the schedule trigger to run at your preferred time (e.g., 8 AM daily).

7. Activate the workflow and test with a manual run.

Pro tip: Add a Google Sheets node to log found sales for future reference, or integrate with Google Calendar to automatically block time for visits.

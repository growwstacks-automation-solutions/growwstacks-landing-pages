What This Workflow Does

This automation solves the time-consuming process of manually editing, saving, and distributing images for content creation. Social media managers, designers, and marketing teams often spend hours each week on repetitive image tasks: uploading files, applying edits, saving versions, and sending them to team members.

The workflow creates a seamless pipeline where users submit images through a form with editing prompts. The system automatically saves the original to Google Drive, processes it through DALL-E 2's AI editing capabilities, converts the result to a standardized format, and delivers the final image directly to a Telegram chat. This eliminates manual steps while ensuring consistent quality and organization.

How It Works

1. Form Submission & Initial Processing

The workflow begins with a web form where users upload an image and provide editing instructions. This triggers the automation, which validates the file type and extracts the editing prompt for AI processing.

2. Cloud Storage Backup

Before any editing occurs, the original image is automatically uploaded to a designated Google Drive folder. This creates a backup and establishes version control, ensuring no original files are lost during the AI editing process.

3. AI-Powered Image Editing

The workflow sends the image and editing prompt to DALL-E 2's API. The AI interprets the natural language instructions and applies the requested edits—whether that's changing styles, adding elements, adjusting composition, or applying filters based on the description.

4. Format Conversion & Organization

Once edited, the image is converted to a standardized PNG format for consistency across platforms. The system applies naming conventions based on the edit type and timestamp, then saves the final version to a separate Google Drive folder for edited content.

5. Instant Delivery via Telegram

The finalized image is sent directly to a specified Telegram chat or channel. Team members receive immediate notification with the edited image ready for review, approval, or direct sharing to social media platforms.

What You'll Need

  1. OpenAI API key with access to DALL-E 2 image editing capabilities
  2. Google Cloud Platform account with Drive API enabled and OAuth2 credentials
  3. Telegram Bot Token created via BotFather and the target chat ID
  4. n8n instance (self-hosted or cloud) with internet access
  5. Web server for hosting the submission form (can be n8n's webhook)

Pro tip: Create separate Google Drive folders for "Originals" and "Edited" to maintain clear version history. Use date-based subfolders (YYYY-MM-DD) for automatic organization that scales with your content volume.

Quick Setup Guide

  1. Import the template: Download the JSON file and import it into your n8n instance through the workflow import function.
  2. Configure credentials: Add your OpenAI API key, Google Drive OAuth2 credentials, and Telegram bot token to n8n's credential management.
  3. Set destination parameters: Update the Google Drive folder IDs and Telegram chat ID in the respective nodes.
  4. Customize image settings: Adjust the DALL-E 2 parameters like image size (default 1024×1024) and edit strength based on your quality requirements.
  5. Test the form: Use the webhook URL provided by n8n to submit a test image with editing instructions and verify the complete pipeline.
  6. Deploy for team use: Create a simple frontend form pointing to your webhook URL and share it with your team or clients.

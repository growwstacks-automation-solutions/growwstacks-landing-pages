What This Workflow Does

Manual LinkedIn research is one of the biggest time-sinks in sales and recruitment. This automation eliminates that grind by turning LinkedIn profile URLs into rich, structured lead data automatically. Simply paste URLs into Google Sheets, run the workflow, and get back complete profiles with names, job titles, companies, industries, company websites, sizes, career history, and even recent LinkedIn posts—all neatly organized in your spreadsheet.

The workflow intelligently detects which rows need enrichment (where the "First Name" field is empty), scrapes each LinkedIn profile using Apify's professional scraping actor, waits for completion with smart polling, extracts over a dozen valuable data points, and writes everything back to your sheet. It handles errors gracefully, tracks status for each row, and stores the raw Apify dataset ID for future reference if you need deeper analysis.

For sales teams, recruiters, and marketers, this means moving from hours of manual profile review to batch-processing hundreds of leads in minutes. You get consistent, structured data ready for segmentation, personalization, and integration into your CRM or marketing automation platforms.

How It Works

1. Read Unenriched Rows from Google Sheets

The workflow starts by scanning your designated Google Sheet for rows where the "First Name" column is empty. This smart detection ensures you only process new leads and avoid duplicate work. It reads the LinkedIn profile URLs you've added and prepares them for scraping.

2. Trigger Apify LinkedIn Profile Scraper

For each URL, the workflow sends a request to Apify's "dev_fusion~linkedin-profile-scraper" actor. This professional scraping tool handles LinkedIn's structure, authentication challenges, and rate limiting—saving you from building and maintaining your own scraper. The workflow initiates the scrape and receives a dataset ID to track progress.

3. Poll for Completion with Smart Retry Logic

LinkedIn scraping takes time. Instead of failing or timing out, the workflow checks the Apify dataset status every 15 seconds (up to 20 attempts). This patient polling ensures reliable completion even for larger profiles or during peak traffic, without overwhelming Apify's API or your n8n instance.

4. Extract and Structure Profile Data

Once scraping completes, the workflow fetches the raw JSON data and extracts key information: personal details (name, location, headline, bio), current role (title, description, company, industry, size, website), additional concurrent positions, most recent previous employer, and the last two LinkedIn posts with links. This transforms unstructured profile HTML into clean, usable fields.

5. Write Enriched Data Back to Google Sheets

The final step updates the original Google Sheets row with all extracted information. Each field gets its own column: First Name, Last Name, Job Position, Location, Industry, Company Name, Company URL, Company Size, and a rich text summary in "LI Other Profile Information." The workflow also marks the status as "Completed" and stores the Apify dataset ID for auditability.

What You'll Need

  1. Apify Account & API Token: Sign up at Apify.com, get your API token from settings, and add credits to run the LinkedIn scraper actor.
  2. Google Sheets with OAuth2 Access: A Google Cloud project with Sheets API enabled and OAuth2 credentials configured in n8n.
  3. Google Sheet Template: A sheet with these columns: LinkedIn (URL), First Name, Last Name, Job Position, Location, Industry, Company Name, Company URL, Company Size, LI Other Profile Information, Status, Apify ID, Add date, row_number.
  4. n8n Instance: Either n8n.cloud or self-hosted n8n with internet access to Apify and Google APIs.

Pro tip: Start with a test sheet containing 5-10 LinkedIn URLs to verify everything works before processing large batches. Monitor your Apify usage credits—scraping many profiles consecutively may require sufficient balance.

Quick Setup Guide

  1. Duplicate the template sheet or create your own with the required column structure.
  2. Add your LinkedIn profile URLs to the "LinkedIn" column, leaving "First Name" blank to trigger enrichment.
  3. Import this workflow into your n8n instance using the downloaded JSON file.
  4. Configure three HTTP Request nodes with your Apify API token (replace "YOUR_APIFY_API_KEY").
  5. Connect Google Sheets OAuth2 credentials to both Google Sheets nodes in the workflow.
  6. Update the document ID in the Google Sheets nodes if using your own sheet instead of the template.
  7. Run the workflow manually—it will process all unenriched rows sequentially and update your sheet.

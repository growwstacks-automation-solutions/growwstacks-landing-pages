Expense Logging with Telegram & Google Sheets

Automate personal expense tracking. Log purchases via Telegram, auto-save to Sheets, get weekly summaries and budget alerts.

Download Template JSON · n8n compatible · Free

What This Workflow Does

Manual expense tracking is tedious, error-prone, and often forgotten. This automation solves that by creating a seamless system where you can log expenses instantly from your phone using Telegram, with everything automatically organized in Google Sheets.

The workflow captures expense details via simple Telegram commands, categorizes them, timestamps entries, and maintains a clean, searchable financial record. Beyond just logging, it provides weekly spending summaries and sends immediate alerts if you exceed your budget, giving you real-time control over your finances without manual spreadsheet work.

How It Works

The automation follows a logical flow from input to analysis, ensuring every expense is captured and processed correctly.

Step 1: Expense Input via Telegram

You send a message to your Telegram bot using a simple format like "/spent 15.50 coffee". The Telegram Trigger node listens for these messages and captures the amount, description, and timestamp.

Step 2: Data Processing & Validation

The workflow extracts the numerical amount and text description, validates the format, and adds metadata like date, category (based on keywords), and user information. A Code node can perform additional calculations or transformations if needed.

Step 3: Google Sheets Integration

Processed expense data is appended to a designated Google Sheet. Each entry creates a new row with columns for date, amount, description, category, and any additional tags you've configured.

Step 4: Weekly Summary Generation

A Schedule Trigger node runs every Sunday, querying the Google Sheet for the past week's expenses. It calculates totals by category, identifies spending patterns, and formats this into a readable summary.

Step 5: Budget Monitoring & Alerts

The workflow compares weekly spending against your predefined budget threshold. If you exceed the limit, it immediately sends a Telegram alert, allowing you to adjust spending before the week ends.

What You'll Need

  1. A Telegram account and a bot created via BotFather
  2. Google Sheets with edit permissions
  3. n8n instance (cloud or self-hosted)
  4. Google Cloud credentials with Sheets API enabled
  5. Basic understanding of Telegram bot commands

Pro tip: Create expense categories in your Google Sheet before importing the workflow. Common categories include Food, Transportation, Supplies, Entertainment, and Business. This makes reporting and analysis much easier.

Quick Setup Guide

Follow these steps to get your expense tracking automation running in minutes:

  1. Download and import the template JSON file into your n8n instance.
  2. Create your Telegram bot via @BotFather, copy the API token, and add it to the Telegram Trigger node.
  3. Set up Google Sheets connection in n8n using OAuth2 credentials with Sheets API access.
  4. Duplicate the provided Google Sheet template and update the Sheet ID in the workflow nodes.
  5. Configure your weekly budget threshold in the "Check Budget" node (default is €100).
  6. Test the workflow by sending "/spent [amount] [description]" to your Telegram bot.
  7. Activate the workflow and start tracking expenses automatically.

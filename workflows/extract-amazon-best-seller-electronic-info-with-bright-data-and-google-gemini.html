What This Workflow Does

Keeping up with Amazon's best sellers in electronics is a manual, time-consuming task prone to errors and delays. This automation solves that by scraping Amazon's Electronics section using Bright Data's reliable Web Unlocker API, processing the raw HTML with Google Gemini's AI to extract structured product data (rankings, prices, ratings, promotions), and forwarding clean JSON to any webhook endpoint.

It transforms unstructured webpage content into actionable business intelligence. Instead of manually copying and pasting data, you get scheduled, automated insights delivered directly to your analytics tools, CRM, or internal dashboards. This workflow is designed for eCommerce analysts, product intelligence teams, and marketers who need real-time competitive data without the manual overhead.

The integration of Bright Data ensures you bypass Amazon's anti‑scraping measures, while Google Gemini adds smart parsing—identifying key attributes like discount percentages, review counts, and product trends. The result is a reliable pipeline that turns Amazon's dynamic best seller pages into structured datasets ready for analysis.

How It Works

Step 1: Trigger & Scrape

The workflow starts on a schedule (daily, weekly) or manual trigger. It calls Bright Data's Web Unlocker API with the Amazon Electronics best sellers URL. Bright Data returns clean HTML without blocks, ensuring consistent data access.

Step 2: AI‑Powered Parsing

The raw HTML is passed to Google Gemini (Flash Exp model). A custom prompt instructs Gemini to extract product names, prices, star ratings, review counts, and promotional text. Gemini outputs structured JSON with normalized fields.

Step 3: Data Transformation

Using n8n's Set node, the JSON is cleaned and formatted—removing empty fields, converting strings to numbers, and adding metadata like extraction timestamp and category.

Step 4: Delivery to Webhook

The final structured dataset is sent via HTTP Request to your configured webhook URL. This can be a Google Sheets API, Airtable, Slack channel, database endpoint, or custom dashboard.

What You'll Need

  1. Bright Data Account – Sign up at Bright Data, create a Web Unlocker zone, and obtain your API token.
  2. Google Gemini API Key – Access through Google AI Studio or Vertex AI.
  3. n8n Instance – Self‑hosted or n8n.cloud.
  4. Destination Webhook URL – Where you want the structured data sent (e.g., Google Sheets, Airtable, Slack).
  5. Basic n8n Knowledge – Understanding of credentials, nodes, and workflow execution.

Quick Setup Guide

  1. Import the Template – Download the JSON file and import into your n8n workspace.
  2. Configure Bright Data – Create a Header Auth credential in n8n with your Bright Data token (Bearer format).
  3. Set Up Google Gemini – Add your Gemini API key in the Google Gemini node credentials.
  4. Update Amazon URL – Modify the HTTP Request node URL to your target Amazon category (e.g., Computers & Accessories).
  5. Set Destination Webhook – Replace the final HTTP Request node URL with your endpoint.
  6. Test & Schedule – Execute once manually, then set a schedule trigger for daily/weekly runs.

Pro tip: Use the Set node to add custom fields—like “category” or “extraction_date”—before sending to your webhook. This makes the data easier to filter and analyze downstream.

What This Workflow Does

Manual company research is time-consuming and inconsistent. Sales teams spend hours visiting websites, copying information, and organizing data into spreadsheets. This process is prone to errors, misses important details, and doesn't scale.

This AI-powered automation solves that problem completely. When you send any corporate website URL through a chat interface, the workflow uses Google Gemini AI to investigate the website, extract key company information, and return structured data in seconds. It handles everything from company description and industry classification to contact details and service offerings.

The conversational setup means you can ask follow-up questions, try different URLs, or request specific data points—all through natural conversation. It's like having a research assistant available 24/7 that never gets tired or makes transcription errors.

How It Works

The workflow combines AI intelligence with structured automation to deliver reliable company data extraction.

Step 1: Chat Trigger & URL Submission

You start a conversation and send a corporate website URL (like https://example.com). The workflow captures this input and prepares it for processing. The chat interface makes the interaction natural and intuitive—no forms or complicated interfaces.

Step 2: Website Analysis with Gemini AI

The AI agent visits the website, reads its content, and analyzes the structure. It identifies key sections like About Us, Services, Team, Contact, and more. Gemini's advanced understanding allows it to interpret context, recognize patterns, and extract meaningful information even from complex or poorly structured sites.

Step 3: Structured Data Extraction

Based on your configuration, the AI extracts specific data points: company name, description, industry, size, location, services, contact information, social media links, and other relevant details. The workflow formats this into clean, structured JSON that's ready for use in other systems.

Step 4: Validation & Response

Before returning results, the workflow validates the extracted data for completeness and accuracy. It then presents the information in a clear, organized format through the chat interface. You can immediately use this data or ask for additional analysis.

What You'll Need

  1. n8n Instance: A running n8n installation (cloud or self-hosted) to import and execute the workflow.
  2. Google Gemini API Key: Obtain from Google AI Studio—this powers the website analysis and data extraction.
  3. Chat Interface: The workflow is designed for chat interaction but can be adapted to webhooks or other triggers.
  4. Basic n8n Knowledge: Understanding how to import workflows, set up credentials, and modify node configurations.
  5. Target Websites: Corporate websites with publicly accessible information (no login required).

Quick Setup Guide

Get this automation running in your environment in under 15 minutes:

  1. Download the Template: Use the download button above to get the JSON workflow file.
  2. Import to n8n: In your n8n instance, go to Workflows → Import from file and select the downloaded JSON.
  3. Configure Gemini Credentials: Click the Gemini node, add your Google AI Studio API key, and save the credentials.
  4. Customize Data Fields: Open the Config node to modify which company data points to extract (targetCompanyFields) and set your preferred output language.
  5. Test with a URL: Trigger the workflow by sending a corporate website URL through your chat interface and verify the extracted data.
  6. Connect to Your Systems: Add nodes to send extracted data to your CRM, database, or other business applications.

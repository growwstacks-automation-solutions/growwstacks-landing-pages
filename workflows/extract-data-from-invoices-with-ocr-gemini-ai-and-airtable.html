What This Workflow Does

Manual invoice processing is a time-consuming, error-prone task that costs businesses thousands of hours annually. This automation solves that problem by creating a complete hands-off system for invoice data extraction.

The workflow automatically monitors a designated folder for new invoice files (PDF, JPG, or PNG). When a file appears, it uses OCR technology to read the text, then sends that text to Google Gemini AI to intelligently identify and extract specific fields like invoice numbers, dates, totals, vendor information, and line items. Finally, it structures this data and creates a clean record in your Airtable database.

What used to take 5-10 minutes per invoice now happens in seconds with near-perfect accuracy, freeing your team for higher-value work while ensuring your financial data is always up-to-date and searchable.

How It Works

1. File Detection & Trigger

The workflow starts by watching a specific folder on your server or cloud storage. When any new invoice file (PDF, JPG, PNG) appears, it immediately triggers the automation process without any manual intervention.

2. Optical Character Recognition (OCR)

The invoice file is processed through Tesseract OCR, which extracts all readable text from the document regardless of layout or format. This converts visual information into machine-readable text while preserving the structural relationships between data points.

3. AI-Powered Data Extraction

The raw OCR text is sent to Google Gemini AI with a carefully crafted prompt that instructs the model to identify specific invoice fields: invoice number, date (standardized to YYYY-MM-DD), subtotal, tax, total amount, currency, vendor name, contact information, and shipping addresses.

4. Data Validation & Structuring

Extracted data passes through validation checks to ensure completeness and accuracy. The workflow formats dates consistently, converts currency symbols to standard codes, and structures the information into a clean JSON object ready for database insertion.

5. Airtable Integration

The structured data creates a new record in your specified Airtable base. Each field maps to a column in your invoices table, creating a searchable, filterable database of all processed invoices with all key information instantly accessible.

6. Notification & Logging

Optionally, the workflow can send a Telegram notification confirming successful processing or alerting to any issues. All processing steps are logged for audit trails and troubleshooting.

What You'll Need

  1. Self-hosted n8n instance (this workflow uses community nodes not available in n8n Cloud)
  2. Google Gemini API key (available through Google AI Studio)
  3. Airtable account with a base and table configured for invoice data
  4. Folder access where invoice files will be deposited (local server or cloud storage)
  5. Optional: Telegram bot token for success notifications
  6. Community nodes installed: n8n-nodes-tesseractjs and n8n-nodes-pdf-page-extract

Pro tip: Before processing real invoices, test with 5-10 sample files to fine-tune the AI prompt. Adjust the extraction instructions based on your specific invoice formats and required fields.

Quick Setup Guide

Follow these steps to implement this automation in your business:

  1. Install community nodes on your self-hosted n8n instance: n8n-nodes-tesseractjs and n8n-nodes-pdf-page-extract
  2. Import the workflow JSON into your n8n editor using the download button above
  3. Configure credentials for Google Gemini, Airtable, and optional Telegram
  4. Set the watch folder path where invoices will be deposited (ensure n8n has read access)
  5. Map Airtable fields to match the extracted data structure
  6. Test with sample invoices to verify extraction accuracy
  7. Activate the workflow and begin dropping invoices into the watch folder

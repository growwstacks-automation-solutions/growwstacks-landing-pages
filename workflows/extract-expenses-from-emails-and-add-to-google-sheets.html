Automate Expense Reporting: Extract Receipts from Emails to Google Sheets

Stop manual data entry. This free n8n workflow scans your inbox for receipts, extracts data with AI, and logs it to Google Sheets automatically.

What This Workflow Does

Manual expense reporting is a tedious, error-prone chore that wastes valuable time. Employees forget to submit receipts, finance teams spend hours deciphering handwritten notes, and reimbursements get delayed. This automation solves that by creating a seamless pipeline from your inbox to your financial records.

The workflow automatically monitors a designated mailbox for new emails containing receipts or expense reports. When an email arrives with a subject like "Expenses" or "Receipt," it extracts the attachment (PDF, JPG, PNG), sends it to an AI-powered data extraction service (Mindee), parses the key details (vendor, date, amount, tax), and appends a clean, structured record directly into a Google Sheet. It turns a multi-step, manual process into a hands-off, reliable system.

Beyond simple logging, this setup ensures no expense gets lost in an inbox. It provides immediate visibility into spending, accelerates month-end closing, and creates a searchable digital paper trail. For businesses and freelancers alike, it transforms financial admin from a weekly headache into a fully automated background task.

How It Works

The automation follows a logical, event-driven sequence, mimicking the steps a human would take but with perfect consistency and speed.

1. Email Monitoring & Filtering

The workflow starts with an Email Trigger (IMAP) node configured to connect to your business or team mailbox. It polls for new messages at a set interval (e.g., every 15 minutes). A filter condition checks the email subject line for keywords like "Expense," "Receipt," or "Invoice." Only relevant emails proceed, ignoring newsletters and other noise.

2. Attachment Extraction & Preparation

For each matching email, the workflow identifies and extracts file attachments. It handles common formats like PDFs and images. The file is prepared and passed to the next stage for data extraction, ensuring the AI service receives a clean, accessible file.

3. AI-Powered Data Extraction

The attachment is sent to the Mindee node (or a similar OCR/AI service). Here, machine learning models analyze the document, locate key fields—such as merchant name, transaction date, total amount, tax breakdown, and even line items—and convert the unstructured image or PDF data into structured JSON data. This step replaces manual data entry with high-accuracy automated reading.

4. Data Validation & Transformation

Raw extracted data often needs cleaning. A Set node or Function node formats dates consistently, converts currency strings to numbers, and applies any business rules (e.g., categorizing expenses based on merchant keywords). This ensures the data is ready for database entry.

5. Google Sheets Update

The final step uses the Google Sheets node to append the processed expense record as a new row in a predefined spreadsheet. Each column maps to a data point: Date, Vendor, Amount, Category, Description, and a link to the original email. The sheet becomes a live, automatically updated expense ledger.

What You'll Need

  1. An n8n Instance: A self-hosted n8n setup or a n8n.cloud account.
  2. Email Account Access: Login credentials (email/password or app password) for the mailbox you want to monitor. IMAP must be enabled.
  3. Mindee API Key: A free or paid account with Mindee to use their document parsing API.
  4. Google Cloud Project & Service Account: A Google Cloud project with the Google Sheets API enabled, and a service account JSON key file with write access to your target spreadsheet.
  5. A Google Sheet: A prepared spreadsheet with column headers (Date, Vendor, Amount, Category, etc.) where the data will be logged.

Pro tip: For the email connection, consider creating a dedicated service email address (e.g., expenses@yourcompany.com) to keep all receipt emails organized and separate from personal communication.

Quick Setup Guide

Follow these steps to get your automated expense system running in under 30 minutes.

  1. Import the Template: Download the JSON file using the button above. In your n8n instance, go to Workflows > Import from File and select the downloaded file.
  2. Configure Email Trigger: Open the Email Trigger (IMAP) node. Enter your email server details (host, port, user, password). Set the "Subject Contains" filter to your preferred keywords (e.g., "Expense,Receipt").
  3. Set Up Mindee: Create a Mindee account and generate an API key. In the Mindee node in the workflow, paste your API key and select the appropriate document type (e.g., "Receipt").
  4. Connect Google Sheets: In the Google Sheets node, authenticate using OAuth2 or a Service Account. Paste the ID of your target spreadsheet and the name of the worksheet (e.g., "Sheet1").
  5. Test the Flow: Activate the workflow. Send a test email with a receipt attachment to your monitored inbox. Watch as the data appears in your Google Sheet within minutes.

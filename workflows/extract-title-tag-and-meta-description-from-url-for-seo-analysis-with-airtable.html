What This Workflow Does

Manual SEO analysis is time-consuming and error-prone. This automation solves that by automatically extracting title tags and meta descriptions from any list of URLs stored in Airtable. Whether you're conducting competitor research, auditing your own website, or building a content gap analysis, this workflow turns hours of manual copying and pasting into a fully automated process.

The system connects to your Airtable base, reads URLs from a specified field, fetches each webpage's HTML, extracts the precise title and meta description tags, and writes the results back to your database. It handles missing tags gracefully and can process hundreds of URLs in minutes, giving you clean, structured data ready for analysis.

Beyond basic extraction, this workflow forms the foundation for sophisticated SEO monitoring. You can track changes over time, compare your tags against competitors, ensure length compliance, and identify optimization opportunities—all without touching a browser.

How It Works

The workflow follows a logical sequence to transform URLs into actionable SEO data.

Step 1: Fetch URLs from Airtable

The automation begins by querying your Airtable base for records that have a URL but are missing title and meta description data. Using a smart filter formula, it identifies exactly which records need processing, ensuring no duplicate work.

Step 2: HTTP Request to Fetch Page HTML

For each URL, the workflow sends an HTTP request to retrieve the webpage's HTML content. This mimics what search engine crawlers see, giving you the raw source code exactly as it exists online.

Step 3: Parse HTML and Extract Metadata

Using n8n's HTML node, the workflow scans the retrieved HTML to locate the <title> tag and <meta name="description"> tag. These elements are precisely extracted, including their full content.

Step 4: Write Results Back to Airtable

The extracted title and description are then written back to their respective fields in your Airtable record. If a tag is missing or empty, the corresponding field remains blank, helping you identify SEO issues.

Step 5: Schedule and Scale

Once configured, the workflow can run on a schedule—daily, weekly, or monthly—to continuously monitor changes. It processes URLs in batches, making it scalable for large-scale SEO audits across entire websites or competitor lists.

What You'll Need

  1. An Airtable account with a base containing a table for your URL analysis.
  2. n8n instance (cloud or self-hosted) with access to the HTTP Request and HTML nodes.
  3. Basic Airtable setup with at minimum: a URL field (type: URL), a Title Tag field (type: Single line text), and a Meta Description field (type: Long text).
  4. List of URLs to analyze—these can be competitor pages, your own content, or industry resources.

Pro tip: Add formula fields in Airtable to automatically calculate character counts for your title and description tags. Use LEN({Title Tag}) and LEN({Meta Description}) to instantly see if your tags meet SEO length recommendations.

Quick Setup Guide

Follow these steps to implement this SEO extraction automation in under 15 minutes:

  1. Import the template into your n8n workspace using the download button above.
  2. Connect your Airtable nodes by adding your API key and selecting your base and table.
  3. Configure the filter formula in the first Airtable node to target records where URL exists but title and description fields are empty.
  4. Test with a single URL to ensure the HTTP Request and HTML extraction work correctly.
  5. Add a schedule trigger to run the workflow automatically at your preferred interval (daily, weekly, etc.).
  6. Monitor results in your Airtable base and extend the workflow as needed for additional SEO data points.

What This Workflow Does

Manual YouTube channel research is time-consuming and inconsistent. Marketing teams, content creators, and competitive analysts spend hours copying subscriber counts, view statistics, keywords, and banner details into spreadsheets—only to have that data become outdated within days.

This automation solves that problem by connecting the YouTube Metadata API (via RapidAPI) directly to Google Docs. Simply provide a YouTube channel URL, and the workflow fetches comprehensive metadata—including subscribers, total views, country, keywords, banner URLs, and channel description—then formats it professionally and appends it to a designated Google Doc. What used to take 15-20 minutes per channel now happens in seconds with perfect accuracy.

The system creates organized, timestamped records that teams can reference for competitor tracking, influencer vetting, market analysis, or content strategy development. It eliminates manual data entry errors and ensures your competitive intelligence stays current.

How It Works

1. Form Submission Trigger

The workflow activates when a user submits a YouTube channel URL through a simple web form. This makes it accessible to non-technical team members—anyone with a URL can initiate the research process without understanding APIs or automation.

2. YouTube Metadata API Call

Using the RapidAPI connection, the system sends the channel URL to the YouTube Metadata API. It retrieves structured data including channel title, subscriber count, view totals, country, keywords, banner image URLs, and description metadata directly from YouTube's systems.

3. Data Reformating & Enhancement

A code node transforms the raw API response into a clean, readable format with emojis, markdown styling, and logical section breaks. This turns technical API data into business-ready insights that are immediately understandable for stakeholders.

4. Google Docs Integration

The formatted channel analysis is appended to a specified Google Docs document with timestamps and separators. This creates a growing research library where each channel analysis becomes a searchable, shareable section within a master competitive intelligence document.

What You'll Need

  1. A free or paid RapidAPI account with access to the YouTube Metadata API
  2. Google Workspace account with Google Docs API access enabled
  3. n8n instance (cloud or self-hosted) with HTTP Request and Google Docs nodes
  4. A Google Docs document created to receive the channel analyses
  5. YouTube channel URLs you want to analyze (competitors, influencers, or benchmarks)

Pro tip: Create a dedicated Google Drive folder for your YouTube research and set up a template document with standardized headers. The automation will maintain consistent formatting across all entries, making quarterly reviews much easier.

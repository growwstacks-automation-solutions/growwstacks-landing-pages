What This Workflow Does

This automation solves the problem of manually tracking and saving new YouTube playlist items to your Raindrop collection. It periodically checks specified YouTube playlists and automatically saves any new videos to your designated Raindrop collection.

The workflow eliminates hours of manual work while ensuring you never miss important content additions. It's particularly valuable for content curators, researchers, and teams managing shared knowledge bases who need to systematically archive YouTube content.

How It Works

1. Scheduled Playlist Check

The workflow runs on your specified schedule (e.g. hourly) to check designated YouTube playlists for new additions.

2. New Item Detection

It compares the current playlist contents with previous runs, identifying only the new videos added since the last check.

3. Raindrop Integration

Each new video gets saved to your specified Raindrop collection with relevant metadata including title, URL, and description.

What You'll Need

  1. YouTube playlist ID you want to monitor
  2. Raindrop.io account with API access
  3. Raindrop collection ID for saving items
  4. n8n instance (cloud or self-hosted)

Quick Setup Guide

  1. Download the JSON template file
  2. Import into your n8n instance
  3. Configure YouTube and Raindrop credentials
  4. Set your playlist and collection IDs
  5. Adjust schedule frequency as needed
  6. Activate the workflow

Pro tip: For large playlists, consider adding a filter step to only save videos matching specific keywords or published after a certain date.

What This Workflow Does

This automation solves a critical problem for B2B sales and marketing teams: stagnant lead lists. Instead of manually researching competitors or similar companies—a process that can take hours per account—this workflow automatically analyzes your existing customer domains and returns a list of high-quality lookalike companies with similarity scores.

You provide a Google Sheet with your best customer domains. The workflow sends each domain to the CompanyEnrich API, which uses machine learning to identify companies with similar characteristics (industry, size, technology stack, growth patterns). The enriched data—including company names, domains, and similarity percentages—is then written back to your spreadsheet, ready for your outreach team.

This turns your customer success data into a prospecting engine. Every happy customer becomes a pattern for finding more just like them, ensuring your pipeline stays full of qualified, relevant leads that match your proven ideal customer profile.

How It Works

The workflow follows a clean, four-step process that transforms your customer list into expanded prospect lists.

1. Read Source Customer List

The workflow starts by connecting to your Google Sheets document and reading the "Domain" column from your specified tab. It processes each domain individually, preparing them for enrichment. This step includes basic validation to ensure each entry is a properly formatted domain before proceeding to the API call.

2. Enrich with CompanyEnrich API

Each domain is sent to the CompanyEnrich API with your authentication token. The API analyzes the company's characteristics and returns up to 10 similar companies with similarity scores ranging from 0-100%. The workflow handles API rate limits and errors gracefully, ensuring no data is lost if a single request fails.

3. Process and Structure Results

The raw JSON response from CompanyEnrich is parsed and restructured into a clean format. For each input domain, the workflow creates rows containing the similar company's name, domain, similarity score, and additional metadata when available. This transformation makes the data immediately usable in your sales process.

4. Write to Output Spreadsheet

Finally, the enriched data is appended to a new tab in your Google Sheets document. The workflow creates organized columns for easy sorting and filtering, allowing your team to immediately start outreach to the highest-scoring matches. You can schedule this to run weekly or monthly for continuous list expansion.

Pro tip: Set a similarity threshold of 80% or higher in your workflow to focus only on the strongest matches. This prevents your team from wasting time on marginal prospects and keeps your outreach highly targeted.

Quick Setup Guide

Follow these steps to get your B2B lead enrichment automation running in under 15 minutes:

  1. Prepare your Google Sheet: Create a spreadsheet with two tabs. Name the first tab "Input" with a header row containing "Domain" in column A. Add your customer domains (like "example.com") below. Leave the second tab empty and name it "Output."
  2. Import the workflow: Download the JSON template from this page and import it into your n8n instance using the "Import from File" option in the workflow editor.
  3. Configure Google Sheets nodes: Open both the "Read Source List" and "Write Results" nodes. Authenticate with your Google account and select your spreadsheet and the appropriate tabs (Input and Output).
  4. Add your CompanyEnrich API key: Open the "Fetch Similar Companies" HTTP Request node. In the Authentication section, add a Header with key "Authorization" and value "Bearer YOUR_API_KEY_HERE" (replace with your actual key from CompanyEnrich).
  5. Test and schedule: Click "Execute Workflow" to test with 2-3 domains. Check the Output tab for results. Once verified, activate the workflow and set a schedule (weekly recommended) for automatic enrichment.

Important: Start with a small list of 5-10 domains to test API response quality and similarity scores before processing hundreds of companies. This helps you fine-tune your similarity threshold before scaling.

Automate Multi-Platform Video Publishing with AI Descriptions

Publish videos from Google Drive to Instagram, TikTok, and YouTube simultaneously with AI-generated descriptions and centralized Airtable tracking.

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automation solves the time-consuming challenge of publishing video content across multiple social platforms. Content creators, social media managers, and marketing teams spend hours each week manually uploading videos to Instagram, TikTok, and YouTube, writing unique descriptions for each platform, and tracking publication status.

The workflow automatically detects new videos uploaded to a specific Google Drive folder, uses OpenAI to transcribe audio and generate engaging platform-specific descriptions, publishes the video simultaneously to all three platforms via the Upload-Post.com API, and maintains detailed tracking records in Airtable. It transforms a 2-3 hour manual process into a fully automated system that runs in minutes.

How It Works

1. Video Detection & Download

The Google Drive Trigger node monitors your designated folder for new video uploads. When a video is detected, it automatically downloads the file and extracts metadata including filename, upload date, and file size.

2. AI Description Generation

The workflow sends the video to OpenAI's transcription service to convert audio to text. This transcript is then analyzed by ChatGPT to create compelling, platform-optimized descriptions with appropriate hashtags and calls-to-action for Instagram, TikTok, and YouTube.

3. Airtable Record Creation

Before publishing, the system creates a comprehensive record in Airtable containing the video name, generated descriptions, planned publication platforms, and initial status. This serves as your central tracking dashboard.

4. Multi-Platform Publishing

Using the Upload-Post.com API, the workflow simultaneously uploads the video to your connected Instagram, TikTok, and YouTube accounts. Each platform receives the video file and its customized AI-generated description.

5. Status Tracking & Error Handling

The Airtable record is updated with publication status, platform-specific URLs, and any error messages. If any platform fails to publish, Telegram notifications alert you immediately while successful uploads are logged for future reference.

Pro tip: Use separate Airtable views for "Pending," "Published," and "Failed" videos to quickly monitor your content pipeline without checking each platform individually.

What You'll Need

  1. Google Drive account with a dedicated folder for videos
  2. OpenAI API key for audio transcription and description generation
  3. Airtable base with tables for video tracking
  4. Upload-Post.com account (free trial available) for multi-platform publishing
  5. Connected social accounts on Instagram, TikTok, and YouTube
  6. n8n instance (cloud or self-hosted) to run the automation

Quick Setup Guide

1. Download the template and import it into your n8n instance. 2. Configure the Google Drive trigger with your folder ID and OAuth credentials. 3. Add your OpenAI API key to the transcription and chat nodes. 4. Set up your Airtable base with the required fields and add your API token. 5. Connect your Upload-Post.com account and add your API token to the HTTP request nodes. 6. Test with a sample video to verify all connections work properly.

Pro tip: Start with a test folder in Google Drive and dummy social media accounts to verify the workflow works correctly before connecting your main accounts.

What This Workflow Does

This automation solves a common communication challenge: explaining the same complex topic to different audiences with varying levels of understanding. Traditionally, educators, content creators, and businesses would manually rewrite explanations for children, teenagers, graduates, researchers, and executives—a time-consuming process that often leads to inconsistencies.

The workflow transforms any question into five distinct explanations simultaneously. It generates kid-friendly stories for 5-year-olds, relatable content for teenagers, professional explanations for graduates, academic analysis for PhD researchers, and strategic insights for business executives. All five versions are delivered in under 10 seconds via Telegram and automatically archived to Google Docs.

This eliminates hours of manual rewriting while ensuring each audience receives information tailored to their comprehension level. It's particularly valuable for educational institutions creating multi-grade curriculum content, content marketers producing material for diverse demographics, and businesses explaining technical concepts to various stakeholder groups.

How It Works

1. Question Input via Telegram

Users send any question through a Telegram bot. The workflow captures this input and prepares it for multi-level processing.

2. Parallel AI Processing

Five AI agents process the question simultaneously, each configured with specific prompts for different audiences. This parallel architecture ensures 3–8 second response times.

3. Audience-specific Generation

Each agent generates a tailored explanation: simple stories with analogies for children, relatable examples for teens, professional terminology for graduates, academic depth for researchers, and strategic implications for executives.

4. Formatting and Merging

The workflow formats each explanation with appropriate headers and character limits, then merges them using a binary tree architecture for reliable data handling.

5. Delivery and Archiving

Six formatted Telegram messages (header plus five explanations) are delivered instantly. Simultaneously, the complete conversation is saved to a Google Doc for permanent archiving.

What You'll Need

  1. Telegram Bot API token (free, created via @BotFather)
  2. OpenAI API key (pay-per-use, required for GPT-4.1-mini processing)
  3. Google account with Docs access (free, for automatic archiving)
  4. n8n instance (cloud or self-hosted, to run the workflow)
  5. Basic understanding of audience personas (to customize prompts if needed)

Pro tip: Before activating the workflow, clearly define your five audience personas. Knowing exactly who you're targeting (e.g., "5-year-olds who love animals," "teenagers interested in technology") helps the AI generate more relevant and engaging explanations.

Quick Setup Guide

  1. Create a Telegram bot via @BotFather and add the API token to n8n credentials.
  2. Obtain an OpenAI API key and add it to n8n credentials.
  3. Connect your Google account to grant Docs access within n8n.
  4. Create a blank Google Doc and paste its URL into the workflow's Google Docs node.
  5. Import the downloaded JSON template into your n8n instance.
  6. Activate the workflow and test with any question via your Telegram bot.

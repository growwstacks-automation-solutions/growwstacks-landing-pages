What This Workflow Does

This automation solves the time-consuming challenge of creating regular visual content for social media, blogs, and marketing materials. Instead of manually designing images or searching stock photos, you can generate custom AI visuals on-demand through a simple Telegram bot interface.

The workflow connects Telegram commands to advanced AI image generation (Nano Banana PRO), then optionally publishes the results directly to your social media accounts. It transforms what would normally be a multi-hour creative process into a few clicks from your phone, enabling consistent visual content production without designer dependency.

Beyond basic generation, the system can enhance prompts, compress images for web delivery, generate platform-specific captions, and distribute content across multiple channels simultaneously—creating a complete content pipeline from idea to publication.

How It Works

1. Telegram Command Initiation

Users send a simple command like /text_to_image to your Telegram bot. The bot responds with a form to collect parameters: prompt, style, aspect ratio, and quality preferences. This mobile-friendly interface makes AI image generation accessible to anyone on your team.

2. AI Prompt Enhancement & Generation

If enabled, an AI agent refines the user's prompt for better results before sending it to the Nano Banana PRO image model. The workflow submits the generation request and monitors the task completion, handling API queuing and retrieval automatically.

3. Image Processing & Delivery

Once generated, the image downloads to your server. Optional compression via TinyPNG reduces file size for faster social media uploads. The final image delivers back to the user via Telegram for preview and approval before any publishing occurs.

4. Social Media Publishing Automation

If publishing is enabled, another AI agent creates platform-optimized captions and hashtags. The workflow then posts the image and caption to configured social platforms (Facebook, Instagram, X) via Blotato integration, scheduling for optimal engagement times.

What You'll Need

  1. Telegram Bot Token – Created via @BotFather for user interaction
  2. Kie.ai API Access – For Nano Banana PRO image generation capabilities
  3. Cloudinary Account – For hosting uploaded images and generated content
  4. OpenAI API Key – Optional for prompt enhancement and caption generation
  5. Social Media Platform Access – API credentials for Facebook, Instagram, or X if auto-publishing
  6. n8n Instance – Self-hosted or cloud version to run the workflow

Pro tip: Start with text-to-image generation only, then gradually add social publishing once you're comfortable with the image quality and workflow reliability. Test with a private Telegram group before making the bot public.

Quick Setup Guide

  1. Import the template – Download the JSON file and import it into your n8n instance via the workflow import function.
  2. Configure credentials – Add your Telegram bot token, Kie.ai API key, and Cloudinary credentials in n8n's credential management.
  3. Test basic generation – Activate the workflow and send /text_to_image to your Telegram bot with a simple prompt.
  4. Customize parameters – Adjust image dimensions, quality settings, and style preferences in the workflow nodes to match your brand needs.
  5. Add social publishing – Once satisfied with image generation, configure Blotato nodes with your social media API credentials.
  6. Set permissions & security – Restrict Telegram bot access to authorized users and implement approval workflows if needed.

What This Workflow Does

Manual sales prospecting is slow and inconsistent. Researching each company, understanding their value proposition, and crafting a personalized outreach message can take 15–20 minutes per lead. This n8n workflow automates that entire process.

It takes a simple Google Sheet containing a list of prospect website URLs. For each URL, it automatically scrapes the website content, sends the cleaned text to OpenAI's GPT-4o with a specific prompt to generate a consultative sales pitch, and then writes that personalized message back into the same row of your spreadsheet. The workflow is "resume-safe," meaning it skips rows where a pitch has already been generated, allowing you to run it on a schedule or add new URLs over time.

The result is a scalable system for generating high-quality, context-aware outreach at the click of a button, turning hours of manual work into a fully automated, repeatable process.

How It Works

The automation follows a logical, step-by-step process to ensure accuracy and efficiency.

1. Read from Google Sheets

The workflow starts by connecting to your Google Sheet via the Google Sheets node. It reads all rows from a specified worksheet that has at least two columns: "Website" (the prospect's URL) and "Personalized Message" (initially empty). This setup is your central command center for leads.

2. Filter & Prepare Data

A filter node checks the "Personalized Message" column. If it's already populated, the row is skipped. This prevents re-processing the same lead and wasting API credits, allowing you to run the workflow repeatedly as you add new URLs to your list.

3. Scrape Website Content

For each new URL, the workflow uses a web scraping node (like the HTTP Request node or a dedicated service like Firecrawl) to fetch the main page content. The goal is to extract clean, structured text or markdown that represents the company's services, blog posts, or "About Us" information—the raw material for personalization.

4. Generate AI Pitch with GPT-4o

The scraped content is sent to the OpenAI node configured with the GPT-4o model. A carefully crafted system prompt instructs the AI to act as a sales consultant, analyzing the provided website text and generating a concise, value-driven outreach message that references specific details from the site.

5. Write Back to Google Sheets

The final step uses the Google Sheets node again, this time in "Update" mode. It matches on the "Website" URL column and writes the newly generated AI pitch into the "Personalized Message" column of the corresponding row. Your spreadsheet is now instantly populated with ready-to-use outreach copy.

Pro tip: Add a "Wait" node between processing rows to respect rate limits of the scraping service and OpenAI API, ensuring stable and reliable execution, especially for large lists.

What You'll Need

  1. A Google Sheets document with columns for "Website" and "Personalized Message."
  2. n8n instance (cloud or self-hosted).
  3. Google Service Account credentials or OAuth 2.0 setup for the Google Sheets node.
  4. OpenAI API key with access to the GPT-4o model.
  5. Optional: API key for a dedicated web scraping service (like Firecrawl, ScrapingBee, or ScrapeNinja) for more reliable content extraction than a simple HTTP request.

Quick Setup Guide

Follow these steps to get your automated sales pitch generator running in under 15 minutes.

  1. Download & Import: Click the "Download Template" button above and import the JSON file into your n8n instance.
  2. Configure Google Sheets: In the "Google Sheets" nodes, set up your credentials and specify the ID of your spreadsheet and the worksheet name.
  3. Set Up OpenAI: Add your OpenAI API key in the "OpenAI" node and ensure the model is set to "gpt-4o" or "gpt-4o-mini" for cost efficiency.
  4. Adjust the AI Prompt: Review and customize the system and user prompts in the OpenAI node to match your industry, tone, and call-to-action.
  5. Test with One URL: Run the workflow once in "Manual Execution" mode with a single website URL in your sheet to verify the output.
  6. Schedule or Trigger: Set the workflow to run on a schedule (e.g., daily) or trigger it via a webhook whenever you add new leads to your sheet.

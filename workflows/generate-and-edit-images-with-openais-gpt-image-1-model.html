What This Workflow Does

This automation solves the time-consuming and costly process of creating custom visuals for business needs. Instead of hiring designers or spending hours in complex software, you describe what you want in plain English, and AI generates professional images instantly. The workflow goes further by allowing you to edit those images—adding elements, changing styles, or refining details—all through automated steps.

Perfect for marketing teams needing campaign visuals, e-commerce stores requiring product mockups, content creators building blog illustrations, or any business that regularly uses custom graphics. It transforms creative bottlenecks into scalable, on-demand visual production.

How It Works

The workflow connects OpenAI's advanced image models with your business processes through a logical sequence of steps.

Step 1: Trigger with a Prompt

Start the workflow manually or via a webhook, providing a text description of the image you want. For example, "A modern workspace with natural light, plants, and a laptop on a wooden desk."

Step 2: Generate the Base Image

The workflow sends your prompt to OpenAI's GPT-Image-1 model, which creates a high-quality image and returns it in base64 format. This happens in seconds, with options to specify style, dimensions, and quality.

Step 3: Convert for Editing

Since OpenAI's editing endpoint requires binary files, the workflow converts the base64 image into a proper PNG file format while preserving quality and metadata.

Step 4: Apply Edits

Provide a second prompt for edits, like "Add a coffee cup and notebook beside the laptop." The workflow sends the binary image back to OpenAI with your edit instructions, generating a refined version.

Step 5: Deliver the Final Image

The edited image is converted to a downloadable file format and can be saved to cloud storage, sent via email, posted to social media, or integrated directly into your design tools.

Quick Setup Guide

Get this automation running in under 15 minutes with these simple steps:

  1. Download the Template: Click the download button above to get the JSON file.
  2. Import to n8n: In your n8n instance, go to Workflows → Import from file and select the downloaded JSON.
  3. Configure OpenAI Credentials: Add your OpenAI API key in the credential settings for both the "Create Image" and "Edit Image" nodes.
  4. Test with Sample Prompts: Run the workflow manually with the example prompts provided in the template to verify everything works.
  5. Customize for Your Needs: Modify the prompt nodes with your specific image descriptions and edit instructions.
  6. Add Output Actions: Connect the final image output to storage nodes (Google Drive, Dropbox) or communication nodes (Email, Slack) as needed.
  7. Schedule or Trigger: Set the workflow to run on a schedule, via webhook, or integrate it with other business processes.

Pro tip: Create a library of reusable prompt templates for common image types (social posts, product shots, blog headers) to ensure brand consistency and save time on prompt engineering.

Automate SEO Blog Posts: AI Content to Blogger

This n8n workflow automates the entire pipeline of generating, formatting, and publishing SEO-rich blog posts to a Blogger site—ideal for auto service businesses, marketers, and agencies.

What This Workflow Does

This automation solves a common yet time-consuming problem for businesses: maintaining a consistent, SEO-optimized blog. Manually researching topics, writing articles, finding images, and publishing is a multi-hour weekly task. This workflow eliminates that manual effort by creating a fully automated content pipeline.

It starts by fetching the latest trending news in your chosen category (like technology or auto services) from Mediastack. It then uses OpenRouter AI to generate a complete, SEO-structured blog post—including a compelling title, meta description, headers, and FAQ section. Relevant images are sourced from Pexels, and the formatted HTML is published directly to your Blogger site. Finally, a confirmation is sent via Telegram.

The result is a hands-off content engine that keeps your blog fresh, improves search rankings, and drives organic traffic without you lifting a finger after setup.

How It Works

The workflow is a sequenced chain of intelligent steps that transforms raw data into a published article.

Step 1: Scheduled Trigger & News Fetch

A Schedule Trigger node kicks off the process daily or weekly. It activates an HTTP Request node that calls the Mediastack API, pulling in the latest news articles based on your configured keywords and category (e.g., "technology", "automotive").

Step 2: AI-Powered Content Generation

The core news article data is passed to an AI Agent node (powered by OpenRouter). This node uses a carefully crafted prompt to instruct the AI to write a unique, informative blog post. The prompt includes instructions for SEO structure, tone, keyword inclusion, and specific sections to generate.

Step 3: Image Sourcing & Enhancement

Concurrently, a Pexels node searches for high-quality, royalty-free images related to the article's topic. These images are downloaded and prepared for inclusion in the blog post, adding visual appeal and improving reader engagement.

Step 4: HTML Formatting & Assembly

A Code node or a series of Function nodes takes the AI-generated text and the image URLs, assembling them into clean, Blogger-compatible HTML. This includes proper title tags (H1, H2, H3), image tags with alt text, paragraph formatting, and internal linking structure.

Step 5: Authentication & Publishing

The workflow uses OAuth2 to securely authenticate with the Google Blogger API. The formatted HTML, along with the post title, labels (tags), and other metadata, is sent via an HTTP Request node to create and instantly publish a new post on your designated Blogger blog.

Step 6: Notification & Logging

Finally, a Telegram node sends a message to a specified channel or chat with a link to the new post and a summary. This serves as both a confirmation and a way to share the new content with your team or audience immediately.

What You'll Need

  1. An n8n instance: Either n8n.cloud account or a self-hosted setup.
  2. Mediastack API Key: A free or paid plan from mediastack.com to fetch news data.
  3. OpenRouter API Key: An account and API key from openrouter.ai to power the AI content generation.
  4. Pexels API Key: A free API key from pexels.com for sourcing images.
  5. A Google/Blogger Account: With a blog already created on Blogger.com.
  6. Google OAuth2 Credentials: You'll need to set up a project in the Google Cloud Console to enable the Blogger API and generate OAuth2 credentials for n8n.
  7. Telegram Bot Token: (Optional) Create a bot via BotFather and get its token to enable notifications.

Pro tip: Start with the free tiers for Mediastack, OpenRouter, and Pexels to test the workflow. The Google OAuth setup is the most technical step—follow n8n's detailed documentation for the Blogger node.

Quick Setup Guide

Follow these steps to get your automated blog pipeline running in under 30 minutes.

  1. Download & Import: Click the "Download Template" button above and save the JSON file. In your n8n instance, go to "Workflows" > "Import from File" and select the downloaded JSON.
  2. Configure Credentials: In the workflow canvas, click on each node that requires an API key (Mediastack, OpenRouter, Pexels, Google OAuth). Use the "Credentials" modal to add your respective keys. For Google, you'll need to complete the OAuth2 flow.
  3. Customize the Trigger: Open the "Schedule Trigger" node. Set your desired frequency (e.g., every day at 9 AM). Adjust the Mediastack HTTP Request node to change the news category or keywords.
  4. Tweak the AI Prompt: Locate the AI Agent (OpenRouter) node. Review and edit the system and user prompts to match your desired article tone, length, and structure. This is where you define your blog's voice.
  5. Set Your Blog ID: In the Blogger HTTP Request node, find the field for your blog's ID. You can get this from your Blogger dashboard URL.
  6. Test in Dry-Run Mode: Before going live, activate the workflow with "Execute Once" on a manual trigger node. Check each step's output to ensure news is fetched, content is generated, and the API calls are formatted correctly.
  7. Activate & Monitor: Once testing is successful, activate the workflow. Monitor the first few runs via the "Executions" tab to ensure posts are publishing correctly and notifications are sent.

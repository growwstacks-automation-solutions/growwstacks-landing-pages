Generate ASMR Rainforest Videos from Text with Seedream & Seedance

Transform simple text prompts into captivating, animated ASMR-style rainforest videos automatically. Perfect for content creators, marketers, and wellness brands.

What This Workflow Does

This automation solves a common challenge for content creators and marketers: producing high-quality, engaging visual content consistently without extensive production time or resources. The workflow takes simple text descriptions and transforms them into complete ASMR-style rainforest videos through a seamless two-step AI generation process.

First, it uses Seedream v4 to create a detailed, atmospheric image based on your prompt. Then, it automatically passes that image to Seedance v1 pro, which adds subtle, relaxing motion to bring the scene to life. The result is a complete video asset ready for social media, websites, or digital campaigns—all generated in minutes rather than hours.

How It Works

The workflow follows a logical sequence that mimics a professional video production pipeline, but completely automated.

Step 1: Text Prompt Processing

You provide a descriptive text prompt (like "misty rainforest waterfall at dawn with gentle rain sounds"). The workflow structures this prompt for optimal AI interpretation, ensuring the generated image matches your creative vision.

Step 2: Image Generation with Seedream v4

The formatted prompt is sent to fal.ai's Seedream v4 model, which creates a high-resolution, detailed image. The workflow includes error handling and retry logic to manage API rate limits or temporary service issues.

Step 3: Video Animation with Seedance v1 pro

Once the image is generated and validated, it's automatically passed to Seedance v1 pro. This specialized model analyzes the image composition and applies appropriate ASMR-style motion—like gentle water flow, leaf movement, or atmospheric particle effects.

Step 4: Output and Notification

The completed video is saved with metadata, and you receive a notification with download links. The workflow can be extended to automatically upload to cloud storage or social media platforms.

Pro tip: For consistent branding, create a library of standardized prompts that reflect your visual style. You can then rotate through these prompts to generate varied but thematically consistent content.

Quick Setup Guide

Getting started with this template takes about 15 minutes if you have your API credentials ready:

  1. Download and Import: Download the JSON file above and import it into your n8n instance through the workflow import interface.
  2. Configure fal.ai Credentials: Create a new credential in n8n for fal.ai using your API authorization header.
  3. Set Your Prompt: Edit the "Prompt" node with your desired text description. You can test with something simple like "peaceful rainforest stream" initially.
  4. Adjust Parameters (Optional): Review the Seedream and Seedance node settings. The default parameters work well for most ASMR content, but you can adjust resolution, style intensity, or animation duration.
  5. Test Run: Execute the workflow once to verify everything connects properly and you receive a video output.
  6. Add Extensions: Once basic generation works, consider adding nodes to automatically post to social media, log results to a spreadsheet, or trigger based on a schedule.

Pro tip: Start with shorter animation durations (3-5 seconds) for social media content, as these platforms often favor shorter, looping videos that capture attention quickly.

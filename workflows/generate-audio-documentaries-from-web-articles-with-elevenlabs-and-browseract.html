What This Workflow Does

This automation solves a critical content accessibility problem: turning valuable written articles into engaging audio content that people can consume on the go. Instead of manually recording podcasts or using basic text-to-speech that sounds robotic, this workflow creates high-production-value audio documentaries automatically.

The system takes any web article URL, scrapes the content, transforms it into a narrative script optimized for listening, generates human-like voiceovers using ElevenLabs, and delivers the finished MP3 directly to Telegram. It's like having a personal podcast production team that works 24/7, turning your blog content into a passive audio content stream.

How It Works

1. Content Detection & Classification

When a message arrives via Telegram, an AI agent analyzes whether it's casual chat or contains a valid article URL. This intelligent filtering ensures the workflow only processes actual content requests.

2. Web Content Extraction

BrowserAct visits the webpage in a background browser session, extracting clean text while handling JavaScript-rendered content, paywalls, and complex layouts that simple scrapers can't process.

3. AI Scriptwriting Transformation

Using Claude via OpenRouter, the dry article text gets rewritten into a dramatic narrative script. The AI adds pacing cues, tone suggestions, and structural elements that make for compelling audio storytelling.

4. Professional Voice Synthesis

ElevenLabs converts the script into studio-quality speech using voice models like "Liam" that include natural inflection, emotional variation, and proper breathing patterns indistinguishable from human narration.

5. Delivery & Distribution

The final MP3 file and formatted summary are sent back to the user on Telegram, creating a seamless end-to-end experience from article discovery to audio consumption.

What You'll Need

  1. BrowserAct account with the AI Summarization & Eleven Labs Podcast Generation template saved
  2. ElevenLabs account for high-quality voice synthesis
  3. OpenRouter account or access to an LLM like Claude for scriptwriting
  4. Google Gemini account for initial content classification
  5. Telegram Bot Token for receiving requests and delivering results
  6. n8n instance (cloud or self-hosted) to run the automation

Quick Setup Guide

  1. Import the template into your n8n instance using the downloaded JSON file
  2. Configure credentials for Telegram, ElevenLabs, OpenRouter, Google Gemini, and BrowserAct in n8n's credentials management
  3. Set up BrowserAct by ensuring the required template is saved in your BrowserAct account and note your API key and workflow ID
  4. Select your voice model in the ElevenLabs "Convert text to speech" node according to your brand voice preferences
  5. Test the workflow by sending a article URL to your Telegram bot and verifying the audio output quality
  6. Activate the workflow and start transforming articles into audio documentaries automatically

Pro tip: For consistent branding, create a custom voice clone in ElevenLabs using your own recordings. This makes all your audio content instantly recognizable as part of your brand ecosystem.

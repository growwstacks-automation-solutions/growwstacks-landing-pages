What This Workflow Does

This automation solves the time-consuming and costly process of manually creating visual content for marketing, social media, or creative projects. It connects cutting-edge AI tools into a seamless pipeline: starting with a text prompt, it generates a unique image, ensures that image is publicly available online, and then transforms it into an engaging video—all without manual intervention.

The business value is immense. It turns a creative task that could take hours (concept, design, editing) into a process that runs in minutes on autopilot. This allows small teams, indie creators, and marketers to produce a high volume of professional, unique visual assets at a fraction of the traditional cost, enabling consistent content calendars and rapid prototyping of visual ideas.

How It Works

The workflow follows a logical, three-stage process, each handled by a specialized AI service.

1. Trigger & Image Generation

The workflow can be started manually or on a schedule. It then sends your text prompt to a Flux AI image generation API. You have a choice between using Flux's direct API or a version available via RapidAPI, allowing for flexibility in pricing and access.

2. Image Hosting with Kraken.io

The generated image is received as data. However, the next step (Runway ML) requires a public URL to access the image. This workflow automatically uploads the image to Kraken.io, a dedicated image hosting and optimization service, which returns a secure, public link. This step is crucial for the integration to function correctly.

3. Video Generation with Runway ML

Using the public image URL from Kraken, the workflow calls the Runway ML API. Similar to the first step, you can choose between Runway's official API or a RapidAPI endpoint. Runway's AI analyzes the image and generates a short video based on it, applying motion and style as defined in your settings. The final output is a direct URL to your newly created AI video.

Pro tip: Pair this workflow with a Google Sheet containing a list of prompts. Use n8n's "Read from Sheets" node to loop through them, enabling bulk generation of an entire content library in one automated run.

What You'll Need

  1. An n8n instance (cloud or self-hosted).
  2. API keys for at least one service: Flux (from BlackForest Labs or via RapidAPI) or Runway ML (official or via RapidAPI).
  3. An account and API credentials for Kraken.io (required for hosting).
  4. A clear idea of the image prompts and video styles you want to automate.

Quick Setup Guide

Import and configure this workflow in under 15 minutes.

  1. Download & Import: Click the download button above and import the JSON file into your n8n workspace.
  2. Configure Flux: Open the "Flux" HTTP Request node. Paste your API key (from docs.bfl.ml or RapidAPI) into the authorization header or body as required.
  3. Set Up Kraken.io: In the "Kraken Upload" node, enter your Kraken.io API Key and Secret from your account dashboard.
  4. Configure Runway ML: In the "Runway" HTTP Request node, add your Bearer token (from dev.runwayml.com) or your RapidAPI key to the headers.
  5. Test & Activate: Click "Execute Workflow" to test. Once successful, activate the workflow and optionally add a Schedule Trigger for automatic daily or weekly runs.

Telegram AI Content Generator

A free n8n workflow template that turns your Telegram bot into an on-demand AI content studio, generating images and blog articles instantly.

What This Workflow Does

This automation solves the time-consuming and costly challenge of creating visual and written content from scratch. Instead of hiring designers and writers or spending hours on manual creation, this workflow turns a simple Telegram message into a complete content asset.

The system listens for user commands in a Telegram chat. When someone sends "image [your prompt]" it generates a unique AI image using the Pollinations API. When they send "blog [your title]" it writes a structured, multi-paragraph article using Google's Gemini AI. Everything is logged to Google Sheets and images are saved to Google Drive automatically.

It essentially creates a 24/7 content studio that your team, clients, or community can access from their phones, dramatically reducing turnaround time from days to seconds.

How It Works

1. Telegram Interaction & Command Parsing

The workflow is triggered by any new message in your Telegram bot. It immediately classifies the input—checking if it's a /start command, a help request, a callback from a button, or a content generation command starting with "image" or "blog".

2. Menu Navigation & User Guidance

If users need help or select menu options, the bot responds with clear instructions and buttons for "Generate Image" or "Blog Article." This intuitive interface makes the automation accessible to non-technical users.

3. AI Image Generation Pipeline

For image requests, the system extracts the prompt, shows a "typing" status in Telegram, then constructs a request to the Pollinations API. The generated image is downloaded, sent back to the user, simultaneously logged to a Google Sheet with metadata, and uploaded to a designated Google Drive folder for archival.

4. AI Article Writing Pipeline

For blog requests, the workflow captures the title, logs it, then presents style options (Formal, Casual, News). Once selected, it uses the LangChain node to send a structured prompt to Google Gemini, requesting a three-paragraph article. The AI's response is parsed and formatted before being delivered to the user and logged with a completion timestamp.

What You'll Need

  1. A Telegram bot token (created via @BotFather).
  2. Access to the Pollinations API (free public endpoint).
  3. A Google Cloud project with Sheets and Drive API enabled, and OAuth credentials.
  4. A Google Gemini API key (via Google AI Studio).
  5. An n8n instance (self-hosted or cloud).
  6. A Google Sheet for logging and a Google Drive folder for image storage.

Pro tip: Before setting up, create a dedicated Google Sheet with columns for: Timestamp, User ID, Command Type, Prompt, Image URL/Article Text, Style, and Status. This makes tracking and analyzing usage much easier later.

Quick Setup Guide

  1. Import the template: Download the JSON file and import it into your n8n workspace.
  2. Configure credentials: Set up n8n credentials for Telegram API, Google Sheets OAuth, Google Drive OAuth, and Google Gemini (via LangChain node).
  3. Update resource IDs: Replace the placeholder Google Sheet ID and Google Drive Folder ID with your own.
  4. Test the bot: Activate the workflow and send /start to your Telegram bot. Try "image a futuristic cityscape" and "blog The Future of Remote Work".
  5. Deploy: Once testing is successful, set the workflow to "Active" for continuous operation.

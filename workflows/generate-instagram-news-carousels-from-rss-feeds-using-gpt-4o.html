What This Workflow Does

This workflow acts as a fully autonomous “Viral News Agency” inside your n8n instance. It solves the time‑consuming problem of manually creating Instagram content by automating the entire pipeline: monitoring RSS feeds for trending stories, using GPT‑4o to rewrite them into engaging scripts, generating professional 10‑slide carousels, and publishing directly to your Instagram Business account.

Unlike basic auto‑posters, this system includes intelligent content analysis, dual‑engine image generation (free Gotenberg or paid APITemplate.io), and handles the complex Meta API flow for carousel uploads. It ensures your Instagram feed stays fresh with high‑quality, educational content 24/7, without any manual intervention.

For businesses, agencies, or creators, this automation reclaims 10–20 hours per week spent on content creation, while boosting engagement through consistent, timely posts that leverage viral news angles.

How It Works

1. RSS Monitoring & Cleaning

The workflow starts by polling your configured RSS feeds (tech, sports, politics, etc.). It filters for new articles, extracts clean text, and pulls high‑quality Open Graph images to serve as dynamic backgrounds for the carousel slides.

2. AI Content Rewriting

Each article is sent to GPT‑4o with a specialized prompt that transforms raw news into a “hot take” or educational script. The AI generates a 10‑slide structure with titles, key points, quotes, and branded hashtags—tailored to your audience.

3. Carousel Slide Generation

The script is split into individual slides. Using your chosen image engine (Gotenberg for free Docker‑based generation or APITemplate.io for premium designs), the workflow creates visually consistent slides with overlayed text, maintaining brand aesthetics.

4. Instagram Publishing

The workflow uploads each image to Facebook’s servers, creates a Carousel Container, bundles the media, and publishes the post to your Instagram Business account via the Graph API. It includes error handling and retry logic for reliability.

Quick Setup Guide

Estimated setup time: 10 minutes

  1. Import the template into your n8n workspace using the downloaded JSON file.
  2. Add credentials for OpenAI, Google Drive (for temporary storage), and Facebook Graph API.
  3. Update Instagram ID in the three Facebook nodes (“Create Container”, “Carousel Bundle”, “Publish Carousel”) with your Instagram Business User ID.
  4. Choose image engine: if using Gotenberg, run docker run --rm -p 3000:3000 gotenberg/gotenberg:8. For APITemplate.io, add your API key and Template ID in the “Generate Image” node.
  5. Configure the Setup Form node with your RSS URL and brand name, then toggle the workflow to “Active”.

Pro tip: Test with a single RSS feed first. Use the “Wait” node to control posting frequency—avoid flooding your audience with too many carousels per day.

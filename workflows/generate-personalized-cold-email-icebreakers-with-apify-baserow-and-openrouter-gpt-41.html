What This Workflow Does

Generic cold emails get ignored. Personalized outreach gets replies—but manually researching every lead is impossible at scale. This workflow solves that by fully automating the research and personalization process.

It takes a list of leads from your database, visits each company’s website, intelligently scrapes and summarizes their content, and uses AI to generate a unique subject line and opening line tailored to that specific business. If a website is broken, it flags the lead so you don’t waste time. The result is outreach that feels handcrafted, delivered at the speed of automation.

How It Works

The automation follows a logical, multi-step process to ensure quality and efficiency.

1. Fetch Leads from Database

The workflow starts by pulling a batch of leads from your connected database (Baserow in this template). Each record must contain at least the company name and website URL.

2. Scrape and Analyze Website Content

Using Apify, the lead’s website is fetched. The workflow intelligently extracts internal links and scrapes up to five key pages (home, about, services) to build a comprehensive understanding of the business.

3. Convert and Condense for AI

All scraped HTML is converted to clean markdown and trimmed to a maximum of 5,000 characters per page. This controls token usage and cost while preserving the essential information.

4. Generate Company Overview

The aggregated content is sent to OpenRouter GPT-4.1 (or your chosen LLM), which analyzes the text and produces a concise, accurate summary of the company’s core offering and industry.

5. Create Personalized Icebreaker

A second AI call uses the company summary, lead name, and your predefined messaging framework to generate a compelling email subject line and a personalized opening paragraph that references the prospect’s actual business.

6. Enrich and Update Records

The generated overview, subject line, and icebreaker are written back to the database, enriching each lead record. The workflow is now ready for the next step in your sequence, whether that’s sending the email directly or moving it to a review queue.

Pro tip: Use this workflow as an enrichment step before emails are sent. Add a manual approval node to review AI-generated content for top-tier leads, maintaining a human touch where it matters most.

  3. Baserow, Airtable, or Google Sheets credentials with a leads table.
  4. OpenRouter API key (or access to another LLM like OpenAI, Anthropic).
  5. A basic understanding of how to import a JSON workflow and set up credentials in n8n.

Quick Setup Guide

You can have this automation running in under 15 minutes.

  1. Download the template using the button above.
  2. Import into n8n via the “Import from File” option in your n8n dashboard.
  3. Set up credentials in n8n’s credential manager for Apify, your database, and OpenRouter.
  4. Map your fields by adjusting the Baserow node to match your table’s column names for “Company Name” and “Website URL”.
  5. Test with a single lead by using the “Execute Once” feature. Verify the output in your database.
  6. Activate the workflow and set it to run on a schedule (e.g., every hour) to process new leads automatically.

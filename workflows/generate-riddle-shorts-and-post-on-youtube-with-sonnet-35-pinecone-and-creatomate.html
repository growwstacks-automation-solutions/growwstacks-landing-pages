What This Workflow Does

Creating consistent, engaging short-form video content for YouTube is time-consuming and repetitive. This automation solves that by building a complete pipeline that generates original riddle content, checks for duplicates, renders professional vertical videos, and publishes them directly to your YouTube channel—all without manual intervention.

The system leverages AI (Claude Sonnet 3.5) for creative generation, Pinecone for intelligent duplicate prevention, and Creatomate for high-quality video rendering. It's designed for content creators, educators, and marketers who want to maintain an active YouTube Shorts presence while reclaiming hours each week for strategic work.

How It Works

1. Riddle Generation & Duplicate Check

The workflow begins by prompting an AI agent to create four new riddles with answers. Before accepting them, it converts the text into vector embeddings and queries Pinecone—a vector database—to ensure similar riddles haven't been published recently. This maintains content freshness.

2. Content Organization & Storage

Approved riddles are stored in Pinecone under a namespace organized by month/year, creating an easily searchable archive. The system also pulls the next sequential title (e.g., "Riddles4U #042") from a Google Sheet, keeping your content numbering consistent.

3. Media Asset Selection

A Google Sheet tracks a library of royalty-free audio tracks. The workflow selects the first unused track URL, ensuring variety in your videos' background music. Optionally, you can integrate a similar system for background video clips.

4. Video Rendering

The riddle text, answer, and audio track URL are sent to Creatomate via its API. Creatomate merges these elements with a pre-designed 9:16 video template, producing a polished, platform-ready MP4 file optimized for YouTube Shorts.

5. YouTube Upload & Metadata

Using YouTube's OAuth authentication, the workflow uploads the video binary via a multipart/form-data HTTP request. It sets the title, description, and category, then retrieves the final YouTube URL upon successful publication.

6. Post-Publication Updates

Finally, the system marks the audio track as "used" in the sheet, increments the next riddle title number, and can send a notification email with the video link. If all audio tracks are used, it can reset them, creating a sustainable content loop.

What You'll Need

  2. YouTube Channel with API access enabled and OAuth 2.0 credentials configured.
  3. AI API Access (Anthropic Claude, OpenAI GPT, or similar) for riddle generation.
  4. Pinecone account with an index created to store and query vector embeddings.
  5. Creatomate account with a 9:16 video template configured for API rendering.
  6. Google Sheets document for tracking audio tracks and riddle numbering.

Pro tip: Start with a single, high-quality background video loop in your Creatomate template. YouTube's algorithm often favors consistency in visual style for Shorts. You can rotate the background video monthly based on performance data.

Quick Setup Guide

  1. Download the template using the button above and import it into your n8n workspace.
  2. Configure credentials in n8n for YouTube OAuth, your AI provider, Pinecone, and Google Sheets.
  3. Set up your Creatomate template by creating a 9:16 project, designing your layout, and copying the Template ID and API Key into the respective nodes.
  4. Prepare your Google Sheet with columns for audio track URLs and usage status, and connect it via the Google Sheets node.
  5. Test the workflow in manual execution mode to verify each step—from riddle generation to YouTube upload—works correctly with your accounts.
  6. Schedule the workflow to run daily or weekly using n8n's Schedule Trigger node, setting your desired publishing frequency.

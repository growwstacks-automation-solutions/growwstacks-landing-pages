What This Workflow Does

This automation solves the time-intensive problem of daily market analysis and content creation. Instead of manually checking stocks, calculating indicators, and writing updates, this workflow runs automatically twice daily to generate professional investment insights.

It fetches historical data for selected tickers (like GAZP, SBER, LKOH), calculates key technical indicators including RSI, EMA/SMA, MACD, Bollinger Bands, and ADX, then uses AI to generate a compelling investment thesis. The results are stored for tracking and sent to Telegram for human moderation before publishing to your audience.

The system maintains human oversight through Telegram's inline buttons—allowing quick "Publish" or "Retry" decisions—ensuring quality control while automating 90% of the analytical work.

How It Works

1. Scheduled Data Collection

The workflow triggers twice daily via schedule node, fetching historical market data from broker APIs for your predefined watchlist of stocks.

2. Technical Analysis Calculation

Custom function nodes process the raw price data to calculate multiple technical indicators simultaneously, creating a comprehensive analytical foundation.

3. AI Insight Generation

An AI model (OpenAI/OpenRouter) analyzes the technical data and generates structured JSON output containing a title and detailed investment summary.

4. Database Storage

All analysis results are saved to PostgreSQL for historical tracking, performance analysis, and audit trails.

5. Telegram Moderation

The draft post is sent to a private Telegram channel with "Publish" and "Retry" buttons, enabling quick human approval before public distribution.

6. Final Publishing

When approved, the workflow publishes the polished insight to your public Telegram channel, reaching your audience with timely, vetted analysis.

Quick Setup Guide

Follow these steps to implement this automation in your n8n environment:

  1. Import the template using the downloaded JSON file into your n8n instance
  2. Configure data sources by updating the HTTP Request nodes with your market API credentials and endpoints
  3. Set up AI integration by adding your OpenAI/OpenRouter API key to the LLM node
  4. Connect your database by configuring the PostgreSQL node with your connection details
  5. Setup Telegram bots by creating two bots (one for moderation, one for publishing) and adding their tokens
  6. Customize ticker list by editing the workflow's initial configuration to include your specific stocks
  7. Test thoroughly with historical data before enabling the scheduled trigger

Pro tip: Run the workflow manually with a single ticker first to verify all connections work. Check the AI output format matches your expectations before scaling.

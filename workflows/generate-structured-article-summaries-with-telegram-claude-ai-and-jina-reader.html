What This Workflow Does

Information overload is a real productivity killer. Professionals spend hours each week reading articles, reports, and competitor analyses, often struggling to extract the key insights they need. This automation solves that problem by creating a personal research assistant available 24/7 through Telegram.

When you send any article URL to your Telegram bot, this workflow automatically fetches the clean content (stripping away ads and navigation), processes it through advanced AI models, and returns a structured summary with key points, takeaways, and actionable insights. It transforms hours of reading into seconds of consumption.

How It Works

The automation follows a precise sequence to ensure accurate, useful summaries every time.

1. URL Reception via Telegram

Users send article links directly to your Telegram bot. The workflow triggers instantly, capturing the URL and sender information for personalized responses.

2. Content Cleaning with Jina Reader

The URL is sent to Jina Reader, which extracts only the core article content while removing advertisements, navigation bars, pop-ups, and other distracting elements. This clean text ensures the AI focuses on what matters.

3. AI Analysis with Claude

The cleaned content is processed through Claude AI (or your preferred LLM) with a custom prompt designed to extract key insights, summarize arguments, identify important data points, and highlight actionable takeaways.

4. Structured Output Formatting

The AI's response is parsed into a consistent format including title, summary, key points, tags, and estimated reading time saved. This structure makes the information immediately useful.

5. Delivery Back to Telegram

The formatted summary is sent back to the user in Telegram with clear formatting, making it easy to read and share with team members.

Pro tip: Configure different AI prompts for different team members. Executives might want strategic implications highlighted, while technical teams need methodology details.

What You'll Need

  1. Telegram Bot Token: Create a free bot via @BotFather on Telegram (takes 2 minutes).
  2. AI API Access: An account with OpenRouter, Anthropic (Claude), OpenAI, or similar LLM provider.
  3. Optional Jina Reader API: For best results, a Jina AI account to clean web content effectively.
  4. n8n Instance: Self-hosted n8n or n8n.cloud account to run the workflow.
  5. Basic Configuration Skills: Ability to copy API keys into credential fields.

Quick Setup Guide

Get your personal article summarizer running in under 15 minutes with these steps.

  1. Import the Template: Download the JSON file and import it into your n8n instance via the workflow import function.
  2. Configure Telegram Credentials: Create your bot token via @BotFather and add it to the Telegram trigger node.
  3. Add AI API Keys: Insert your OpenRouter/Claude/OpenAI credentials in the LLM chain node.
  4. Optional Jina Setup: Add Jina Reader API key if using the content cleaning node for premium results.
  5. Test with a URL: Activate the workflow and send an article link to your Telegram bot to verify everything works.
  6. Customize the Prompt: Adjust the AI prompt in the LLM node to match your preferred summary style and depth.

Pro tip: Start with a private Telegram group where team members can share URLs and receive summaries collectively, fostering knowledge sharing.

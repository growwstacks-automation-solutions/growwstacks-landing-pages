What This Workflow Does

Social media managers, content creators, and marketing teams spend hours brainstorming captions for images. This manual process slows down content calendars and creates inconsistency. This n8n automation solves that by turning image storage into a content creation pipeline.

Whenever you upload an image to a specific Google Drive folder, the workflow automatically detects it, uploads it to Cloudinary for hosting, analyzes it with Azure OpenAI's GPT-4o-mini, and generates a complete Twitter/X content package. You receive everything in a formatted email—ready to copy, paste, and publish.

The system creates more than just a caption. It produces headlines, primary tweets, mini-threads with hooks and insights, alt text for accessibility, suggested replies, optimized hashtags, relevant @mentions, and even ideal posting times. This transforms a simple image upload into a complete social media asset.

How It Works

1. Image Detection & Download

The workflow starts with a Google Drive Trigger node monitoring a designated folder. When a new image file appears, it instantly activates. A Google Drive node then downloads the file for processing, ensuring the original remains safe in your Drive.

2. Image Hosting & Analysis Preparation

An HTTP Request node uploads the image to Cloudinary, creating a publicly accessible URL. This URL is crucial—it allows the AI to "see" and analyze the image content directly, rather than working with file data.

3. AI-Powered Content Generation

The core magic happens with LangChain and Azure OpenAI's GPT-4o-mini. The AI receives the image URL along with detailed instructions about your brand voice and content goals. It analyzes the visual elements, context, and potential audience appeal.

4. Structured Content Creation

The AI doesn't just write one caption. It follows a structured prompt to generate: a punchy headline (under 280 characters), a primary tweet, a 3-part mini-thread (hook → insight → action), descriptive alt text, suggested first-reply content, 10-12 relevant hashtags, recommended @mentions, and optimal posting times based on your timezone.

5. Delivery & Ready-to-Publish Formatting

Finally, an SMTP node sends a professionally formatted HTML email containing all generated content, the analyzed image thumbnail, a visual summary, and feature notes. This delivers a complete content package to your inbox, organized for immediate use.

Quick Setup Guide

Importing and configuring this workflow takes about 15-20 minutes if you have your API credentials ready.

  1. Download the template using the button above and import it into your n8n instance.
  2. Configure Google Drive nodes by connecting your account and specifying the folder ID to monitor.
  3. Set up Cloudinary credentials in the HTTP Request node with your cloud name, API key, and upload preset.
  4. Connect Azure OpenAI in the LangChain node with your endpoint, API key, and deployment name for GPT-4o-mini.
  5. Customize the AI prompt in the LangChain node—add your brand voice, content guidelines, and any specific hashtags or mentions.
  6. Configure SMTP email with your sender credentials and recipient email address for content delivery.
  7. Test the workflow by uploading an image to your designated Google Drive folder and checking your email for the generated content package.

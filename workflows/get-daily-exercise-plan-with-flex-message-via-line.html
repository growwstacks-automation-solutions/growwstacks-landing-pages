Automate Daily Yoga Reminders via LINE

Free n8n template that sends personalized yoga pose suggestions with images and links through LINE Flex Messages, powered by Google Sheets and AI.

What This Workflow Does

This automation solves the challenge of consistently engaging yoga students or fitness app users with fresh, personalized content. Manual daily messaging is time-consuming and difficult to scale, while generic bulk messages feel impersonal and see low engagement.

The workflow automatically sends curated daily yoga pose suggestions via LINE's rich Flex Messages at a scheduled time (9:30 PM Bangkok time). It pulls pose names, high-quality images, and educational links from a Google Sheets database, enhances the messaging with AI-generated motivational text, and delivers interactive carousels that users can click to learn more. Every interaction is logged back to Sheets for continuous improvement of recommendations.

How It Works

1. Scheduled Trigger & Data Retrieval

Each day at 21:30, the workflow activates automatically. It connects to your Google Sheets database and retrieves that day's curated yoga poses, including pose names, image URLs, and educational resource links.

2. AI Content Enhancement

Using Azure OpenAI (or your preferred AI service), the system transforms basic pose information into engaging, user-friendly messages. It adds appropriate emojis, motivational phrasing, and clear instructions tailored for mobile viewing.

3. LINE Flex Message Formatting

The workflow structures the content into LINE's specific JSON format for Flex Messages, creating interactive carousels that display pose images beautifully. It ensures proper formatting and validates the JSON before sending.

4. Message Delivery & Logging

LINE Push Messages deliver the final content directly to users' LINE apps. Simultaneously, the system logs which poses were sent, timestamps, and user engagement data back to Google Sheets, creating a feedback loop for optimizing future content.

Who This Is For

This template is ideal for yoga instructors building their digital community, fitness app developers needing automated engagement features, wellness coaches maintaining client consistency, and content creators distributing daily mindfulness content. It's particularly valuable for businesses targeting Asian markets where LINE dominates messaging, but the pattern works equally well for WhatsApp, Telegram, or email delivery.

What You'll Need

  1. Google Sheets Account with a structured database of yoga poses (columns for PoseName, Image URL, Resource Link)
  2. LINE Developer Account with a channel created and push message permissions configured
  3. n8n Instance (cloud or self-hosted) with access to the Schedule Trigger, Google Sheets, HTTP Request, and Code nodes
  4. AI Service Access (Azure OpenAI, OpenAI API, or similar) for content enhancement (optional but recommended)
  5. Basic understanding of JSON formatting for customizing Flex Message layouts

Quick Setup Guide

  1. Download and Import the JSON template into your n8n instance
  2. Configure Google Sheets Connection by adding your sheet ID and range in the "Get PoseName" node
  3. Set Up LINE Credentials in the "Line Push with Flex Bubble" node using your channel access token
  4. Adjust Scheduling in the Schedule Trigger node to match your target audience's timezone
  5. Test with Sample Data using manual execution before enabling the daily schedule
  6. Populate Your Sheets Database with at least 30 poses to ensure variety in daily selections
  7. Activate the Workflow and monitor initial deliveries for formatting issues

Pro tip: Start with a small test group of 5-10 users for the first week. Monitor open rates and click-throughs, then adjust message timing and content style before rolling out to your full audience.

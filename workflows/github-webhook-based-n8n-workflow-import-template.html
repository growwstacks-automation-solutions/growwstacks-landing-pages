What This Workflow Does

This automation template solves the challenge of manually deploying n8n workflows from GitHub repositories. It creates a seamless CI/CD pipeline for your automation workflows by automatically importing and deploying them whenever changes are pushed to your repository.

The solution uses GitHub webhooks to trigger the import process, then executes a series of steps to download, validate, and deploy the updated workflows to your n8n instance. This ensures your production automations stay perfectly synchronized with your version-controlled source files.

How It Works

1. Webhook Trigger

The workflow listens for GitHub webhook notifications configured in your repository settings. When changes are pushed, GitHub sends a payload to your n8n instance.

2. Repository Scanning

The automation lists all files in your repository and filters for JSON workflow files, identifying which workflows need to be updated.

3. File Download

Each identified workflow file is downloaded from GitHub and saved locally for processing.

4. Workflow Import

The template uses n8n's CLI import command to add or update workflows in your instance, maintaining all connections and settings.

5. Cleanup

Temporary files are removed after successful import, and error handling ensures failed imports are properly logged.

Pro tip: Use the companion repository's GitHub Action to automatically export workflows from your n8n instance back to GitHub, creating a complete version control cycle.

What You'll Need

  1. A self-hosted n8n instance (cloud version not supported)
  2. Admin access to configure webhooks in your GitHub repository
  3. Basic authentication credentials for secure webhook communication
  4. GitHub repository containing your n8n workflow JSON files

Quick Setup Guide

  1. Import the template into your n8n instance
  2. Configure webhook authentication credentials
  3. Set up GitHub API credentials
  4. Add the webhook URL to your GitHub repository settings
  5. Test by pushing a workflow change to your repository

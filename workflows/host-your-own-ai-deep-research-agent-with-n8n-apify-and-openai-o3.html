What This Workflow Does

This template replicates OpenAI's DeepResearch feature, which automates multi-step web research tasks using AI. It combines n8n's workflow automation with Apify's web scraping capabilities and OpenAI's language models to create a powerful research assistant that can:

  • Accept research queries through a web form
  • Perform recursive web searches to gather information
  • Scrape and analyze content from multiple sources
  • Synthesize findings into comprehensive reports
  • Save structured outputs to Notion or other destinations

Unlike manual research that takes hours, this automated system completes thorough investigations in minutes while maintaining consistent methodology across all research tasks.

How It Works

1. Research Query Input

The workflow starts with a form that captures the research topic and parameters like depth (how many layers to explore) and breadth (how many sources per layer).

2. Web Search & Scraping

Apify actors perform targeted web searches and content extraction, gathering relevant information from multiple sources based on the query.

3. Data Processing

The raw data goes through cleaning and organization steps to remove duplicates and irrelevant content before analysis.

4. AI Analysis

OpenAI models analyze the collected information, identifying key insights, trends, and patterns in the data.

5. Report Generation

The system generates a structured report summarizing findings, which gets saved to Notion or other destinations.

How It Works

The tutorial progresses from basic to advanced concepts through five carefully designed stages:

1. Sample Data Provision

The workflow begins with a structured dataset of user information, providing real-world data to work with rather than abstract examples. This immediate context helps you understand how your code will process actual business data.

2. Item-by-Item Processing

The first Code node demonstrates the "Run Once for Each Item" mode, teaching you how to access individual data items using $input.item.json. You'll calculate derived fields like full names and ages, learning fundamental data manipulation techniques.

3. External API Integration

This advanced stage introduces the this.helpers.httpRequest function to call the genderize.io API. You'll learn how to enrich data with external information, handle API responses, and manage asynchronous operations—skills essential for modern integrations.

4. Batch Processing

The third Code node switches to "Run Once for All Items" mode, showing how to process entire datasets efficiently using $items(). You'll calculate aggregate metrics like average age, learning when to use batch versus individual processing.

5. Binary File Generation

The final stage teaches file creation using JavaScript's Buffer object. You'll generate a CSV file from processed data, learning how n8n handles binary outputs—a crucial skill for reports, exports, and data sharing.

How It Works

The tutorial follows a progressive learning path with clear objectives and immediate feedback through task completion.

Chapter 1: Node Basics

You'll learn fundamental interactions with individual nodes, including renaming for clarity, editing configurations, duplicating complex nodes, and temporarily deactivating sections for testing. Each task is presented as a sticky note with specific instructions, guiding you to use the correct keyboard combinations for these everyday actions.

Chapter 2: Canvas Navigation & Selection

This section teaches efficient movement around large workflows. You'll practice zooming in on details, panning across complex canvases, selecting multiple nodes simultaneously, and organizing your workspace. These shortcuts are crucial for managing extensive automations without getting lost in the interface.

Chapter 3: Advanced Actions

Discover powerful workflow management techniques like tidying messy connections, creating reusable sub-workflows, and quickly reorganizing node layouts. These advanced shortcuts separate novice builders from automation experts, enabling sophisticated workflow architecture with minimal effort.

Chapter 4: Execution & Debugging

Learn essential shortcuts for testing and troubleshooting. You'll practice pinning data for inspection, toggling execution modes, navigating the executions panel, and quickly testing workflow branches. These skills dramatically reduce debugging time and increase confidence in your automations.

What This Workflow Does

Manual SEO tracking is time-consuming and error-prone. Marketing teams often spend hours each week checking keyword rankings across multiple tools, copying data into spreadsheets, and trying to spot trends. This process delays insights and makes it easy to miss when your website starts ranking for valuable new keywords.

This automation solves that problem by creating a systematic, hands-off approach to keyword monitoring. Each week, it automatically fetches the latest Google ranking data for your target keywords using DataForSEO's API, compares it with previous results, and identifies any keywords where your domain has newly entered the top 10 positions. These discoveries are immediately logged to Airtable for historical tracking and analysis, while your team receives a concise Slack summary highlighting the most important changes.

The workflow transforms reactive SEO monitoring into proactive opportunity identification. Instead of wondering why traffic increased, you'll know exactly which new rankings contributed to the growth, allowing you to double down on what's working and adjust what isn't.

The workflow connects DataForSEO's ranking data to Airtable for storage and Slack for team notifications.

How It Works

1. Scheduled Data Collection

The workflow triggers on a weekly schedule (configurable to daily or monthly). It first reads your target domains and keyword lists from predefined Airtable tables, ensuring you can easily update what you're tracking without modifying the automation itself.

2. API Data Fetching

Using DataForSEO's Labs API, the workflow retrieves current ranking positions, search volumes, and keyword difficulty scores for all monitored keywords. This happens in batches to respect API limits while ensuring complete data collection.

3. Historical Comparison

The system compares the newly fetched rankings against the previous week's data stored in Airtable. It uses logical operations to identify keywords where your domain's position has improved to within the top 10 (positions 1-10) since the last check.

4. Data Logging

All newly ranked top-10 keywords are written to an Airtable "New Rankings" table with timestamp, keyword, domain, position, search volume, and previous position (if any). This creates a searchable history of ranking improvements.

5. Team Notification

A formatted Slack message is sent to your designated channel, summarizing how many new top-10 rankings were discovered, highlighting the most valuable keywords by search volume, and providing a direct link to the Airtable view for deeper analysis.

Pro tip: Configure the workflow to run Monday mornings so your SEO team starts the week with fresh ranking insights and clear priorities for content optimization.

What You'll Need

  1. DataForSEO account with API credentials (Labs API access required)
  2. Airtable base with three tables: Target Domains, Keywords to Track, and New Rankings Log
  3. Slack workspace with permissions to create incoming webhooks
  4. n8n instance (self-hosted or cloud) with access to install the DataForSEO node
  5. List of target keywords you want to monitor (start with 20-50 core terms)

Quick Setup Guide

Follow these steps to implement this automation in under 30 minutes:

  1. Download and import the JSON template into your n8n instance
  2. Configure credentials for DataForSEO, Airtable, and Slack in n8n

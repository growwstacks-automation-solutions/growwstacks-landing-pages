What This Workflow Does

SEO professionals, website owners, and digital marketers spend hours manually checking backlink profiles, tracking new links, analyzing domain authority, and compiling reports. This manual process is time-consuming, inconsistent, and prone to errors.

This automation solves that problem by connecting SE Ranking (a comprehensive SEO tool) with Google Sheets. It automatically pulls your complete backlink profile data, tracks new and lost links over time, analyzes referring domain authority, examines anchor text distribution, and exports everything into organized spreadsheet sections. You get a live, updated backlink dashboard without any manual data collection.

How It Works

Step 1: Pull Overall Backlink Profile Stats

The workflow first retrieves your total backlinks count, number of referring domains, and overall profile metrics from SE Ranking. This provides the baseline snapshot of your link profile health.

Step 2: Check for New Backlinks (Last 30 Days)

It then queries SE Ranking for all new backlinks acquired in the last month, including their source URLs, anchor text, and first seen dates. This helps track link-building campaign effectiveness.

Step 3: Identify Lost Backlinks

The automation detects which backlinks you've lost recently and includes reasons why (domain removed, page deleted, etc.). This early warning system helps address potential ranking drops.

Step 4: List Top Referring Domains by Authority

It ranks your referring domains by their domain authority scores, highlighting which high-quality websites are linking to you—critical for SEO quality assessment.

Step 5: Analyze Anchor Text Distribution

The workflow examines how other sites are linking to you (keyword anchors, brand names, generic links) to ensure healthy, natural anchor text patterns.

Step 6: Export Everything to Google Sheets

All collected data is organized into clear sections within Google Sheets: summary stats, new links table, lost links log, top domains ranking, and anchor text analysis.

This automation is ideal for SEO agencies tracking multiple client websites, website owners monitoring their organic growth, digital marketers analyzing campaign effectiveness, and content teams assessing link acquisition from published articles. Anyone who needs consistent, accurate backlink data without manual effort benefits from this workflow.

What You'll Need

  1. A self-hosted n8n instance or n8n cloud account
  2. SE Ranking API token (available from your SE Ranking account)
  3. Google Sheets account for data export (optional but recommended)
  4. The SE Ranking community node installed in your n8n environment
  5. Basic understanding of SEO metrics and backlink terminology

Quick Setup Guide

  1. Download the template JSON file and import it into your n8n workspace.
  2. Install the SE Ranking community node via npm: @seranking/n8n-nodes-seranking
  3. Add your SE Ranking API credentials in the n8n credential manager.
  4. Replace "example.com" with your actual domain in all SE Ranking nodes.
  5. Connect Google Sheets by adding your credentials and specifying the spreadsheet ID.
  6. Test the workflow with a manual trigger, then schedule it for weekly automatic runs.

Pro tip: Add a Schedule Trigger node to run this automation weekly. This ensures you always have up-to-date backlink data without manual intervention.

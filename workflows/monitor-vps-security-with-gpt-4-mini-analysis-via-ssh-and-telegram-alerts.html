What This Workflow Does

This n8n template provides continuous security monitoring for your virtual private server (VPS) by combining SSH access with AI-powered analysis. It automatically scans running processes and network connections, using GPT-4 to identify potential threats that traditional monitoring might miss.

The system sends real-time Telegram alerts when it detects suspicious activity, allowing you to respond immediately to security incidents. Unlike basic monitoring tools, this workflow provides contextual analysis - explaining why a process might be dangerous and suggesting appropriate actions.

How It Works

1. Scheduled Security Scans

The workflow runs on a schedule you define (hourly by default), connecting to your server via SSH to execute security checks without installing additional software.

2. Comprehensive Data Collection

It gathers detailed system information including running processes (sorted by resource usage), active network connections, and user sessions - all the data needed for thorough security analysis.

3. AI Threat Analysis

The collected data is sent to OpenAI's GPT-4 model which evaluates it for known attack patterns, suspicious binaries, unusual network traffic, and other indicators of compromise.

4. Smart Alerting

When threats are detected, the system sends Telegram notifications with severity ratings, affected services, and recommended mitigation steps - no more sifting through raw logs.

What You'll Need

  1. SSH access to your Linux server (key-based authentication recommended)
  2. OpenAI API key (GPT-4 access required)
  3. Telegram bot token and chat ID for alerts
  4. n8n instance (self-hosted or cloud)

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure SSH credentials in the "Execute Command" node
  3. Add your OpenAI API key to the "GPT-4 Analysis" node
  4. Set up Telegram bot credentials in the alert node
  5. Adjust the schedule trigger to your preferred monitoring frequency
  6. Test with the "Execute Workflow" button to verify all connections

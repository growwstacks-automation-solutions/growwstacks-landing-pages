Monthly Live Music Events Newsletter

Automate monthly live music event discovery. Get curated concert lists for your city delivered by email, saving hours of manual searching.

Download Template JSON · n8n compatible · Free

What This Workflow Does

For music lovers, event promoters, venue owners, and local businesses, staying updated with live music events is both essential and time-consuming. Manual checking of websites like Songkick every month eats up hours that could be spent on more valuable tasks.

This automation solves that problem completely. It automatically fetches upcoming live music events for a specific city from Songkick.com, formats them into a clean, readable list, and delivers that list directly to your email inbox—or to multiple recipients—on a monthly schedule. Each event includes a direct link to the Songkick page for more details and ticket purchasing.

The workflow transforms a repetitive research task into a reliable, hands-free information pipeline. You receive timely, curated event data without ever opening a browser.

How It Works

Step 1: Schedule Trigger

A schedule node initiates the workflow monthly. This ensures the process runs automatically at the beginning of each month without manual intervention.

Step 2: Location & Email Configuration

A setup node holds your pre-configured Songkick location URL and recipient email addresses. This centralizes configuration for easy updates.

Step 3: Event Data Fetching

HTTP Request nodes scrape event data from the Songkick website for your specified city. The workflow extracts performer names, dates, venues, and direct links.

Step 4: Data Processing & Formatting

Set nodes and HTML Extract nodes clean and structure the raw data. They remove unnecessary information, format dates consistently, and prepare the event list for email presentation.

Step 5: Email Delivery

A Gmail node sends the formatted event list to all specified recipients. The email is professionally styled, making it easy to scan and act upon.

What You'll Need

  1. A working n8n instance (cloud or self-hosted).
  2. A configured Gmail account with OAuth credentials for sending emails.
  3. The Songkick location URL for your desired city (obtained via simple search on songkick.com).
  4. Recipient email addresses (one or multiple).

Quick Setup Guide

  1. Download & Import: Download the template JSON file and import it into your n8n workspace.
  2. Configure Gmail: Set up Gmail credentials in n8n using OAuth. Follow n8n's documentation for Gmail node authentication.
  3. Get Your Location URL: Visit songkick.com, search for your city, click the location result, and copy the page URL.
  4. Enter Configuration: Paste the Songkick URL and your recipient emails into the "Setup location and email" node.
  5. Activate: Save and activate the workflow. It will now run monthly and deliver event lists automatically.

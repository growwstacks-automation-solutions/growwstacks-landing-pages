What This Workflow Does

This automation solution solves the problem of manually checking Epic Games Store for free game releases. It continuously monitors the Epic Games website for changes to their free games section and instantly sends rich notifications to your Discord channel whenever new titles become available.

The workflow eliminates the need for daily manual checks while ensuring your gaming community never misses out on valuable free game offers. It transforms a repetitive task into a fully automated process that operates 24/7 with perfect reliability.

How It Works

1. Website Monitoring

The workflow checks the Epic Games Store HTML for changes to the container that holds free game information. Using web scraping techniques, it identifies when new content appears in this section.

2. Change Detection

A comparison mechanism detects when the monitored HTML structure changes, indicating new game availability. This triggers the notification process only when actual changes occur.

3. Discord Notification

When changes are detected, the workflow formats an attractive Discord message with game details and sends it to your specified channel via webhook or bot integration.

Pro tip: Configure the workflow to run every 6-12 hours for optimal performance without overloading Epic's servers.

What You'll Need

  1. A Discord server with webhook or bot posting permissions
  2. n8n installed (cloud or self-hosted)
  3. The n8n-nodes-puppeteer community node installed
  4. Basic understanding of Discord channel settings

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure the Discord webhook URL in the notification nodes
  3. Set your preferred monitoring frequency (recommended: every 6-12 hours)
  4. Test with the "Execute Workflow" button to verify notifications
  5. Activate the workflow for continuous monitoring

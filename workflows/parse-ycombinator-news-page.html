What This Workflow Does

Manually checking Hacker News (Y Combinator's news aggregator) for trending stories, startup insights, and tech discussions is time-consuming. This automation solves that by programmatically scraping the Hacker News homepage, extracting key data points like article titles, URLs, scores, and comment counts, then organizing everything into a clean, structured format.

The workflow transforms raw HTML into actionable business intelligence. Instead of browsing the site, you receive a formatted spreadsheet or list delivered via email or saved to cloud storage. This enables continuous market monitoring without manual effort, keeping you informed about trending technologies, startup news, and industry discussions that matter to your business.

How It Works

The automation follows a logical sequence to fetch, parse, structure, and deliver news data.

Step 1: Fetch the Webpage

An HTTP Request node retrieves the HTML content from the Hacker News homepage. The node handles connection, headers, and response processing, acting as a virtual browser that visits the site programmatically.

Step 2: Extract Structured Data

HTML Extract nodes parse the raw HTML using CSS selectors to locate specific elements. Each node targets different data points: one for article titles, another for links, another for scores, and another for comment counts. This separation ensures clean, isolated data extraction.

Step 3: Combine and Organize

A Merge node brings all extracted data streams together, aligning titles with their corresponding links, scores, and comments. The Item Lists node then structures this combined data into a proper table format with clear columns and rows.

Step 4: Format and Deliver

The Spreadsheet File node converts the structured table into an Excel file with automatic date-based naming. Finally, an Email node sends the file as an attachment to specified recipients, completing the automated pipeline from webpage to delivered intelligence.

What You'll Need

  1. A running n8n instance (self-hosted or cloud).
  2. Basic understanding of CSS selectors for HTML extraction (or willingness to use our pre-configured selectors).
  3. Email credentials configured in n8n if using the email delivery option.
  4. Access to the target website (Hacker News is publicly accessible).

Pro tip: Modify the CSS selectors to scrape other news sites or forums. The same workflow structure works for Reddit, product hunt, tech blogs, or any site with consistent HTML patterns.

Quick Setup Guide

  1. Download the template JSON file using the button above.
  2. In your n8n instance, go to Workflows → Import from file and select the downloaded JSON.
  3. The workflow canvas will appear with all nodes pre-configured.
  4. Test the HTTP Request node by executing it—you should see HTML response.
  5. Adjust the CSS selectors in HTML Extract nodes if Hacker News changes its layout.
  6. Configure the Email node with your SMTP credentials and recipient address.
  7. Activate the workflow and set a schedule (e.g., daily at 9 AM).

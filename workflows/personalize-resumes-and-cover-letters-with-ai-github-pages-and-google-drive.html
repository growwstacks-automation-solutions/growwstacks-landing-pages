What This Workflow Does

This automation solves the time-consuming process of manually tailoring resumes and cover letters for each job application. By combining AI analysis with GitHub Pages hosting and Google Drive storage, it creates a seamless system that:

Generates unique, position-specific resumes by matching your skills and experience with job requirements, converts them to polished PDFs automatically, and stores them in an organized Google Drive folder - all while maintaining your master resume template in GitHub for easy updates.

How It Works

1. Job Description Analysis

The workflow begins by analyzing the job description using AI to identify key requirements and keywords.

2. Content Matching

Your master resume content is matched against the job requirements to highlight the most relevant experiences.

3. HTML Generation

A tailored HTML resume is created using your template and pushed to GitHub Pages for hosting.

4. PDF Conversion

The hosted HTML is converted to a professional PDF using Gotenberg and saved to Google Drive.

5. Cover Letter Creation

An AI-generated cover letter is created to complement each customized resume.

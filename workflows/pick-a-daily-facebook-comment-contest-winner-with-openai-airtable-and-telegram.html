What This Workflow Does

Running daily social media contests manually is time-consuming, prone to bias, and difficult to scale. This automation solves that by creating a completely hands-off system for managing Facebook comment contests. It ensures fairness, rewards genuine engagement, and saves community managers hours of repetitive work each week.

The workflow automatically monitors a specified Facebook post, collects all new comments, cross-references them against a blocklist of recent winners in Airtable, uses OpenAI to analyze sentiment and filter out spam, randomly selects a winner from positive entries, logs the winner to your database, and announces the result via Telegram—all on a daily schedule.

How It Works

Step 1: Daily Trigger & Data Collection

Each day at your specified time (e.g., 9 PM), the workflow activates. It simultaneously fetches fresh comments from your Facebook post via the Graph API and retrieves the list of past winners from your Airtable base.

Step 2: Pre-Filtering & Blocklist Application

A custom code node processes the data: it removes comments from users who have won in the last 30 days (configurable), filters out very short comments or single emojis, and prepares clean data for sentiment analysis.

Step 3: AI-Powered Sentiment Analysis

OpenAI's GPT model analyzes each eligible comment, classifying it as Positive, Neutral, or Negative. Only comments with genuinely positive sentiment proceed to the winner pool, ensuring your contests reward constructive engagement.

Step 4: Random Winner Selection

The system randomly selects one winner from the filtered positive comments. This randomization ensures fairness while the sentiment filter maintains quality standards.

Step 5: Record Keeping & Notification

The winner's details (name, Facebook ID, comment, timestamp) are saved to Airtable for permanent record-keeping. Simultaneously, a celebratory announcement is sent to your Telegram channel, notifying your community instantly.

What You'll Need

  1. A running n8n instance (cloud or self-hosted)
  2. Facebook Developer App with Graph API access token
  3. Airtable base with a "Winners" table
  4. OpenAI API key (GPT-4o-mini works well)
  5. Telegram Bot token and channel ID
  6. Basic understanding of API credentials (we guide you through setup)

Pro tip: Start with a low-stakes contest to test the system. Once confident, scale to higher-value giveaways. The automation handles volume effortlessly—whether 50 or 5,000 comments.

Quick Setup Guide

  1. Import the downloaded JSON file into your n8n instance.
  2. Open the "Get FB Comments" node and add your Facebook Graph API credentials and Post ID.
  3. Configure the "Get Past Winners" Airtable node with your base and table details.
  4. Add your OpenAI API key to the sentiment analysis node.
  5. Set up the Telegram node with your bot token and target channel ID.
  6. Adjust the schedule trigger to your preferred daily time.
  7. Activate the workflow—your first automated contest will run at the next trigger time.

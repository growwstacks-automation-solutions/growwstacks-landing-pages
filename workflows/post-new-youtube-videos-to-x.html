Post New YouTube Videos to X

Automatically promote your latest YouTube content on X (formerly Twitter) with AI-generated posts that drive engagement and cross-platform traffic.

Download Template JSON · n8n compatible · Free

What This Workflow Does

Content creators and marketers waste hours manually promoting each new YouTube video across social platforms. This repetitive task not only consumes valuable time but often leads to inconsistent posting schedules, missed opportunities, and generic promotional messages that fail to engage audiences effectively.

This automated workflow solves these problems by connecting your YouTube channel directly to X (formerly Twitter). Whenever you publish a new video, the system automatically detects it, generates an engaging promotional message using AI, and posts it to your X account—all within minutes of publication. This ensures your content reaches your audience while it's fresh, with optimized messaging that drives clicks and engagement.

How It Works

The automation follows a precise three-step process that requires zero manual intervention once configured.

1. Scheduled YouTube Monitoring

The workflow runs on a schedule you define (typically every 30-60 minutes) and checks your YouTube channel for newly published videos. It compares publication timestamps against previous runs to identify fresh content that hasn't been promoted yet.

2. AI-Powered Content Generation

When a new video is detected, the system extracts key information including the title, description, and channel details. This data is sent to an AI language model (like OpenAI's GPT) which generates an engaging social media post tailored for X's audience and character limits.

3. Automated X Posting

The generated post is automatically published to your connected X account. The workflow can include the video link, relevant hashtags, and even schedule the post for optimal engagement times if configured.

Pro tip: Customize the AI prompt to match your brand voice. Include specific instructions about tone, hashtag preferences, and call-to-action formats to ensure generated posts align perfectly with your social media strategy.

What You'll Need

  1. YouTube Channel: A YouTube channel with published content (API access required)
  2. X Developer Account: Access to X's API with posting permissions
  3. AI API Key: OpenAI or compatible AI service API key for content generation
  4. n8n Instance: Self-hosted n8n or n8n.cloud account
  5. Basic Configuration: Channel IDs, API credentials, and posting preferences

Quick Setup Guide

Follow these steps to implement this automation in under 30 minutes:

  1. Download and Import: Download the template file and import it into your n8n instance
  2. Configure YouTube Node: Add your YouTube API credentials and channel ID to the YouTube trigger node
  3. Set Up AI Integration: Connect your OpenAI (or alternative) API key to the AI content generation node
  4. Connect X Account: Add your X API credentials to the X posting node with appropriate permissions
  5. Customize and Test: Adjust the schedule frequency, review the AI prompt, and run a test with an existing video
  6. Activate Workflow: Turn on the workflow and monitor the first few automated posts for quality assurance

Pro tip: Start with a private test video on YouTube to verify the entire flow works correctly before enabling for public content. This prevents accidental posts during configuration.

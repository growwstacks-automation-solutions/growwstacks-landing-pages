AI Review Analysis & Lead Qualification

Automatically analyze customer feedback, detect sentiment, and qualify leads using AI—no manual review required.

Download Template JSON · n8n compatible · Free

What This Workflow Does

Manually reading and categorizing customer feedback is time‑consuming, inconsistent, and often misses critical insights buried in text. This intelligent automation solves that by turning unstructured reviews into actionable, structured data instantly.

The workflow captures new customer feedback from Google Sheets, sends it to an AI model (Azure GPT‑4o‑mini) for analysis, classifies the intent (praise, complaint, suggestion, question), detects sentiment (positive, neutral, negative, mixed), assigns a score from 1–10, and generates a concise summary. All this enriched data is then merged with the original customer details and saved back to a structured sheet, ready for reporting, CRM import, or dashboard visualization.

It’s a complete hands‑off system for sales, product, and customer success teams to qualify leads, identify risks, and spot opportunities at scale.

How It Works

1. Trigger on New Feedback

The workflow watches a specific Google Sheet where your customer feedback form responses or review exports land. Each new row triggers the automation.

2. AI Analysis & Classification

The customer review text is sent to Azure OpenAI’s GPT‑4o‑mini model with a carefully crafted prompt. The AI returns a clean JSON object containing the classified intent, detected sentiment, a numerical score, and a short summary.

3. Data Parsing & Enrichment

A Code node parses the AI’s JSON response, extracting each insight into separate fields. The original customer details (name, email, date) are preserved and merged with the new AI‑generated insights.

4. Structured Output

The complete, enriched dataset—customer info plus AI analysis—is appended to a destination Google Sheet. This creates a searchable, sortable master database of analyzed feedback for your team.

Pro tip: Use the sentiment score to create instant alerts. Set up a simple conditional branch to send a Slack notification whenever a review scores below 3, so your customer success team can intervene immediately.

What You'll Need

  1. A Google Sheets spreadsheet containing customer feedback (from forms, exports, or manual entry).
  2. A Google Cloud service account with Sheets API enabled for n8n to read/write.
  3. An Azure OpenAI resource with GPT‑4o‑mini model deployed (or another compatible AI provider).
  4. An n8n instance (self‑hosted or n8n.cloud) to run the workflow.
  5. Basic familiarity with n8n to configure credentials and map your sheet columns.

Quick Setup Guide

Get this automation running in under 15 minutes.

  1. Download & Import: Click the download button above and import the JSON file into your n8n instance.
  2. Configure Google Sheets: Set up the Google Sheets node with your service account credentials. Point it to your source feedback sheet and a destination sheet for results.
  3. Connect AI: Add your Azure OpenAI endpoint and API key to the AI node. Test the connection with a sample review.
  4. Map Fields: Ensure the workflow correctly maps your source sheet columns (customer name, review text, etc.) to the AI prompt.
  5. Test & Activate: Add a test row to your source sheet and run the workflow once manually. Verify the enriched data appears in your destination sheet, then activate the workflow.

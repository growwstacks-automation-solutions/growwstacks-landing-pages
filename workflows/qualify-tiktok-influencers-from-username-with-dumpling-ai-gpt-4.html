What This Workflow Does

Manual influencer qualification is time-consuming and inconsistent. Marketing teams spend hours checking TikTok profiles, counting followers, calculating engagement rates, and manually updating spreadsheets. This process slows down campaigns and causes missed opportunities with fast-rising creators.

This automation solves that by creating a seamless qualification pipeline. Submit a TikTok username, and the system automatically fetches the creator's complete profile data using Dumpling AI, analyzes it against your specific criteria using GPT-4, and then updates your Google Sheets database—adding new qualified influencers or updating existing records. What used to take 15-20 minutes per influencer now happens in seconds with perfect consistency.

How It Works

1. Form Trigger & Data Collection

The workflow starts when a team member submits a TikTok username through a simple form. This could be from a Google Form, Typeform, or directly within n8n. The username is passed to the next step without any manual intervention.

2. TikTok Profile Fetching with Dumpling AI

The system sends the username to Dumpling AI's TikTok API, which returns comprehensive profile data including follower count, following count, total likes, video count, bio, and other engagement metrics. This eliminates the need for manual profile visits and data entry.

3. AI-Powered Qualification Analysis

GPT-4 analyzes the profile data against your predefined qualification rules. You can customize these rules, but the template includes common criteria: 40+ videos, 100,000+ followers, and 300,000+ total likes. The AI provides a clear "qualified" or "not qualified" decision with reasoning.

4. Google Sheets Integration & Database Management

The workflow checks your Google Sheets database to see if the influencer already exists. If they're new, it adds a complete row with all their data and qualification status. If they exist already, it updates their metrics and qualification status. This keeps your outreach list always current.

Pro tip: Customize the GPT-4 prompt to include your brand's specific qualification criteria, such as niche relevance, audience demographics, or content style preferences. This makes the automation perfectly aligned with your marketing strategy.

What You'll Need

  1. Dumpling AI API Key: For fetching TikTok profile data (HTTP Header authentication)
  2. OpenAI API Key: For GPT-4 analysis and qualification decisions
  3. Google Sheets Access: With a spreadsheet containing columns for TikTok user, User ID, Follower Count, Following Count, Heart Count, Video Count, and Qualified status
  4. n8n Instance: Either self-hosted or n8n.cloud account
  5. Form Trigger: Basic form setup to collect TikTok usernames (can be internal or external)

Quick Setup Guide

  1. Download the template using the button above and import it into your n8n instance
  2. Configure the Dumpling AI HTTP Request node with your API key in the headers
  3. Add your OpenAI API key to the GPT-4 node settings
  4. Connect the Google Sheets nodes to your specific spreadsheet and worksheet
  5. Test with a known TikTok username to verify data flow and qualification logic
  6. Set up your form trigger (Google Forms, Typeform, or n8n's built-in form)
  7. Deploy the workflow and start submitting usernames for automatic qualification

Pro tip: Before going live, test with 5-10 influencers you've already manually qualified to ensure the AI's decisions match your team's judgment. Adjust the GPT-4 prompt as needed based on these test results.

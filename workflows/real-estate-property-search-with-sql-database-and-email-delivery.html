What This Workflow Does

This automation solves the time-consuming problem of manually searching property databases and compiling results for clients. Instead of spending hours filtering spreadsheets or MLS systems, real estate professionals can provide instant, personalized property matches 24/7 through a simple web form.

The workflow captures client preferences through a customizable form, dynamically builds optimized SQL queries based on their criteria, searches your property database in milliseconds, generates professional CSV reports with complete property details, and delivers personalized emails with attached property lists—all without manual intervention.

How It Works

The automation follows a logical sequence that transforms client input into actionable property recommendations.

1. Client Submission & Data Capture

A web form collects client search criteria including budget, location preferences, property type, bedrooms/bathrooms, and special requirements. This form can be embedded on your website, shared via link, or integrated into existing client portals.

2. Dynamic SQL Query Building

The workflow intelligently constructs optimized SQL queries based on which criteria the client provided. Empty fields are excluded from the search, ensuring flexible matching without requiring clients to complete every field.

3. Database Search & Filtering

Using Microsoft SQL Server (or compatible databases), the system queries thousands of properties in milliseconds, applying all specified filters to return only relevant matches. The search can be expanded to include proximity calculations, school district data, and market trends.

4. Results Processing & Report Generation

Matching properties are formatted into a professional CSV report with complete details—address, price, specifications, broker information, and unique identifiers. The system can optionally enrich results with calculated metrics like price per square foot or estimated monthly payments.

5. Automated Email Delivery

Using Gmail or your preferred email service, personalized messages are sent to clients with the CSV attachment, property highlights, and next-step instructions. The email template can be fully branded with your agency's identity and contact information.

Pro tip: Add a "Save Search" option that automatically notifies clients when new properties matching their criteria hit the market, creating ongoing engagement opportunities.

Quick Setup Guide

Follow these steps to implement this automation in your business:

  1. Import the workflow template into your n8n instance using the download button above
  2. Prepare your property database using the included CSV template or connect to your existing MLS system
  3. Configure database credentials in the Microsoft SQL node with your connection details
  4. Set up Gmail OAuth2 authentication following n8n's documentation for email automation
  5. Customize the webhook form to match your branding and property search criteria
  6. Test the workflow with sample searches to ensure proper database matching and email delivery
  7. Embed the form on your website or share the direct link with clients

Implementation note: Start with a small subset of properties for testing, then gradually expand to your full database once you're confident in the matching logic and email formatting.

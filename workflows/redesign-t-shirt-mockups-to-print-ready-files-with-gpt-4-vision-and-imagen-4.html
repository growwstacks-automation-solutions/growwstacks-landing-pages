Automate T-shirt Mockup Redesign with AI

Transform rough T-shirt mockups into print-ready artwork using GPT-4 Vision & Imagen 4. Free n8n workflow template for apparel businesses.

What This Workflow Does

This automation solves a common problem for apparel businesses and designers: outdated or rough T-shirt mockups that need to be transformed into professional, print-ready artwork. Manual redesign processes are time-consuming, inconsistent, and delay production cycles.

The workflow uses AI to analyze any uploaded T-shirt mockup image—even if it's low-quality or contains distracting elements like collars and shadows—and automatically generates a refined, print-ready design optimized for production. It preserves the original concept while enhancing visual composition, typography, and removing non-essential mockup elements.

Businesses save hours of manual design work per mockup, reduce errors in print file preparation, and can scale their design production without increasing staff. The output is a clean artwork locked on a pure black background, ready for direct printing.

How It Works

Step 1: Image Analysis with GPT-4 Vision

The workflow begins when a T-shirt mockup image URL is provided. OpenAI's GPT-4 Vision analyzes the image to extract key design elements: characters, text, layout, graphic style, composition, and visual tone. This understanding forms the foundation for the redesign.

Step 2: AI Agent Creates Refined Prompt

An AI agent processes the extracted details to create a clean, artistic prompt. It preserves the original layout and message while enhancing visual composition, removing mockup elements (collars, sleeves, shadows), and locking the artwork on a pure black background only.

Step 3: Text Escaping for API Compatibility

A JavaScript function node escapes the prompt—handling quotes, slashes, and line breaks—to make it safe for downstream JSON requests. This ensures the generated prompt works seamlessly with AI image generation APIs.

Step 4: Image Generation via GPT-Image-1 or Imagen 4

The final prompt is sent to OpenAI's GPT-Image-1 or Google's Imagen 4 to generate brand-new artwork. The result is a high-quality, print-ready design ideal for direct printing on black T-shirts, with no visible shirt or mockup framing.

Step 5: Optional Delivery & Storage

The generated artwork can be automatically sent to Telegram, uploaded to Notion, stored in Google Drive, or integrated into your design management system. This completes the automation pipeline from concept to delivered file.

Pro tip: Configure cost monitoring for GPT-Image-1 usage—each generation costs approximately $0.25. For high-volume production, consider batch processing and usage optimization.

Quick Setup Guide

  1. Download and import the JSON template into your n8n workspace.
  2. Configure OpenAI credentials in the workflow nodes for both vision and image generation.
  3. Test with a sample mockup URL to verify the analysis and generation process works correctly.
  4. Adjust prompt refinement settings if needed to match your specific design requirements.
  5. Connect output nodes to your preferred storage or communication tools (Telegram, Notion, etc.).
  6. Set up triggers for automated execution—webhook, schedule, or manual input.
  7. Monitor initial runs and optimize costs by adjusting image resolution or batch settings.

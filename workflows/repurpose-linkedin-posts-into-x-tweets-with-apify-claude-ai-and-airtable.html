What This Workflow Does

Content creators and marketers spend hours manually rewriting LinkedIn posts for X (Twitter), often missing opportunities or posting inconsistently. This workflow automates the entire repurposing pipeline, transforming your professional LinkedIn content into engaging X threads and standalone tweets.

The system scrapes your recent LinkedIn posts using Apify, extracts text and carousel content, processes it through Claude AI to create platform-optimized variations, stores everything in Airtable for team review, and automatically publishes approved content to X on your schedule. It turns a 2-3 hour weekly task into a completely hands-off process.

How It Works

1. Weekly LinkedIn Scraping

Every Sunday, Apify scrapes your LinkedIn profile for recent posts based on configurable parameters. It captures the full text, images, PDF carousels, engagement metrics, and publication dates, creating a clean data feed for processing.

2. Carousel Content Extraction

If posts contain PDF carousels, OpenAI's vision capabilities extract text from each slide via OCR. This ensures all valuable content from visual presentations is included in the repurposing process, not just the post caption.

3. AI-Powered Transformation

Claude AI (via OpenRouter) analyzes each LinkedIn post and generates two X-optimized variations: a concise standalone tweet (280 characters) and a threaded breakdown (3-7 tweets). The AI adjusts tone, adds relevant hashtags, and structures content for maximum engagement.

4. Airtable Approval Hub

All generated tweets are stored in Airtable with "Pending" status. Your team can review, edit, approve, or reject content, set publication dates, and maintain version control. A "Tweet Now" button allows instant publishing when needed.

5. Scheduled X Publishing

Daily at 12:30 PM, the workflow checks for approved tweets whose publication date has passed and publishes them to X. Threads are properly chained as replies, and analytics are tracked for performance monitoring.

What You'll Need

  1. Apify account with LinkedIn Profile Posts Scraper actor credits
  2. Airtable account with a base containing Config, LK Posts, and X Tweets tables
  3. OpenAI API access for carousel OCR (GPT-4 Vision)
  4. OpenRouter API access for Claude AI processing
  5. X Developer account with API v2 access and elevated permissions
  6. n8n instance (cloud or self-hosted) to run the workflow

Pro tip: Start with scraping just the last 5-10 LinkedIn posts to test the AI transformation quality before scaling to your full archive. Adjust Claude's instructions in the workflow to match your brand voice.

Quick Setup Guide

  1. Download the template and import into your n8n instance
  2. Create the three Airtable tables (Config, LK Posts, X Tweets) with the specified field structures
  3. Add your LinkedIn profile URL and scraping parameters to the Config table
  4. Configure the "Tweet Now" button in Airtable with your n8n webhook URL
  5. Connect all credentials: Apify, Airtable, OpenAI, OpenRouter, and X
  6. Adjust the weekly and daily schedule triggers to match your timezone
  7. Test with one LinkedIn post, then activate the full workflow

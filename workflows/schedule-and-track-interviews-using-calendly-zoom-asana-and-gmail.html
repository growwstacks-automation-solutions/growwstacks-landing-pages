What This Workflow Does

Manual interview scheduling is a time-consuming, error-prone process that frustrates recruiters and creates a poor candidate experience. This automation solves that by connecting Calendly, Zoom, Asana, and Gmail into a single, reliable hiring pipeline.

When a candidate books an interview slot, the workflow automatically creates a Zoom meeting, assigns a structured task in Asana to the appropriate interviewer, and sends a detailed notification email—all without any manual intervention. It ensures consistency, reduces scheduling errors, and keeps your entire hiring team aligned.

The system handles different interview types (like HR screens vs. technical interviews) with conditional routing, ensuring the right stakeholders are involved while maintaining a professional, seamless experience for candidates.

How It Works

1. Trigger from Calendly Booking

The workflow starts when a candidate books a time through your Calendly scheduling page. It captures all booking details: candidate name, email, interview type, scheduled time, and timezone.

2. Data Normalization & Processing

Interview details are structured and formatted consistently. Timezone conversions are handled automatically, and the data is prepared for the next steps in the pipeline.

3. Automated Zoom Meeting Creation

A Zoom meeting is generated with the correct date, time, and duration. The join link and dial-in details are captured for inclusion in notifications and task descriptions.

4. Interview Type Routing

The workflow checks the interview type (e.g., "Technical," "HR," "Final Round") and routes it accordingly. This determines which interviewer gets assigned and what notification template is used.

5. Asana Task Creation & Assignment

A task is created in your designated Asana project. It includes the candidate details, Zoom link, interview type, and any specific instructions. The task is automatically assigned to the designated interviewer.

6. Gmail Notification to Interviewer

An email is sent to the interviewer with everything they need: candidate info, interview time, Zoom link, and a link to the Asana task. This ensures they have all context in one place.

If you're scheduling more than 5 interviews per week, dealing with multiple interviewers, or struggling with missed meetings and manual follow-ups, this template will save you significant time.

What You'll Need

  1. Calendly account with webhook access enabled (Professional plan or higher).
  2. Zoom account with API credentials (app creation permissions).
  3. Asana account with OAuth2 access to create tasks in your projects.
  4. Gmail/Google Workspace account with API access to send emails.
  5. n8n instance (cloud or self-hosted) to run the workflow.
  6. Basic understanding of how to set up API connections in n8n.

Pro tip: Before connecting live, test the workflow with a dummy Calendly event type and a sandbox Asana project. This lets you verify the entire sequence works correctly without affecting real interviews.

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance.
  2. Set up credentials for Calendly (webhook trigger), Zoom, Asana, and Gmail in n8n.
  3. Configure the Calendly webhook by copying the webhook URL from the first node and adding it in your Calendly account under Integrations → Webhooks.
  4. Update the Asana project and assignee IDs in the "Create Asana Task" node to match your actual project and team member IDs.
  5. Customize the Gmail notification template with your company branding and specific interview instructions.
  6. Test the workflow by booking a test interview in Calendly and verifying that a Zoom meeting is created, an Asana task appears, and an email is sent.
  7. Activate the workflow and let it handle your real interviews automatically.

What This Workflow Does

Staying ahead in your industry requires constant awareness of news, competitor moves, and market trends. Manually scanning dozens of websites, blogs, and news feeds is a massive time drain that often leads to missed information. This automation solves that problem completely.

This n8n workflow acts as your automated research assistant. It systematically scrapes targeted news sources using Bright Data, processes each article with OpenAI to generate a concise summary and relevant tags, and then compiles everything into a professional daily digest. This digest is delivered to your team via Slack and email, while detailed summaries are archived in Notion for future reference. You get comprehensive market intelligence without lifting a finger.

How It Works

Step 1: Scheduled Trigger & Source Configuration

The workflow starts on a schedule (e.g., daily at 9 AM). A configuration node defines the list of target news websites, blogs, RSS feeds, and specific keywords relevant to your sector.

Step 2: Web Scraping with Bright Data

Bright Data nodes visit each configured source, reliably extracting the latest article titles, URLs, publication dates, and full content. It handles anti-bot measures and site structure changes.

Step 3: AI Processing with OpenAI

Each scraped article is sent to an OpenAI model. The AI generates a one-paragraph summary capturing the essence, extracts key entities (company names, products, people), and assigns topic tags (e.g., "Product Launch," "Funding," "Regulation").

Step 4: Compilation & Distribution

All processed summaries are compiled into a clean Markdown or HTML format. A concise version is posted to a designated Slack channel for immediate team visibility. A more detailed version is sent via Gmail to key stakeholders. Every summary, with its source link and tags, is also saved as a page in a Notion database, creating a searchable knowledge base.

What You'll Need

  1. A self-hosted or cloud n8n instance (this template uses community nodes).
  2. Bright Data account and API credentials for web scraping.
  3. OpenAI API key (GPT-3.5-Turbo or GPT-4).
  4. Slack workspace with a webhook URL for your target channel.
  5. Gmail account (or SMTP credentials) for sending email digests.
  6. Notion account and a database ID for archiving summaries.

Pro tip: Start with 3-5 high-quality, niche news sources specific to your field. It's better to have highly relevant summaries from a few sources than a flood of irrelevant noise from many. You can easily expand the list later.

Quick Setup Guide

  1. Import the Template: Download the JSON file and import it into your n8n workspace.
  2. Configure Credentials: In the n8n credentials menu, set up connections for Bright Data, OpenAI, Slack, Gmail, and Notion.
  3. Customize Sources: Edit the "Set" node to input your target website URLs, RSS feeds, and keywords.
  4. Set Delivery Channels: Update the Slack channel webhook and the recipient email addresses in the respective nodes.
  5. Test & Activate: Run the workflow once manually to verify everything works, then activate the schedule trigger.

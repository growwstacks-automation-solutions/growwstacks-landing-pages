What This Workflow Does

This automation solves the tedious manual process of checking Trustpilot for customer reviews. Businesses spend hours each week visiting their Trustpilot profile, copying reviews, pasting them into spreadsheets, and trying to analyze trends. This workflow automates the entire collection process, pulling reviews directly into Google Sheets where they can be sorted, analyzed, and integrated with other systems.

The template creates two structured sheets: one for raw review data collection, and another formatted specifically for HelpfulCrowd import. This enables businesses to maintain a complete historical record of all customer feedback while streamlining their reputation management workflow. Whether you're monitoring your own brand or tracking competitors, this automation provides consistent, reliable data collection without manual effort.

How It Works

The workflow follows a logical sequence to extract, transform, and store review data efficiently.

Step 1: Trigger and Configuration

The workflow can be triggered manually or scheduled to run at regular intervals (daily, weekly, etc.). You configure the Trustpilot business profile name and pagination settings in a global node, making it easy to switch between different companies or profiles.

Step 2: Data Extraction

Using HTTP requests, the workflow navigates through Trustpilot's review pages, extracting key information including reviewer name, rating, date, review text, and any business responses. It handles pagination automatically to collect all available reviews.

Step 3: Data Transformation

Extracted data is cleaned and structured using Set nodes. The workflow formats dates consistently, normalizes ratings, and prepares the data for two different output formats: a raw collection sheet and a HelpfulCrowd-compatible format with review_id for upsert operations.

Step 4: Google Sheets Integration

The processed data is written to your Google Sheets document. The template maintains data integrity by checking for existing entries and only adding new reviews, preventing duplicates even when the workflow runs multiple times.

Pro tip: Schedule this workflow to run daily at 9 AM to have fresh review data ready for your morning team meeting. This ensures you're always working with the latest customer feedback.

Quick Setup Guide

Follow these steps to get your Trustpilot review automation running in under 15 minutes:

  1. Clone the Google Sheets template using the provided link in the workflow description.
  2. Download this JSON template using the button above and import it into your n8n instance.
  3. Configure the Global node with your Trustpilot business name and desired pagination settings.
  4. Set up Google Sheets credentials in n8n and connect them to the Google Sheets nodes.
  5. Update the spreadsheet ID in the Google Sheets nodes to point to your cloned document.
  6. Test the workflow with a manual trigger to verify data flows correctly.
  7. Schedule regular execution (daily recommended) or keep it as an on-demand tool.

Pro tip: Start with manual triggers to verify everything works, then add a schedule trigger. For businesses with high review volume, consider running the workflow twice daily to catch new reviews quickly.

What This Workflow Does

Manual competitor analysis is time-consuming and inconsistent. Marketing teams spend hours reading through Trustpilot reviews trying to identify patterns, only to struggle with turning those insights into compelling ad copy. This creates missed opportunities and reactive marketing strategies.

This n8n workflow automates the entire competitor intelligence process. It scrapes Trustpilot reviews using Bright Data's professional scraping infrastructure, filters for negative feedback (1–2 star reviews where real pain points emerge), analyzes common themes, and uses GPT-4o-mini to generate three variations of Facebook ad copy that directly address those pain points. The complete analysis and ad variations are then emailed to your marketing team, ready for testing.

The automation handles everything from data collection to insight delivery, turning what was a multi-hour manual task into a 10-minute automated process that delivers superior, data-driven results.

How It Works

Step 1: Trigger & Input Collection

The workflow starts with a manual form trigger where you input the competitor's Trustpilot URL and select the review timeframe (30 days, 3 months, 6 months, or 12 months). This ensures you're analyzing recent, relevant feedback that reflects current market conditions.

Step 2: Professional Review Scraping

Using Bright Data's Dataset API, the workflow submits the scraping request and polls until the snapshot is ready. Bright Data handles anti-bot measures, JavaScript rendering, and structured data extraction—capabilities that basic scrapers lack. This ensures reliable, complete data collection without IP bans.

Step 3: Data Storage & Organization

All extracted reviews are saved into a structured Google Sheet with columns for reviewer name, rating, date, review text, and sentiment indicators. This creates a searchable historical database of competitor feedback that grows with each analysis run.

Step 4: Pain Point Analysis

The workflow filters to only 1–2 star reviews—where genuine customer frustrations are most visible. It then aggregates common complaints, identifies recurring themes, and creates a summary of the competitor's weakest points from the customer perspective.

Step 5: AI-Powered Ad Copy Generation

The pain point summary is sent to OpenAI's GPT-4o-mini with specific instructions to generate three distinct Facebook ad variations. Each variation targets different emotional angles (problem-focused, solution-focused, social proof-focused) while using language mirroring actual customer complaints.

Step 6: Insight Delivery

The complete package—review summary, pain point analysis, and three ad variations—is emailed to your marketing team. This delivers actionable intelligence immediately, without requiring additional processing or interpretation.

Quick Setup Guide

  1. Copy the Google Sheet template – Use the provided link to create your own copy. Do not modify column headers as the workflow depends on this structure.
  2. Configure n8n credentials – Set up OAuth2 for Google Sheets, API key for OpenAI, and authorization header for Bright Data.
  3. Import the workflow JSON – Download the template and import it into your n8n instance.
  4. Update configuration nodes – Point to your Google Sheet, set your dataset ID, and configure your email delivery settings.
  5. Test with a single competitor – Run the workflow with one Trustpilot URL to verify everything works before expanding your analysis.

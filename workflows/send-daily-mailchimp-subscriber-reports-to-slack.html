Send Daily Mailchimp Subscriber Reports to Slack

Automate your email list reporting. Get daily subscriber counts delivered directly to your team's Slack channel—no manual checks required.

What This Workflow Does

This automation solves a common problem for marketing teams: staying updated on email list growth without constantly logging into Mailchimp. Manually checking subscriber counts wastes valuable time and creates information silos where only one person knows the numbers.

This workflow automatically fetches your total Mailchimp subscribers at a scheduled time each day and posts the count to a designated Slack channel. It transforms raw data into actionable team communication, ensuring everyone from marketing to leadership sees audience growth trends in real-time.

The automation runs reliably in the background, providing consistent visibility without manual intervention. It's particularly valuable for tracking campaign effectiveness, monitoring list health, and celebrating growth milestones as they happen.

How It Works

The workflow follows a simple three-step process that runs automatically on your schedule.

1. Scheduled Trigger

A Cron node initiates the workflow daily at your specified time (default 9:00 AM). You can easily adjust the frequency to weekly, hourly, or any custom schedule that fits your reporting needs.

2. Fetch Mailchimp Data

The workflow connects to Mailchimp's API using your credentials and retrieves the total subscriber count for your selected audience list. It handles authentication securely and can be configured for multiple lists if needed.

3. Post to Slack

The subscriber count is formatted into a clean, readable message and posted to your chosen Slack channel. The message includes timestamps and can be customized with emojis, additional metrics, or growth percentages compared to previous periods.

Pro tip: Add a "Compare with yesterday" function by storing yesterday's count in a simple database node, then calculate and display daily growth percentage in your Slack message.

What You'll Need

  1. A Mailchimp account with an active audience list
  2. Your Mailchimp List ID (found in Audience Settings)
  3. A Slack workspace where you have permission to post messages
  4. n8n instance (cloud or self-hosted) with credentials set up for both services
  5. Basic understanding of where to find API keys in both platforms

Quick Setup Guide

Follow these steps to implement this automation in under 10 minutes:

  1. Import the template into your n8n instance using the downloaded JSON file
  2. Configure Mailchimp credentials in n8n with your API key
  3. Enter your List ID in the Mailchimp node (find this in Mailchimp under Audience → Settings)
  4. Set up Slack credentials in n8n with a bot token or webhook
  5. Specify your Slack channel name (like #marketing or #growth)
  6. Test the workflow with the "Execute Workflow" button to verify everything works
  7. Activate the workflow and let it run automatically on your schedule

Pro tip: Before going live, test with a private Slack channel or direct message to yourself to verify the format and data accuracy.

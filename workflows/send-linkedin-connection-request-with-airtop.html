What This Workflow Does

Manual LinkedIn outreach is time-consuming and repetitive. Sales teams spend hours each week visiting profiles, clicking connect buttons, and typing personalized messages. This automation eliminates that manual work by automatically sending LinkedIn connection requests to your target prospects.

The workflow intelligently handles the entire process: it checks if you're already connected, avoids duplicate requests, and can include personalized messages. It works through browser automation using Airtop, which means it can perform actions that LinkedIn's official API doesn't allow, giving you more flexibility in your outreach strategy.

By automating this process, you can scale your LinkedIn networking efforts while maintaining personalization. Whether you're building a prospect list, expanding your professional network, or conducting targeted outreach campaigns, this workflow turns hours of manual work into minutes of automated execution.

How It Works

Step 1: Input Preparation

The workflow accepts three key inputs: the LinkedIn profile URL of your target prospect, your authenticated Airtop profile name, and an optional personalized message. You can feed these inputs from a spreadsheet, CRM, or any data source containing your prospect list.

Step 2: Browser Session Initialization

Airtop starts a browser session using your pre-authenticated LinkedIn profile. This is crucial because it maintains your personal account context, making the connection requests appear to come from you personally rather than a generic automation tool.

Step 3: Profile Verification

The automation navigates to the target LinkedIn profile and checks your current connection status. It detects if you're already connected, if a previous request is pending, or if the "Connect" button is available. This prevents awkward duplicate requests.

Step 4: Connection Request Execution

If the connection button is available, the workflow clicks it. If you've provided a personalized message, it selects the "Add a note" option, types your message, and sends the request. If no message is provided, it sends a simple connection request without a note.

Step 5: Session Cleanup

After completing the action, the browser session is terminated cleanly. The workflow can then move to the next prospect in your list, creating a seamless, scalable outreach sequence.

What You'll Need

  1. Airtop account (free tier available) with an API key generated from your dashboard
  2. Airtop browser profile authenticated with your LinkedIn account (one-time setup)
  3. n8n instance (cloud or self-hosted) to run the workflow

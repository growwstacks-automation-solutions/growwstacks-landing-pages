What This Workflow Does

This automation solves a common but time-consuming challenge for sales and marketing teams: identifying and connecting with engaged professionals who comment on competitor LinkedIn posts. Instead of manually scrolling through hundreds of comments and sending individual connection requests, this workflow automatically extracts all commenters, filters out competitor employees, and sends personalized connection requests with built-in daily limits that respect LinkedIn's guidelines.

The system transforms passive competitor engagement into active lead generation opportunities. When your competitors post content that generates discussion, those commenters represent a warm audience already interested in your industry. This workflow captures that audience before your competitors can, turning their engagement into your pipeline.

What makes this automation particularly valuable is its intelligent pacing system. When you hit your daily connection limit (configurable from 5-20 requests), the workflow automatically calculates the time until midnight, pauses, and resumes the next day. This means one submission can process hundreds of commenters over multiple days without manual intervention.

How It Works

The workflow follows a logical sequence that balances automation effectiveness with platform compliance. Here's the step-by-step process:

Step 1: Form Submission and Configuration

Users submit a simple form with the LinkedIn post URL, competitor company name for filtering, and an optional personalized connection message template. The system validates the input and sets up the processing parameters, including the daily connection limit that prevents LinkedIn restrictions.

Step 2: Comment Extraction and Deduplication

The workflow connects to the ConnectSafely.ai API to extract all commenters from the specified LinkedIn post. It then deduplicates profiles to ensure the same person doesn't receive multiple connection requests, maintaining professionalism and preventing spammy behavior.

Step 3: Sequential Profile Processing with Limits

The system processes each profile sequentially, checking against the daily limit before each action. If the limit is reached, it calculates the exact seconds until midnight, pauses the workflow, and automatically resumes the next day. This intelligent pacing is crucial for long-term account health.

Step 4: Competitor Filtering and Personalization

Each profile is checked against the competitor company name to filter out employees who work for the competition. For remaining profiles, the system sends a personalized connection request using the message template, which can include placeholders like {firstName} for automatic personalization.

Step 5: Logging and Summary Reporting

Every action is logged with timestamps and outcomes. When processing completes, the workflow generates a comprehensive summary report showing total profiles processed, connections sent, profiles skipped (with reasons), and whether the daily limit was reached.

What You'll Need

  1. ConnectSafely.ai API Credentials: You'll need an active account with API access to extract LinkedIn comments and send connection requests through their compliant platform.
  2. n8n Instance: Either self-hosted n8n or n8n.cloud account with sufficient execution time for processing potentially hundreds of profiles over multiple days.
  3. LinkedIn Account: A professional LinkedIn account with connection request privileges (not restricted) and established presence to maintain reasonable acceptance rates.
  4. Target Competitor Posts: Public LinkedIn posts from competitors or industry influencers with engaged comment sections (20-100 comments ideal for quality leads).
  5. Personalized Message Templates: Thoughtful connection request templates that provide value and context rather than generic sales pitches.

Pro tip: Start with a conservative daily limit of 5-8 connection requests for new accounts, gradually increasing to 10-15 as you establish consistent acceptance rates. Never exceed 20 per day to avoid LinkedIn's automated restrictions.

Quick Setup Guide

Getting this automation running takes about 15 minutes if you have all prerequisites ready. Follow these steps:

  1. Download and Import: Download the template file and import it into your n8n instance through the workflow import function.
  2. Configure API Credentials: Add your ConnectSafely.ai API key as a Header Auth credential in n8n with the name "Authorization" and value "Bearer YOUR_API_KEY".
  3. Set Daily Limits: Open the "Set Config" node and adjust the dailyLimit parameter based on your account age and LinkedIn activity level.
  4. Test with Sample Post: Use the "Test Workflow" function on the Form Trigger node with a real LinkedIn post URL to verify extraction works.
  5. Activate and Deploy: Activate the workflow and use the production form URL to start processing commenters automatically.

For ongoing use, bookmark the form URL and submit new competitor posts as you discover them. The workflow will process each submission completely, even if it spans multiple days due to connection limits.

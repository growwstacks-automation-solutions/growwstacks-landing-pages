Send Typeform Results to Google Sheet, Slack & Email

Automate customer feedback & issue reporting. Capture submissions, log them centrally, and alert teams based on severity.

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automation solves a common business problem: manually handling customer feedback forms. When customers submit issues, feature requests, or feedback via Typeform, someone has to read each submission, decide who needs to know, and manually log it somewhere. This process is slow, error-prone, and critical issues can get missed.

The workflow automatically captures every Typeform submission, writes the data (name, email, problem description, severity) to a Google Sheet for permanent record, then evaluates the severity. If the problem is marked as "very severe," it immediately posts a message to a Slack channel to alert the team. If it's less severe, it simply sends an email notification or logs it quietly. This ensures urgent issues get instant attention while all feedback is systematically captured.

How It Works

Step 1: Trigger on New Typeform Submission

The workflow starts when a user submits your Typeform. It captures all form fields—typically name, email, problem description, and a severity rating.

Step 2: Write Data to Google Sheet

The submission details are automatically appended as a new row in a designated Google Sheet. This creates a centralized, searchable log of all customer feedback.

Step 3: Check Severity of Problem

The workflow reads the "severity" field from the submission. You can configure thresholds—for example, "critical" or "high" versus "low" or "suggestion."

Step 4: Conditional Routing

If severity is very high, the workflow posts a formatted message to a Slack channel (like #customer-support or #urgent-issues). If severity is lower, it triggers an email notification to a designated team member or simply continues logging.

What You'll Need

  1. A Typeform form with questions named consistently (e.g., "Name," "Email," "Severity," "Problem").
  2. A Google Sheet with columns matching those field names.
  3. Access to a Slack workspace and a channel for alerts.
  4. An email service (like Gmail or SMTP) for notifications.
  5. n8n instance (cloud or self-hosted) to run the workflow.

Pro tip: Name your Typeform fields exactly as your Google Sheet columns. This avoids mapping errors and keeps data clean.

Quick Setup Guide

  1. Download the template JSON file and import it into your n8n workspace.
  2. Configure the Typeform trigger node with your form ID and API credentials.
  3. Connect the Google Sheets node to your target spreadsheet and sheet name.
  4. Set up the Slack node with your workspace credentials and channel ID.
  5. Adjust the "If" node conditions to match your severity thresholds (e.g., "severity = 'critical'").
  6. Test by submitting a sample Typeform form and verifying the Sheet entry and Slack/email alert.

Snooze Todoist Tasks Until 3 Days Before Due Date

Automatically hide distant tasks and unsnooze them when they become urgent. Keep your inbox clean and focus on what matters now.

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automation solves a common problem for busy professionals and teams: a cluttered Todoist inbox filled with tasks that aren't immediately relevant. When you have dozens of tasks with varying due dates spread over weeks or months, it's easy to feel overwhelmed and lose focus on what truly needs attention now.

The workflow automatically moves any Todoist task with a due date more than 3 days away into a dedicated "Snoozed" project. It then monitors these snoozed tasks and, exactly 3 days before their due date, moves them back to your inbox or relevant project. This creates a dynamic, self-cleaning task list that ensures you only see items when they become urgent, reducing cognitive load and improving daily productivity.

How It Works

Step 1: Trigger & Task Detection

The workflow runs on a scheduled trigger (e.g., hourly or daily). It fetches all tasks from your Todoist inbox or specified projects and checks their due dates.

Step 2: Snooze Logic

For each task where the due date is more than 3 days in the future, the workflow moves it to a pre-defined "Snoozed" project within Todoist. This action hides the task from your main view without deleting or losing it.

Step 3: Unsnooze Timing

The workflow also checks the "Snoozed" project for tasks whose due dates are now within 3 days. When a task reaches this threshold, it's automatically moved back to its original location, ensuring it appears in your inbox just when you need to start working on it.

Step 4: Continuous Maintenance

This process runs continuously, creating a living system that constantly organizes your tasks based on urgency without any manual intervention.

What You'll Need

  1. A Todoist account with tasks that have due dates.
  2. A Todoist project named "Snoozed" (you'll create this during setup).
  3. n8n instance (cloud or self-hosted) with the Todoist node installed.
  4. Your Todoist API credentials (available in Todoist settings).
  5. Basic understanding of how to import JSON workflows into n8n.

Quick Setup Guide

  1. Download and import: Download the template file and import it into your n8n instance.
  2. Create the Snoozed project: In Todoist, create a new project named "Snoozed". Note its project ID.
  3. Configure credentials: Add your Todoist API credentials to the Todoist nodes in the workflow.
  4. Set project IDs: Update the workflow nodes with your Todoist inbox ID and the new "Snoozed" project ID.
  5. Test and activate: Add a test task with a future due date to your Todoist inbox. Run the workflow once to see it move to "Snoozed". Change the due date to tomorrow and run again to see it return.
  6. Schedule it: Set the schedule trigger to run hourly or daily based on your preference.

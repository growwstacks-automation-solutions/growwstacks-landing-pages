What This Workflow Does

Manual spreadsheet aggregation—like summing revenue columns, totaling expenses, or calculating inventory counts—is repetitive, error-prone, and time-consuming. This automation workflow solves that by automatically pulling numeric data from spreadsheet columns, database tables, or apps like Airtable, performing the aggregation with precise JavaScript logic, and delivering the calculated total instantly.

The workflow mimics what you'd do in Excel or Google Sheets with SUM functions, but operates automatically across multiple data sources. It ensures accuracy, eliminates manual calculation mistakes, and updates totals whenever source data changes—providing real-time insights without human intervention.

Whether you need daily sales totals, weekly expense summaries, monthly inventory counts, or any other column-based aggregation, this template provides the foundational automation to replace manual spreadsheet work with reliable, scheduled computation.

Example tabular data showing numeric columns ready for automated aggregation.

How It Works

Step 1: Data Input

The workflow begins with mock data or connects directly to your source—Excel files, Google Sheets, Airtable bases, PostgreSQL databases, or other tabular data sources. It extracts the specific column containing numeric values you want to aggregate.

Step 2: JavaScript Aggregation

A Function node executes custom JavaScript code that iterates through each row, extracts the numeric values from the target column, and sums them together. The code includes comments for easy modification if you need to sum multiple columns or apply filters.

Step 3: Output Delivery

The calculated total is output as a clean result. This can be sent to a dashboard, stored in another database, emailed to stakeholders, posted to Slack, or used to trigger further actions—like alerts when totals exceed thresholds.

Pro tip: Modify the JavaScript slightly to aggregate averages, counts, or filtered subsets (e.g., sum only values from last month). The flexible code structure makes customization straightforward.

What You'll Need

  1. An n8n instance (cloud or self-hosted) to run the workflow.
  2. Access to your data source: Excel file, Google Sheet, Airtable base, database, or other tabular data.
  3. The column name or field containing numeric values to sum.
  4. Basic understanding of your data structure (optional—the template includes mock data for testing).

Quick Setup Guide

  1. Download the template JSON file and import it into your n8n workspace.
  2. Replace the mock data node with your actual data source node (Google Sheets, Airtable, MySQL, etc.).
  3. Configure the source node to fetch the correct column/field containing numeric values.
  4. Review the JavaScript in the Function node—adjust if you need to sum multiple columns or apply conditions.
  5. Test the workflow with a sample dataset to verify the aggregation output.
  6. Schedule the workflow to run daily, weekly, or on-demand for automated totals.

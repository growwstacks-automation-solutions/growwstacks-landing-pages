What This Workflow Does

If you're a content creator, marketer, or business using YouTube, you know how valuable your video data is—but manually tracking titles, descriptions, tags, and performance metrics across dozens or hundreds of videos is time-consuming and error-prone. This automation solves that problem by automatically syncing your entire YouTube channel's content into an organized Google Sheets database.

The workflow connects to your YouTube channel via Google OAuth, fetches all uploaded videos (including pagination for large channels), extracts comprehensive metadata, and writes everything to a structured spreadsheet. You get a centralized, searchable content library that updates automatically, saving hours of manual work each month.

How It Works

Step 1: Authentication & Channel Identification

The workflow begins by authenticating with your Google account (used for both YouTube API and Google Sheets access). It identifies your personal YouTube channel and retrieves the uploads playlist ID, which contains all your published videos.

Step 2: Video Data Collection

Using the YouTube Data API v3, the system fetches video metadata including titles, descriptions, tags, publish dates, privacy status, thumbnails, and basic statistics. The workflow handles pagination automatically, ensuring all videos are captured regardless of channel size.

Step 3: Caption Retrieval (Optional)

If captions are available for your videos, the workflow can retrieve SRT format captions and include them in your database. This is particularly valuable for SEO analysis, content repurposing, and accessibility tracking.

Step 4: Data Transformation & Organization

Code nodes and conditional logic clean and structure the data, ensuring consistent formatting. The system handles empty fields, manages API rate limits, and prepares the information for spreadsheet insertion.

Step 5: Google Sheets Integration

Finally, the workflow uses Google Sheets' "appendOrUpdate" functionality to write the data to your specified spreadsheet. Each video becomes a row with organized columns for every metadata field, creating an instantly usable database.

This automation is ideal for YouTube creators who want to analyze their content strategy, marketing teams tracking video performance, SEO specialists optimizing video metadata, agencies managing multiple client channels, and businesses using YouTube for training or documentation. What You'll Need

  1. A YouTube channel with uploaded videos
  2. A Google account (for YouTube API and Google Sheets access)
  3. A Google Sheets document with a "Videos" tab
  4. Basic n8n setup (cloud or self-hosted)
  5. Google OAuth credentials configured in n8n

Quick Setup Guide

  1. Download the template using the button above and import it into your n8n instance.
  2. Configure Google OAuth in both the YouTube API and Google Sheets nodes with your credentials.
  3. Create your Google Sheet with a "Videos" tab and columns for: youtube_id, title, description, tags, privacyStatus, uploadStatus, thumbnail, captions.
  4. Update the spreadsheet ID in the Google Sheets nodes to point to your document.
  5. Test the workflow manually first to ensure data flows correctly.
  6. Add a Schedule Trigger (optional) to run the sync weekly or monthly automatically.

Pro tip: Start by running the workflow manually to populate your initial database. Once confirmed, add a Cron trigger for automatic weekly updates. This keeps your content database current without any ongoing effort.

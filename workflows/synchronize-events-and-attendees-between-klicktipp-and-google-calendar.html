What This Workflow Does

This automation solves the frustrating manual process of managing events across different platforms. When you run webinars, workshops, or coaching sessions, you need your email marketing contacts and calendar attendees to stay perfectly synchronized. Without automation, you waste hours copying information between systems and risk missing important attendee responses.

The workflow creates complete two-way synchronization between KlickTipp and Google Calendar. When a contact is tagged in KlickTipp, an event invitation is automatically created in Google Calendar. When attendees respond in Google Calendar, their RSVP status is immediately reflected in KlickTipp with appropriate tags. This keeps your contact database, campaigns, and event insights always current without manual intervention.

Perfect for marketers, coaches, consultants, and event organizers who want to automatically send event invites and instantly capture participation status. The system handles the full event lifecycle including creation, updates, cancellations, and attendee responses.

How It Works

The automation operates in both directions, ensuring complete synchronization between your marketing and scheduling platforms.

Google Calendar → KlickTipp Direction

This path keeps your KlickTipp contact lists synchronized with Google Calendar events. For every incoming event, the workflow processes each attendee individually to ensure accurate tracking.

KlickTipp → Google Calendar Direction

When specific tags are applied to contacts in KlickTipp, the workflow automatically creates corresponding events in Google Calendar using mapped field data.

Pro tip: Set the poll frequency to 1-5 minutes for near real-time synchronization. This ensures attendee responses are captured quickly for timely follow-up.

What You'll Need

  1. KlickTipp account with API access and appropriate permissions to create custom fields and tags
  2. Google Calendar access with API credentials (Client ID and Client Secret from Google Cloud Console)
  3. n8n instance (self-hosted or cloud) with community nodes installed for KlickTipp integration
  4. Prepared custom fields in KlickTipp for event details (summary, description, location, start/end times)
  5. Defined tags in KlickTipp for event status tracking and invitation triggers

Quick Setup Guide

  1. Download and import the template JSON file into your n8n instance
  2. Configure credentials for both KlickTipp (username/password) and Google Calendar (OAuth)
  3. Set up custom fields in KlickTipp as specified in the workflow documentation
  4. Create necessary tags in KlickTipp for event status tracking and invitation triggers
  5. Test the synchronization by creating a test event in either platform and verifying the sync works both directions
  6. Adjust polling intervals based on your needs (1-5 minutes recommended for timely updates)

Important: Ensure event end times are always after start times. If you prefer fixed-duration events, add a Date & Time node to calculate end times automatically.

What This Workflow Does

This automation transforms your Telegram bot into a powerful AI vision system. It automatically analyzes incoming images using NVIDIA Vila and GPT-4o-Mini, while also generating new images from text prompts via Stable Diffusion 3. The workflow handles the entire process from receiving content through Telegram to delivering processed results via email.

Perfect for content moderation teams, e-commerce businesses, and creative agencies, this solution eliminates manual image processing while providing consistent, AI-powered analysis at scale. The system can categorize products, detect inappropriate content, generate marketing visuals, and more - all through simple Telegram interactions.

How It Works

1. Telegram Trigger

The workflow begins by monitoring your Telegram bot for incoming messages. It detects whether the content is an image (for analysis) or text (for image generation).

2. Content Routing

A router node directs images to the AI analysis path and text prompts to the image generation path. This ensures each content type gets processed appropriately.

3. Image Analysis Path

Images are downloaded and sent to NVIDIA Vila for initial processing, then to GPT-4o-Mini for detailed analysis. Results are formatted into comprehensive reports.

4. Image Generation Path

Text prompts are sent to Stable Diffusion 3, which generates high-quality images matching the description. Multiple variations can be created automatically.

5. Results Delivery

Both paths merge at this stage, where processed results (analysis reports or generated images) are sent via Gmail to designated recipients.

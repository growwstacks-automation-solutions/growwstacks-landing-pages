Automatically answer repeated questions in Telegram groups using AI and escalate complex queries to human experts

What This Workflow Does

Customer support teams waste countless hours answering the same basic questions repeatedly. This AI-powered Telegram bot solves that problem by automatically handling common inquiries while intelligently escalating complex issues to human experts.

The workflow listens to messages in your Telegram support group, identifies genuine questions, searches your Pinecone knowledge base for matching answers, and responds instantly. When questions are new or unclear, it creates support tickets and forwards them to human experts in a private chat. The expert's response is then saved to Pinecone, creating a self-improving support system that gets smarter over time.

How It Works

1. Question Detection & Validation

The Telegram Trigger node monitors your group for new messages. It filters out casual conversation and identifies actual questions using pattern matching and intent detection. Only valid questions proceed to the next stage.

2. Knowledge Base Search

Each question is converted into embeddings using OpenAI and searched against your Pinecone vector database. Pinecone finds semantically similar questions and their approved answers, not just keyword matches, ensuring higher accuracy.

3. Automatic Response or Escalation

If Pinecone returns a high-confidence match, the bot instantly replies with the approved answer. If no good match exists or confidence is low, the workflow creates a support ticket and forwards the question to a designated human expert via private message.

4. Expert Response & Learning Loop

The human expert answers the question in the private chat. Their response is automatically added to Pinecone as a new knowledge entry, training the system to handle similar future questions automatically.

What You'll Need

  1. A Telegram bot token (create via @BotFather)
  2. OpenAI API key for embeddings and language processing
  3. Pinecone account with an index created
  4. n8n instance (cloud or self-hosted)
  5. Telegram group where the bot has admin privileges
  6. Initial knowledge base of common Q&A (can start empty)

Quick Setup Guide

Follow these steps to implement this automation in under 30 minutes:

  1. Import the template into your n8n instance using the downloaded JSON file
  2. Configure Telegram credentials with your bot token and add the bot to your support group as admin
  3. Set up Pinecone by adding your API key, environment, and index name in the Pinecone nodes
  4. Add OpenAI API key for embeddings and select your preferred model (GPT-3.5-turbo or GPT-4)
  5. Test the flow by asking a question in your Telegram group and verifying the bot responds or escalates appropriately
  6. Populate initial knowledge by having experts answer a few questions—their responses will automatically train the system

Pro tip: Start with a small, curated knowledge base of 20-30 common questions. As experts answer new questions, Pinecone will automatically expand your knowledge base organically.

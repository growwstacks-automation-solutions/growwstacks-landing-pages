What This Workflow Does

Manually checking stock prices and updating spreadsheets is a tedious, error-prone task that wastes valuable time. This automation solves that by creating a hands-free system that monitors your investment portfolio daily.

The workflow connects your Baserow database (where you store your holdings) with live financial data sources. It automatically fetches current prices for each ETF, stock, or exchange-traded product, calculates your total portfolio value, determines daily profit/loss, and compiles everything into a clean, professional email report delivered to your inbox every morning.

You get a consistent, accurate snapshot of your investments without ever opening a brokerage app or touching a spreadsheet. It transforms reactive, manual tracking into proactive, automated insight.

How It Works

The workflow is triggered on a schedule and executes a series of steps to gather, process, and deliver your portfolio data.

1. Scheduled Trigger

A Cron node initiates the workflow daily before market open. You can adjust the schedule to weekly, hourly, or any interval that fits your strategy.

2. Fetch Your Holdings

The Baserow node retrieves your investment records. Each record typically includes the asset name, ticker/ISIN, quantity owned, and purchase price.

3. Get Live Market Data

For each holding, an HTTP Request node queries a financial data source (like a stock exchange website or API) to fetch the current trading price. The HTML Extract node then parses the response to isolate the precise price figure.

4. Calculate Performance

Set nodes process the raw data. They calculate current value (quantity × price), total gain/loss, and percentage change since purchase. This logic is fully customizable for different tax lots or averaging methods.

5. Generate the Report

A Function node builds a professional HTML email body, formatting the data into a readable table with clear highlights for top gainers, losers, and overall portfolio health.

6. Deliver the Insights

Finally, the SendGrid node (or your chosen email service) sends the formatted report directly to your specified recipients. You can easily adapt this to send to Slack, Teams, or a dashboard instead.

Pro tip: You can extend this workflow to include additional metrics like dividend yield, sector allocation, or comparison to benchmark indices by adding more data processing steps after the price fetch.

Quick Setup Guide

Import and configure this template in under 15 minutes:

  1. Download the template using the button above and open your n8n editor.
  2. Import the JSON file into n8n via the "Import from File" option in the workflow menu.
  3. Configure the Baserow node with your API key and specify the correct table and view IDs.
  4. Adjust the HTTP Request node if needed—the template uses Tradegate, but you can point it to any financial data source.
  5. Set up the SendGrid node with your API key and customize the "From" email and recipient list.
  6. Test the workflow with a manual trigger, then activate the schedule.

Your first automated portfolio report will arrive at the next scheduled run.

Pro tip: Use n8n's "Test Workflow" feature step-by-step to verify each node pulls and transforms data correctly before enabling the schedule. This helps catch configuration issues early.

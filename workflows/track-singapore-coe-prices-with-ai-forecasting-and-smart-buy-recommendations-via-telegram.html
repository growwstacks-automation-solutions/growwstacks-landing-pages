What This Workflow Does

Singapore's Certificate of Entitlement (COE) system creates a volatile market where vehicle prices can swing by thousands of dollars weekly. Manually tracking these prices, analyzing trends, and timing purchases is time-consuming and prone to missed opportunities.

This automation solves that problem by systematically scraping official LTA data, enriching it with economic indicators, applying AI forecasting models, and delivering actionable buy/wait recommendations directly to your Telegram. It transforms a complex research task into a simple, automated decision-support system.

Whether you're a first-time car buyer, a fleet operator expanding your vehicles, or a financial advisor helping clients with automotive investments, this workflow provides the data intelligence needed to make informed, timely decisions.

How It Works

The workflow operates on a weekly schedule, synchronized with LTA's COE bidding cycles, to provide timely insights when they matter most.

Step 1: Data Collection & Validation

Every Thursday after bidding closes, the automation scrapes official LTA COE results for all vehicle categories. It validates data completeness, checks for anomalies, and timestamps each record for accurate historical tracking.

Step 2: Data Enrichment & Storage

The raw COE data is enriched with Singapore economic indicators (GDP growth, interest rates, vehicle quota changes) and stored in Google Sheets. This creates a comprehensive historical database for analysis.

Step 3: AI Forecasting Analysis

AI models analyze the enriched historical data to forecast COE price trends for the next 3-6 months. The system calculates confidence intervals and identifies patterns like seasonality, volatility cycles, and correlation with economic factors.

Step 4: Multi-Scenario Decision Engine

The workflow runs multiple "what-if" scenarios based on different economic conditions and quota changes. It compares current prices against historical averages, trend directions, and forecasted movements to generate objective buy/wait/monitor recommendations.

Step 5: Actionable Alert Delivery

When specific conditions are met (price drops below threshold, favorable buying window detected, or significant trend change), instant alerts with detailed analysis are sent via Telegram. The system includes dashboard summaries for weekly review.

Quick Setup Guide

Follow these steps to implement this COE tracking automation in under 30 minutes:

  1. Import the Template: Download the JSON file and import it into your n8n instance using the "Import from File" option
  2. Configure Credentials: Set up credentials for OpenAI/NVIDIA API, Google Sheets, and Telegram in n8n's credentials management
  3. Customize Google Sheet: Create a new Google Sheet and update the workflow with your Sheet ID. The template will create necessary tabs automatically
  4. Set Telegram Chat ID: Start a chat with your Telegram bot and note your chat ID for alert delivery configuration
  5. Adjust Alert Thresholds: Modify the conditional nodes to set your personal price thresholds and buying criteria
  6. Schedule Execution: Set the schedule trigger to run weekly on Thursdays at 9:30 AM SGT (30 minutes after bidding results are published)
  7. Test & Validate: Run the workflow manually once to ensure all connections work and data flows correctly to your Sheet and Telegram

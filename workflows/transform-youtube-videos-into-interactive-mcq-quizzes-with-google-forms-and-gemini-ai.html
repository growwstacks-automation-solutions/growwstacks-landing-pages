What This Workflow Does

This automation solves a common but time-consuming problem for educators, corporate trainers, and online course creators: creating engaging assessments from video content. Manually watching a 30-minute training video, identifying key points, writing questions, and building a quiz can take 2-3 hours. This workflow reduces that to minutes.

It automatically analyzes any YouTube video (up to 50 minutes) using Google Gemini AI, extracts the core educational content, and generates a complete multiple-choice quiz with automatic grading in Google Forms. The system handles everything from video processing to form creation and grading setup, delivering a ready-to-share quiz link.

The business value is substantial: reduce content creation time by 90%, ensure consistent assessment quality, scale training programs effortlessly, and provide immediate feedback to learners. Whether you're creating compliance training, onboarding materials, or educational content, this automation turns passive video watching into active learning.

How It Works

The workflow follows a logical pipeline that mimics how an expert educator would create assessments, but completely automated.

1. User Input & Trigger

The process starts with a simple n8n web form where users submit the YouTube video URL, their email, quiz title, and desired number of questions (1-90). This form trigger initiates the entire automation without any manual intervention.

2. AI Content Analysis

Google Gemini 2.5 Flash AI analyzes the video transcript, identifying key concepts, facts, procedures, and learning objectives. The AI is prompted to focus on educational content and avoid trivial details, ensuring questions are meaningful and relevant.

3. Question Generation

The AI generates structured JSON output containing questions, correct answers, and plausible distractors. A structured output parser validates the format, and error handling manages API limits or content issues, providing clear feedback if adjustments are needed.

4. Google Forms Creation

The workflow creates a new Google Form via API, configures it as a quiz with automatic grading, and populates it with all generated questions. Each question is set with the correct answer and point values, creating a fully functional assessment tool.

5. Delivery & Completion

The user receives the direct link to their generated quiz via the web form response. The form is immediately ready for sharing with students, trainees, or audience members, complete with analytics and response tracking in Google Sheets.

Pro tip: For corporate training, add an email notification step to automatically send quiz links to specific employee groups based on department or role.

Quick Setup Guide

Follow these steps to implement this automation in your n8n instance:

  1. Import the template using the download button above. In n8n, create a new workflow and import the JSON file.
  2. Configure Google Gemini by creating a "Google Gemini (PaLM) API" credential in n8n with your API key from Google AI Studio.
  3. Set up Google OAuth by creating a "Google OAuth2 API" credential with Forms and Drive permissions. Authorize it through the OAuth flow.
  4. Connect credentials to the "Google Gemini Chat Model" node and both HTTP Request nodes for Forms API calls.
  5. Test the workflow by activating it and submitting the web form with a test YouTube URL. Check that the Google Form is created correctly.
  6. Customize prompts in the "Set Prompt and model" node if you need different question styles or difficulty levels.

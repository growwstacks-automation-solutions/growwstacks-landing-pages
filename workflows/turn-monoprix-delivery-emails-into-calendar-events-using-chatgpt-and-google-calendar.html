What This Workflow Does

This automation solves the problem of manually tracking grocery delivery schedules by automatically converting confirmation emails into calendar events. When you receive a delivery confirmation from Monoprix (or similar services), the workflow:

  • Triggers on new emails in your Gmail inbox
  • Extracts key details like delivery date and time using ChatGPT
  • Creates a corresponding event in your Google Calendar
  • Can notify family members through shared calendars

The solution eliminates the need to manually enter each delivery, reduces scheduling conflicts, and ensures you never miss a grocery delivery window again.

How It Works

1. Email Trigger

The workflow monitors your Gmail inbox for new messages matching specific criteria (like sender or subject line). When a matching email arrives, it captures the full message content for processing.

2. Information Extraction

ChatGPT analyzes the email text to identify and extract key details including:

  • Delivery date and time window
  • Order reference number
  • Special delivery instructions

3. Calendar Event Creation

The extracted details are formatted and used to create a new event in your specified Google Calendar. The event includes all relevant details and can be configured to send reminders.

4. Error Handling

The workflow includes validation steps to ensure data quality and can notify you if any issues occur during processing.

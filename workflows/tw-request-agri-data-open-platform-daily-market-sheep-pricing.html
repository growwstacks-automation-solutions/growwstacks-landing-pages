What This Workflow Does

This automation solves the tedious problem of manually collecting agricultural market data for analysis. Businesses in the agricultural sector, commodity trading, or supply chain management need daily pricing information to make informed decisions, but manually checking government portals and updating spreadsheets consumes valuable time and introduces errors.

The workflow automatically connects to Taiwan's Agricultural Products Open Data Platform, retrieves detailed sheep pricing and transaction data, processes each record individually, and appends the clean, structured data directly into a Google Sheets document. This creates a reliable, automated pipeline that transforms raw API data into actionable business intelligence without manual intervention.

How It Works

1. Manual Trigger Initiation

The workflow starts with a manual trigger, giving you control over when data collection occurs. This is ideal for scheduled daily or weekly updates, or for on-demand data fetching when specific market events happen.

2. API Data Fetching

An HTTP Request node connects to the Taiwan Agricultural Products Open Data Platform API with configured parameters including date ranges, market selection (like Taipei Second Market), and API authentication. The system retrieves comprehensive transaction data including upper, middle, lower, and average prices along with transaction quantities.

3. Data Processing

A Split Out node processes the API response, breaking down the data array into individual transaction records. This ensures each data point is handled accurately and can be validated or transformed independently before storage.

4. Google Sheets Integration

The processed data automatically appends to a designated Google Sheets document with proper field mapping. Each record includes transaction dates, crop codes, market information, pricing data, and quantities—creating a structured historical database ready for analysis.

What You'll Need

  1. n8n instance (cloud or self-hosted) with active workflow execution capabilities
  2. Google Sheets access with edit permissions for the target spreadsheet
  3. API credentials for the Taiwan Agricultural Products Open Data Platform (free registration)
  4. Basic spreadsheet structure with columns for date, crop, market, prices, and quantities
  5. Internet connectivity to access both the government API and Google Sheets

Quick Setup Guide

Import this template into your n8n instance and follow these steps to configure it for your needs:

  1. Import the template using the JSON file downloaded from this page
  2. Configure API parameters in the HTTP Request node with your date ranges and market selection
  3. Add your API key from the Taiwan Agricultural Products Open Data Platform
  4. Connect Google Sheets by authenticating n8n with your Google account
  5. Set your spreadsheet ID and verify field mappings match your sheet structure
  6. Test the workflow manually to ensure data flows correctly into your spreadsheet
  7. Schedule automation (optional) using n8n's schedule trigger for daily updates

Pro tip: Use the "Curl Import" feature in n8n to quickly test API responses before building the full workflow. This helps verify your API credentials and understand the data structure you'll be working with.

What This Workflow Does

This automation solves the common problem of outdated or incomplete grocery lists. Instead of manually opening Asana to check off items or add new ones, you simply send a text or voice message to a Telegram bot. The AI interprets your natural language, finds the corresponding item in your Asana grocery list, and updates it automatically—including marking items as purchased, updating quantities, and setting expiration dates.

For example, you can say "We just bought 10 cartons of milk that expires in 6 months" via voice note, and the workflow will find the "milk" task in Asana, check it off, update the quantity to 10, and set an expiration date field. This eliminates manual data entry, reduces shopping errors, and keeps your household inventory perfectly synchronized.

The system works for both individual shoppers and shared household lists, making grocery management collaborative and effortless. It's particularly valuable for busy families, roommates sharing groceries, or anyone who wants to maintain an accurate pantry inventory without administrative overhead.

How It Works

The workflow connects three powerful platforms to create a seamless grocery management system:

1. Telegram Trigger

When you send a text message or voice note to your configured Telegram bot, the workflow activates. Text messages proceed directly to processing, while voice notes are first transcribed using OpenAI's Whisper technology.

2. AI Processing & Understanding

The transcribed or text message is sent to GPT-4o mini (via OpenRouter), which extracts key information: the grocery item, action (bought/need), quantity, and any expiration details. The AI understands natural variations like "we're out of eggs" or "just purchased 2 lbs of apples."

3. Asana Search & Update

The workflow searches your Asana project for matching grocery items. If found, it updates the task accordingly—checking it off if purchased, updating custom fields for quantity and expiration, or adding notes. If the item doesn't exist, it can create a new task with the extracted details.

4. Confirmation & Feedback

Finally, the bot responds in Telegram with what changes were made and includes a direct link to the updated Asana task for verification. This creates a closed feedback loop so you know your command was processed correctly.

Quick Setup Guide

Follow these steps to implement this grocery automation in under 30 minutes:

  1. Import the template into your n8n instance using the download button above.
  2. Create your Telegram bot via @BotFather and note the authentication token.
  3. Set up Asana API access in your developer console and create a personal access token.
  4. Configure OpenAI and OpenRouter credentials in the respective nodes.
  5. Map your Asana project and custom fields for quantity, expiration, and status.
  6. Test with simple commands like "bought milk" before expanding to complex instructions.

Pro tip: Start with a test Asana project containing just a few items to verify the AI matching works correctly before connecting it to your main grocery list.

Website Downtime Alert Automation

Get instant notifications when your sites go down with automated LINE alerts and Supabase logging

Download Template JSON · n8n compatible · Free

What This Workflow Does

This automated system continuously monitors your website uptime using UptimeRobot's API, providing immediate alerts through LINE when issues are detected while maintaining a complete historical record in Supabase.

Unlike basic uptime monitors, this solution adds intelligent alerting with contextual information about the outage, helping your team quickly understand and resolve issues. The Supabase integration creates an audit trail of all downtime events for compliance and performance analysis.

How It Works

1. Scheduled Monitoring

The workflow runs on a fixed interval (typically every 1-5 minutes) to check your website status through UptimeRobot's API.

2. Status Evaluation

Each check evaluates the response against predefined thresholds to determine if the site is up, unstable, or down.

3. Alert Generation

When issues are detected, the system generates a natural-language alert using AI to provide context about the outage.

4. LINE Notification

The formatted alert is sent to your designated LINE group, ensuring your team is immediately notified on mobile devices.

5. Supabase Logging

All confirmed downtime events are recorded in Supabase with timestamps, duration, and other relevant metadata.

What You'll Need

  1. UptimeRobot API key (free tier available)
  2. LINE developer account and channel access token
  3. Supabase project with a configured database
  4. OpenAI API key for alert generation (optional)

Quick Setup Guide

  1. Create monitors in UptimeRobot for your websites
  2. Set up a LINE messaging channel and note the access token
  3. Create a Supabase table with fields for website, status, and timestamp
  4. Import the workflow JSON into your n8n instance
  5. Configure all credentials in the workflow nodes
  6. Test with scheduled triggers before going live

Pro tip: Customize the AI-generated alerts to match your team's communication style and include relevant troubleshooting steps.

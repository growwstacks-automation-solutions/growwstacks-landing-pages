What This Workflow Does

This automation transforms WhatsApp into a fully automated receipt-processing system. When employees or clients send receipt photos via WhatsApp, the workflow instantly extracts key data using advanced OCR and AI, stores everything in Google Sheets, and sends back a confirmation request—all without manual intervention.

It solves the painful problem of manual receipt entry: lost receipts, data entry errors, delayed expense reporting, and messy spreadsheets. By combining WhatsApp's convenience with AI's intelligence, businesses get real-time, accurate expense tracking that integrates directly into their accounting systems.

How It Works

1. WhatsApp Submission

Users simply send a receipt photo or PDF via WhatsApp to your business number. The Twilio WhatsApp integration captures the media file and triggers the workflow.

2. OCR Processing

The receipt is sent to LlamaParse for high-resolution OCR extraction. This specialized tool understands invoice layouts, extracts text from tables, and handles poor-quality images better than standard OCR.

3. AI Data Extraction

Two AI models (Claude via OpenRouter and Google Gemini) analyze the OCR output to extract vendor, cost, tax, currency, date, and ABN. Using multiple models increases accuracy through cross-validation.

4. Data Normalization

Custom logic cleans and standardizes the data: dates are formatted consistently, ABN numbers are validated, tax amounts are correctly interpreted (including Australian GST logic), and currencies are identified.

5. Storage & Response

The receipt is saved to Google Drive with a shareable link. All extracted fields are logged in Google Sheets. Finally, a summary is sent back via WhatsApp with quick-reply buttons for user confirmation.

What You'll Need

  1. Twilio WhatsApp Business API – A business number configured with webhooks
  2. LlamaParse API key – For advanced OCR processing
  3. OpenRouter account – For Claude AI model access
  4. Google Gemini API key – For secondary AI validation
  5. Google Drive & Sheets access – For storage and logging
  6. n8n instance – Self-hosted or cloud version

Pro tip: Start with a test WhatsApp number and a few sample receipts before rolling out to your team. This lets you verify extraction accuracy and adjust any business rules specific to your vendors.

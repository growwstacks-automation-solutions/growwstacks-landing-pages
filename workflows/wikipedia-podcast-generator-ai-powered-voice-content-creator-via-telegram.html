What This Workflow Does

This intelligent automation solves the problem of content creation fatigue. Instead of manually researching, writing, and recording educational content, this workflow automatically transforms Wikipedia articles into professional podcast episodes delivered through Telegram.

Users simply send a topic (like "Berlin" or "Shakespeare") via text or voice message to your Telegram bot. The system researches the topic using Wikipedia, creates a structured 5-minute podcast script with AI, converts it to natural-sounding audio, and sends the finished podcast back to the user—all within minutes.

The business value is significant: you can provide on-demand educational content to your audience, build authority in your niche, and engage users through voice—a highly personal medium—without the time and cost of manual production.

The workflow architecture showing Telegram integration, AI processing, and voice synthesis nodes

How It Works

1. User Request via Telegram

The workflow starts when a user sends a message to your Telegram bot. This can be text ("Tell me about quantum computing") or a voice message saying a topic name. The Telegram Trigger node captures this input and passes it to the next stage.

2. Input Type Detection & Processing

A Switch node determines whether the input is text or voice. If it's voice, the system retrieves the audio file from Telegram and uses OpenAI's Whisper model to transcribe it to text. Text inputs proceed directly to the next stage.

3. Wikipedia Research & Script Creation

The core AI agent (using Anthropic Claude) researches the requested topic using Wikipedia integration. It creates a professional 5-minute podcast script (600-750 words) with proper structure: engaging intro, main educational content, and concluding outro—all in a conversational, accessible tone.

4. Voice Synthesis & Delivery

The completed script goes to ElevenLabs for high-quality text-to-speech conversion. The AI selects an appropriate voice and generates natural-sounding audio. Finally, the Send Voice Response node delivers the podcast audio file back to the user through Telegram.

What You'll Need

  1. Telegram Bot API: Create a bot via @BotFather on Telegram and get your bot token
  2. Anthropic API key: For Claude AI to research and write podcast scripts
  3. OpenAI API key: For Whisper speech-to-text transcription of voice messages
  4. ElevenLabs API key: For high-quality text-to-speech voice synthesis
  5. Self-hosted n8n instance: This workflow uses community nodes requiring self-hosted n8n
  6. Basic server setup: To run n8n and handle webhook connections from Telegram

Quick Setup Guide

  1. Download and import the JSON template into your n8n instance
  2. Configure credentials for Telegram, Anthropic, OpenAI, and ElevenLabs in n8n
  3. Update the Telegram chat ID in the trigger node for your specific bot
  4. Test with a simple topic like "Mars" to verify the complete flow works
  5. Customize the AI prompt to adjust podcast length, tone, or structure if needed
  6. Deploy the workflow and share your bot link with your audience
